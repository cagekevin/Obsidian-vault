# Clippings/instructions.md

## 这是我关注的世界

这里是我在网上看到的有价值的东西。你读了这里，就知道我平时在关注什么领域、对什么话题感兴趣、什么信息能吸引我的注意。

---

## 文件结构

```
Clippings/
├── raw/              ← 原始资料（不可变），AI 只读不改
│   └── 转录稿、PDF摘录等
├── 来源-日期-标题.md   ← 我的整理笔记
└── ...
```

## 命名规则

`来源-日期-标题.md`
- 来源：twitter / article / video / code
- 日期：YYYYMMDD
- 标题：简短描述

例如：`twitter-20260620-Obsidian第二大脑.md`

## 输入方式

**Obsidian Web Clipper 浏览器插件**一键保存。

### 推文模式
点击插件 → 自动读取标题、链接、作者、内容 → 一键添加

### 视频阅读模式
适合学习内容型的长视频：
1. 点击插件切换**阅读模式**
2. 边看视频边看字幕
3. 点击底下的时间戳跳到对应片段
4. 一键 add to Obsidian
