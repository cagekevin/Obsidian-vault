# If You Remove It Does It Change

必要性测试。删掉它，结果会变吗？

**如果删掉一个场景，电影结局没变 → 场景多余，删掉。**

---

## W6 怎么用

### 每个场景都要过这个测试

问自己：把这个场景删掉，电影结局会变吗？

- **会变** → 场景需要，保留
- **不会变** → 场景多余，毫不犹豫删掉
- **只改变语气/情绪** → 呼吸场景。缺时间时可以删，但会损失质感

### 删减时的优先级

在 `timing-and-cutting.md` 的删减优先级里：
- 最安全的删减：重复的、平行的、多余的反应镜头
- 绝对不能碰：高潮、突转、发现、关键转折点
