# AI 创意方法论

当所有人都能轻易做出高画质视频时，单纯的视觉刺激已经很难留住观众。创意往往来自"不合理"——观众在信息流里不是被"清晰度"吸引的，而是被"违和感"抓住的。这套方法论教你如何打破思维定势，让 AI 作品脱颖而出。

---

## 思路一：打破物理定律

很多人下意识追求画面"真实"——光影符合自然、重力往下掉、物体稳稳扎在地上。但这种追求"合理"的行为，正是困住想象力的主要原因。

### 维度一：逆转重力（Anti-Gravity）

让水往上流，万物失重。这是最直接、视觉冲击力最强的做法。

**静态图公式**：
```
主体 + 常规环境 + 反常理的物理状态 + 超现实主义风格 + 电影级光影
```

**示例**：
```
电影级真实摄影。一只毛发逼真、戴着黑色粗项圈的北极熊，
正用两只爪子捧着智能手机专注地看。北极熊坐在一块长满青苔的
巨大岩石上，但这块岩石正悬浮在半空中。背景是绿色的森林，
一条巨大的瀑布违背地心引力，水流正咆哮着向天空倒流。
空中漂浮着失重的水珠和碎石。8k分辨率，极致细节。
```

**视频生成技巧**：用"图生视频"，将静态图作为首帧导入。视频提示词只强调环境动态，不要描述主体动作，避免 AI 把画面"纠正"回正常状态。

### 维度二：质量与体积的悖论

让巨型建筑轻微漂浮。极其沉重、本应稳固连接地面的东西展现出轻盈姿态，产生巨大的视觉冲击力和科幻感。

**示例**：
```
一座高耸入云的摩天大楼，底部没有连接地面，
像一个失重的气球在云层中轻微缓慢地上下浮动。
底部能看到断裂的钢筋和掉落的泥土。
```

> **关键提示词**：Anti-gravity（反重力）、Levitating/Floating subtly（漂浮/轻微悬空）、Flowing upwards（向上流）

## 思路二：让时间失控

大部分人对 AI 视频中"时间"的处理认知单一——慢放、加速、卡点。真正有冲击力的时间处理，是让时间"出问题"。

### 时空割裂感

让画面里的一部分时间飞逝，另一部分时间停滞。

**出图技巧**：强调背景的"运动模糊"和主体的"绝对静止"。

**示例**：
```
电影级摄影，深夜的繁华街道，车流和行人在背景中
化作高速运动模糊的光轨。画面正中央，一个穿风衣的
人物纹丝不动地站在原地，仿佛时间在他身上停滞。
```

### 动作的非自然阻断

动作做到一半突然停住。

**视频提示词**：
```
人物正在奔跑，动作流畅自然，在某一帧突然完全静止，
像被按下了暂停键，保持奔跑姿态悬浮在空中。
```

## 思路三：把镜头变成角色

很多人把镜头当"记录工具"——永远在画面外旁观。创意很多时候不是内容新，而是"观看方式"新。

### 巨物降临视角

从地面看巨人，或动物视角看人类。改变常规平视角度，将镜头放置在极低位置，赋予它微小生物的身份。

**示例**：
```
极端低角度仰视镜头，从地面杂草的视角仰望，
一个巨大的身影遮天蔽日般走过，靴子每一步
都让地面轻微震动，草叶在镜头前晃动。
```

### 局限的物理视角

把镜头塞进原本不可能放摄像机的地方，赋予它"物品"的视角。

**示例**：
```
从透明玻璃杯内部向外看，光线穿过水和杯壁产生折射，
外部世界被扭曲放大。一只手从杯口伸入，指尖触碰水面，
泛起涟漪，杯中的冰块轻轻碰撞杯壁。
```

> **关键提示词**：First-person POV from...（来自...的第一人称视角）、Extreme low/high angle looking at...（极端仰/俯视看着...）

---

## 总结

在 AI 的世界里，规则可以被重写：

1. **打破物理定律** — 逆转重力、质量悖论，制造反常理的视觉奇观
2. **让时间失控** — 时空割裂、动作阻断，打造时空错乱的警觉感
3. **把镜头变成角色** — 巨物视角、物品视角，用全新观看方式重塑故事
