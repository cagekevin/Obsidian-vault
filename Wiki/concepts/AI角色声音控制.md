# AI 角色声音控制

控制 AI 角色声音的核心不是堆砌情绪词，而是搭建声音结构。不要只告诉 AI 角色是谁，要告诉 AI 角色怎么说话。

---

## 控制音色：声音公式

不要只写"一个少年将军说话""一个冷艳公主开口"，这类写法只告诉 AI 角色是谁，没有告诉 AI 声音是什么样的。使用声音公式：

> **年龄感 + 声线质感 + 语速 + 情绪底色 + 发声习惯**

示例对比：

❌ 错误写法：`一个年迈谋士严肃地说。`
✅ 正确写法：
```
年龄约五十岁，声音低沉有轻微沙哑，语速缓慢沉稳，
语气中带有常年运筹帷幄的平静与压迫感，
字与字之间有停顿，呼吸深沉。
```

❌ 错误写法：`女主悲伤地说。`
✅ 正确写法：
```
年轻女性声音，声线柔和但微微颤抖，
语速比平时慢，每句话尾音轻微拖长，
气息不稳，有压抑的哽咽感。
```

---

## 控制语气：情绪词是结果，不是过程

"愤怒地说""悲伤地讲"只描述了情绪结果，没有描述声音过程。AI 更容易理解的是声音结构：

| 情绪 | 声音结构 |
|------|----------|
| 愤怒 | 音量提高，语速加快，咬字变重，句尾收紧 |
| 悲伤 | 音量下降，语速缓慢，气息变轻，尾音拖长 |
| 冷静 | 语速稳定，语气平和，每个字清晰 |
| 压迫 | 前半句平静，后半句语速渐快、音量渐低，尾音下沉 |

### 案例：反派军师从平静到压迫

❌ 错误写法：`"你以为自己赢了？"他愤怒地说，"不！你只是走进了我的陷阱！"`

✅ 正确写法：
```
前半句语速平缓，带有讽刺和掌控感，
"不！"字加重后语速加快、尾音下沉，
逐渐展现威胁和压迫感。
停顿和尾音下沉让台词更具压迫感。
```

### 案例：古装美女的隐忍

```
第一句"你为何迟迟不来"：语气平稳，尾音轻微上扬，
表现出隐忍的期待和轻微的不解。

第二句"是我哪里做错了吗"：语气开始轻微颤抖，
尾音拖长，表现出内心的不安与期盼。
```

## 控制多句台词：按时间顺序写声音变化

在长对话中，角色的情感往往是逐渐变化的。按时间顺序控制声音的递进：

1. **初始状态** — 语速较慢，带有思考或犹豫
2. **情感升温** — 语速加快，语气出现爆发
3. **情感消解** — 语气恢复平静或释然，语速放慢

### 案例：父亲与女儿的对话

场景设定：
```
父亲威严，语气低沉，语速较慢。
女儿年轻，语气从犹豫到坚定。
```

声音提示词：
```
父亲：语气低沉威严，语速缓慢，每句话尾音下沉，
表现出不容置疑的权威感。
女儿：开始时语气犹豫，语速较慢，气息不稳；
逐渐变得坚定，语速加快，音量提高，
最后一句语气恢复平静，但尾音带有微微颤抖。
```

## 控制语速：标点符号就是节奏器

| 标点 | 作用 |
|------|------|
| 逗号 ， | 短暂停顿，语速稍慢 |
| 句号 。 | 完整停顿，语速正常 |
| 省略号 …… | 延长停顿，语速放慢，带有犹豫或思考 |
| 感叹号 ！ | 情绪爆发，语速加快 |
| 换行 | 段落停顿，节奏切换 |
| 短句 | 节奏加快 |
| 长句 | 情绪铺陈 |

示例：
```
"你……难道就这么离开了吗？"
```
省略号带出角色的犹豫与不舍，语速放慢，情感加深。

## 标点控声实战深化（Seedance 2.0）

> 来源：[[sources/Seedance 2.0 标点控声]]（AI 智能夏老师 2026-06-28）

### 核心结论

AI 默认不懂情绪，必须在提示词中显式写**发声方式 + 节奏描述 + 音调变化 + 标点符号**，角色说话才能符合想象。

### 标点的核心地位：省略号

在 Seedance 2.0 演示中，「省略号可以达到声断气连，意犹未尽的感觉」是整段视频的核心技巧。原文案例：

```
"你……到底爱不爱我……"
```

省略号不是装饰，是 AI 视频生成中**最被低估的情绪控制工具**：
- AI 不一定理解"悲伤地说"
- 但能映射"……"到"声断气连、意犹未尽"的物理发声特征

### 拆句 + 标点 = 分镜节奏

通过「换行拆句 + 标点组合」控制多段台词的情感阶段，每个阶段对应一个镜头画面：

```
你，
到底爱不爱我，
我知道你一直很爱我，
可是我已结婚了，
```

逐句解析：

| 句 | 标点 | 情感阶段 | 镜头表现 |
|---|------|---------|---------|
| `你，` | 逗号 | 试探开场 | 人物特写 + 短促开口 |
| `到底爱不爱我，` | 逗号 | 质问克制 | 镜头推近 + 语气下沉 |
| `我知道你一直很爱我，` | 逗号 | 情感升温 | 镜头稳持 + 语速放慢 |
| `可是我已结婚了，` | 逗号 | 无奈收尾 | 镜头缓拉 + 尾音拖长 |

**关键洞察**：每一行 = 一个情感落点 = 一个镜头分镜。逗号控制画面内的停顿，换行控制画面间的切换。

### 与现有方法的连接

- 配合「[[AI角色声音控制]]」中"控制语气"章节的声音过程法：标点 = 声音过程的可视化指令
- 配合「[[AI角色声音控制]]」中"多句台词"章节的时间序：拆句 = 多镜头的时间线拆分
- 配合「[[AI空镜设计]]」：拆句的换行处可插入空镜作为情绪缓冲

## 声音提示词模板

### 单人单句模板

```
角色声音设定：
【年龄感】+【声线质感】+【整体语速】+【情绪底色】

台词表达方式：
【音量】+【语速】+【咬字】+【句尾处理】
```

### 多句台词模板

```
角色声音设定：
【年龄感】+【声线质感】+【整体语速】+【情绪底色】

台词节奏：
第一句：【声音状态】+【语速】+【停顿】
第二句：【情绪变化】+【重音】+【句尾】
第三句：【音量变化】+【爆发或收住】
第四句：【最终情绪落点】
```

---

## 总结

控制 AI 声音的三个核心方法：

1. **用声音公式控制音色**：年龄感 + 声线质感 + 语速 + 情绪底色 + 发声习惯
2. **用声音过程控制语气**：把"他很生气"改成"音量变高、咬字变重、句尾短促"
3. **用标点符号控制语速**：逗号减速，省略号停顿，感叹号爆发
