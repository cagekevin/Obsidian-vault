# 产物先行

写 AI 图像提示词的第一原则：**先定义交付物类型，再写它长什么样子**。

## 核心思想

不同产物触发模型不同的精修和布局逻辑：

- 广告图 → 创意简报写法，品牌约束清晰
- 信息图 → 结构+标签+箭头写法，密集文字用高档
- Logo → 品牌约束（扁平/纯色/可缩放）
- 漫画 → 以动作为中心，每个版面一个动作
- UI 模型 → 像描述已上线产品一样，不是设计草图
  - 教育图 → 受众+课程目标+视觉格式
  - 幻灯片 → 像写工具一样规范，真实文本和数据
  - **九宫格批量预览** → 三段式结构：整体布局说明 + 逐屏内容 + 全局风格统一描述

提示词应该先说"这是一张什么图"，再说"它长什么样子"。

---

## 九宫格预览的特殊写法

九宫格（或任何多图网格预览）是一种特殊的交付物类型，它的提示词结构和单图不同：

### 三段式结构

```
[第一部分：整体布局说明]
描述网格布局格式（几行几列、整体比例），让 AI 知道这是多图预览，不是单张图。

[第二部分：逐屏内容描述]
Frame 1: [视觉描述]. 完整文案
Frame 2: [视觉描述]. 完整文案
...

[第三部分：全局风格统一描述]
全局风格（applies to all frames），只写一次，不在每个格子重复。
```

### 要点
- 每个 Frame 的视觉描述要简洁（只写"有什么"，不写"长什么样"）
- 每个 Frame 的文案必须完整保留
- 特殊 Frame 标注特殊视觉要求（如成分屏的微观科技感）
- 全局风格只写一次，不重复
