# AI 视频提示词生成工作流

从故事到最终提示词的完整流程。按顺序执行，每步完成后检查再进入下一步。**每步按需加载**——用不到的就跳过。

---

## 第1步：定故事

**需要时**：任何视频都需要

- [ ] 核心行动是什么？一个人试图做什么？所有镜头围绕它展开
- [ ] 场景价值有变化吗？情绪从哪到哪？
- [ ] 每个场景是由前一个导致的，不只是时间顺序？
- [ ] 一句话说清：谁、做什么、为什么、结果

**一致性检查**：故事本身有没有矛盾？核心行动和结果一致吗？

**参考**：[[AI导演创作思维#核心行动驱动]]、[[Scene Value]]、[[Causal Chain Audit]]

---

## 第2步：定影调

**需要时**：任何视频都需要

- [ ] 什么风格？电影感 / 皮克斯 / 广告 / 纪实 / 其他？
- [ ] 什么影调？低长调 / 中长调 / 高长调？
- [ ] 什么色温？暖 / 冷 / 中性？
- [ ] 什么对比度？高 / 中 / 柔和？
- [ ] 什么质感？写实 / 半写实+手绘 / 胶片颗粒 / 干净数码？

**一致性检查**：全片影调统一吗？不会镜1低长调、镜2高长调吧？

**参考**：[[AI影调控制]]、[[AI氛围与色彩控制]]

---

## 第3步：定分镜

**需要时**：任何视频都需要

- [ ] 主镜定了吗？核心画面3-4个，删掉任何一个故事都不完整？
- [ ] 每个主镜的信息价值和情绪价值都判断了吗？
- [ ] 辅助镜都在帮主镜讲故事吗？
- [ ] 景别/运镜/节奏/时长都标了吗？
- [ ] 空镜放在哪里？情绪需要停顿的位置？
- [ ] 节奏曲线设计了吗？快慢有依据吗？
- [ ] 有大运镜或大场景变换吗？需要首尾帧控制吗？

**一致性检查**：
- 时间线一致吗？全是白天/全是夜晚？有没有前后矛盾？
- 空间关系合理吗？人物从哪来到哪去，逻辑通顺吗？
- 情绪弧线合理吗？紧张度曲线有没有逻辑？

**参考**：[[AI分镜设计]]、[[Pacing and Rhythm]]、[[AI首尾帧控制]]、[[AI空镜设计]]

---

## 第4步：定镜头语言

**需要时**：任何镜头都需要

- [ ] 镜头信息写在最前面了吗？（反向写提示词）
- [ ] 人物移动时用了"朝镜头/背对镜头/从左侧进入"吗？
- [ ] 运镜词选了吗？推拉摇移跟绕甩、手持/固定/航拍？
- [ ] 需要前景遮挡来制造空间层次吗？
- [ ] 角度选了吗？低角度/高角度/平视？

**一致性检查**：同一个场景的机位跳切合理吗？不会180度越轴吧？

**参考**：[[AI镜头语言进阶]]、[[AI Camera Movements]]、[[视觉平衡与动线构图]]

---

## 第5步：定光影方案

**需要时**：任何镜头都需要

- [ ] 主光源方向在哪？左/右/上/下/前/后？
- [ ] 色温是什么？暖黄/冷白/混合？
- [ ] 补光怎么打？正面弱补光/环境反弹？
- [ ] 需要特殊光效吗？体积光/轮廓光/逆光剪影？
- [ ] 同一个场景内的所有镜头光影一致吗？

**一致性检查**：每个场景的光源方向统一吗？全片影调没跑偏吧？

**参考**：[[AI打光五法]]、[[光线布置模块化]]、[[AI光源提示词模板库]]、[[AI室内光控制]]

---

## 第6步：定人物

**需要时**：画面里有人时需要

- [ ] **人物圣经**写了吗？三个记忆点、一个强项两个辅助、妆造公式（身份+服装+发型+状态+画面质感）、变装/真实两套状态？
- [ ] **角色锚定**做了吗？写详细角色描述锁定外观/比例/服饰/气质，后续复用。关键句："不要重新设计我的角色"
- [ ] **微表情**：有面部特写时写肌肉状态不写情绪标签。用控制强度/动作驱动/动态演变三种方法
- [ ] **情绪降噪词**：subtle/faint/slight（基础降维），suppressed/restrained（克制隐藏），barely visible/a hint of（极微），fleeting/gradual（时间流逝），relaxed/neutral（状态）
- [ ] **微表情模板**：内疚/藐视/忍笑/微笑/失落/惊讶/挑眉/惊喜/惊恐/忧郁隐忍
- [ ] 多镜头同一个人物？资产三件套保持一致了吗？

**一致性检查**：人物服装/外貌在镜头间一致吗？角色锚定描述是否复用？

**参考**：[[AI角色资产搭建]]、[[角色锚定]]、[[sources/微表情情绪特征集]]

---

## 第7步：定动作

**需要时**：画面里有人时需要

- [ ] 只有一个主动作吗？其余从属于它？
- [ ] **四层动态**写全了吗？身体+跟随+环境+声音？
- [ ] **活人感**：人物在"顺便做某件事"吗？公式：人物+正在做的小事+下意识反应+情绪落点。同一动作不重复超过一次
- [ ] 双人以上镜头？对手戏四镜头调度结构用了吗？（遮挡关系→观察判断→距离变化→改变对方行动）
- [ ] 多镜头人物一致性？风格提取与约束系统做了吗？

**一致性检查**：动作有因果逻辑吗？人物移动方向在镜头间一致吗？

**参考**：[[AI视频动作设计]]、[[风格提取与约束系统]]

---

## 第8步：定声音

**需要时**：有人物对白或关键动作时需要

- [ ] 有人物说话吗？声音公式用了吗？（年龄感+声线+语速+情绪+发声习惯）
- [ ] 关键动作配音效了吗？不要加太满
- [ ] 环境音设计了吗？风声/车流/空调声？
- [ ] 音乐什么时候进、什么时候出？

**一致性检查**：声音和画面匹配吗？音效有逻辑来源吗？

**参考**：[[AI角色声音控制]]、[[AI视频动作设计#声音连接]]

---

## 第9步：写提示词

**需要时**：所有信息齐了之后

- [ ] **五板块结构**：情绪+主体+光线+镜头+细节
- [ ] **克制原则**：用状态描述代替程度描述。不说"非常亮"说"柔和的光线"，不说"快速跑"说"步伐急促"
- [ ] **视觉描述优先**：用视觉画面词代替抽象词。"嘴角下压微颤"代替"难过"，"45度侧光"代替"高级感"
- [ ] **描述比重**：核心信息描述字数最多，次要信息少写
- [ ] 不变量锁死了吗？变量开放了吗？
- [ ] 如果是图片——先做信息版再做情绪版了吗？

**一致性检查**：提示词本身有没有自相矛盾的指令？

**参考**：[[AI视觉语法体系]]、[[克制原则]]、[[视觉描述优先原则]]、[[描述比重原则]]、[[变量与不变量分离]]、[[AI生图两段式]]

---

## 第10步：检查与修正

**需要时**：提示词写完后，或 AI 生成结果不对时

- [ ] **反向约束**：明确说不要什么，比只说要什么更有效
- [ ] **权重竞争**：强词有没有碾压弱词？词之间有没有冲突？
- [ ] **知识诅咒**：有没有默认 AI 知道的东西？
- [ ] **必要性测试**：每个镜头/元素删掉它结果会变吗？没变就删掉
- [ ] **编辑三要素**：变量（改什么）+ 不变量（保持什么）+ 关键句（只改什么）
- [ ] **小步迭代**：先出基础图再逐步编辑，每步只改一个变量
- [ ] **生图决策规则**：注意力前置、冗余信号稀释、省略接管、加法非增强
- [ ] AI 生成结果不对？三步修正：删高干扰动作→修正强磁→明确动作歧义
