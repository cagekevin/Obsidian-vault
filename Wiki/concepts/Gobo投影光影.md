# Gobo投影光影

Gobo（Go-Between）是放在光源前的图案模板，用来产生特定形状的投影光影。在产品摄影和商业出图中，Gobo 光影可以快速提升画面的质感和氛围。

## 核心原理

不是写提示词描述光影效果，而是**用一张光影参考图作为垫图**，让模型直接参考那张图的光影形状来渲染。

## 完整提示词模板

### 图生图（已有图片加光影）

```
给上传的这张图加上戏剧性的投影光影。
只编辑这张图，只改灯光。
保持构图、相机角度、透视、主体、轮廓、材质、标签、文字完全不变。
保持背景色调不变，不要重新设计背景。
不要添加、删除或重新设计任何元素。
只使用上传的光影参考图来塑造投影光和阴影形状。
光影必须是真实的投影光包裹在 3D 表面上的效果，不能是平面叠加贴图。

光源方向：[左侧/右侧/顶部/后方]
光线质感：[锐利/柔和/雾化]
覆盖范围：[背景为主/主体为主/全画面]
环境补光：[低/中/高]
氛围：[空气干净/薄雾/水汽]
情绪：[高端商业/时尚/自然/梦幻]

保持产品标签和 Logo 的可读性。
只改灯光和阴影，其他一切不变。
```

### 文生图（直接生成带光影的新图）

```
生成一张新的场景图，光影参考上传的光影参考图。
场景：[你的场景描述]
光源方向：[左侧/右侧/顶部]
光线质感：[锐利/柔和/雾化]
覆盖范围：[主体和墙面/全画面]
环境补光：[低/中/高]
氛围：[薄雾/水汽/空气干净]
情绪：[高级静奢/自然/时尚]
光影必须是真实的投影光，不能是平面贴图。
```

## 参数调节

| 参数 | 选项 | 效果 |
|------|------|------|
| 光线质感 | 锐利 / 柔和 / 雾化 | 锐利=百叶窗感，柔和=商业感，雾化=梦幻感 |
| 覆盖范围 | 背景为主 / 主体为主 / 全画面 | 控制光影打在哪儿 |
| 环境补光 | 低 / 中 / 高 | 低=对比强，高=画面亮 |
| 氛围 | 空气干净 / 薄雾 / 水汽 | 影响画面通透感 |

## 光影类型与场景匹配

| 光影类型 | 适合场景 |
|---------|---------|
| 百叶窗光 | 时尚人像、产品、酒店、黑色电影 |
| 窗格光 | 香水/护肤品、室内场景、建筑感 |
| 植物叶影 | 度假风、热带、美妆、自然感 |
| 水波光 | spa、浴室、泳池、高级感 |
| 线性光 | 极简产品、科技感、舞台光 |
| 抽象光影 | 实验性、烟雾感、梦幻 |
