# AI 镜头语言进阶

镜头不是记录，而是在选择。让观众看到什么、感受什么，都是你决定的。这套进阶镜头语言超越了基础运镜，深入到"如何用镜头讲故事"。

---

## 反向写提示词：镜头包裹动作

AI 在生成视频时并不是在理解你的画面，只是在按顺序执行提示词。正常的写法（先写动作、再写细节、最后写镜头）会导致画面像拼出来的一样。

**核心技巧**：把镜头信息写在最前面，让所有动作都被包裹在这个镜头里发生。比如：
- 先写"低角度镜头缓慢推进"，再写人物动作
- 先写"手持镜头轻微晃动"，再写场景内容

这样画面会有"被拍出来的感觉"，而不是"生成出来的感觉"。

---

## 以摄像机为参照系

AI 没有方向感，尤其是在画面只有局部的时候。很多人写"人物向前走""人物往后退"，但 AI 只能随机判断前后。

**正确写法**：把自己想象成一台摄像机，让角色对着这台摄像机做动作：
- 朝镜头走 / 背对镜头离开
- 人物从画面左侧进入
- 人物正在靠近镜头

只要给了参照方向，就不会出错。

---

## 低角度原理

低角度可以快速去掉 AI 视频的假感，因为它天然带来两样东西：

1. **遮挡** — 地面前景、脚步进入画面，制造空间层次
2. **透视** — 人物被拉长，空间被放大，大脑会产生"这是被拍出来的"感觉

**进阶**：不要只写"低角度"，要给它一个拍摄状态（低角度手持镜头轻微晃动、低角度前景虚化）。再进阶一步：不要一上来就给全画面信息，而是让观众有一个**发现的过程**（低角度从遮挡缓慢拉出）。

---

## 空镜三作用与放置位置

空镜不只是风景画面，摇晃的烛火、掉落在地上的发簪、门外突然停下的脚步都可以承担空镜的作用（电影叙事中这叫"插入镜头"）。

> 空镜的完整方法论（三类提取方法、五要素提示词公式、叙事改写、常见问题）详见 [[AI空镜设计]]

**三个作用**：
1. 交代故事发生的环境
2. 控制视频的节奏和呼吸感
3. 代替人物表达情绪

**三个放置位置**：
1. 一场戏开始之前 — 用环境空镜快速交代时间和地点
2. 人物说完重要台词以后 — 加入道具或环境空镜，给观众留出消化情绪的时间
3. 两场戏转换的时候 — 用空镜完成过渡，避免画面直接跳到下一个场景

**注意**：空镜不是越多越好。一个空镜控制在一到三秒，只放在情绪需要停顿的位置。

## 情绪蒙太奇三法

很多人喜欢用人物状态去表达情绪（紧张、生气、失望），但真正能控制情绪节奏的是拍摄手法。不要只告诉 AI 角色"看起来怎么样"，要告诉 AI 画面"让观众感受到什么"。

### 方法一：用环境代替情绪

**底层原理：客观对应物法则（Objective Correlative）**

人的内心情绪是抽象的。导演不会让演员干巴巴地说"我好难过"，而是把情绪投射到具象的天气、光影、场景的衰败感上。环境怎么变，观众的心就怎么走。

❌ 错误写法（把情绪结果当指令）：
```
A young man standing in the room, looking very sad, crying, depressed.
```
→ AI 生成一张高清哭泣大头照，直白、没有留白、像假哭。

✅ 正确写法（用环境承载情绪）：
```
电影级广角镜头，一个孤独的身影坐在昏暗空荡的公寓里，背对镜头。
大雨拍打着巨大的玻璃窗，水滴模糊了窗外城市的霓虹灯。
桌子上有一盆枯萎的盆栽。冷蓝和灰色的色调，高对比度，忧郁的氛围。
```
→ 阴冷光线、模糊雨滴、枯萎植物，已经把"失落"写满整个屏幕。人物甚至不需要露脸。

> **核心公式**：隐藏面部/背影 + 恶劣天气 + 衰败静物 + 冷色调光影

### 方法二：用对比制造情绪落差

**底层原理：库里肖夫效应（Kuleshov Effect）**

观众对一个镜头的理解，很大程度上取决于它前后的镜头。情绪需要"参照物"。如果全片都是高潮，那等于没有高潮。极动之后接极静，大喜之后接大悲，反差才能引爆情绪。

❌ 错误写法（情绪叠加法）：
```
人物奔跑 → 汽车追逐 → 怒砸桌子 → 激烈争吵
```
→ 画面过于密集，观众视觉疲劳，只觉得吵闹。

✅ 正确写法（用反差制造情绪）：
```
镜头A（情绪高点：狂欢与拥挤）：
拥挤的宴会厅，人群欢笑，灯光炫目，镜头快速扫过人群。

镜头B（情绪低点：微小的脆弱感）：
狭窄的阳台角落，人物靠在栏杆上，眼神放空，
与身后热闹的宴会形成鲜明对比，环境声被闷住。
```
→ 从极度拥挤瞬间切入绝对安静，孤独感直击观众。

> **核心公式**：极度喧闹（家中）→ 衬托绝对孤绝（角落）

### 方法三：用特写放大情绪

**底层原理：视觉聚焦法则（Forced Attention）**

特写的本质是强行剥夺观众的全局视野。当画面去掉背景、去掉多余肢体，只剩下一个极小的细节时，观众的注意力无处可逃，情绪被无限放大。

❌ 错误写法（半身/全身像）：
```
A man is very angry and trying to hold back his anger, standing in the office.
```
→ AI 生成半身像，周围信息稀释了情绪。

✅ 正确写法（极端景别 + 局部细节）：
```
手部特写，手指用力捏着陶瓷杯柄，指节发白，
骨节微微颤抖，暖黄灯光在杯沿形成锐利高光。
```
```
眼部微距特写，瞳孔轻微收缩，睫毛沾着细小的水珠，
冷蓝环境光从左侧照亮半边眼眶。
```
→ 微距视角下质感更无敌，情绪穿透力成倍增加。

> **核心公式**：[极端景别] + [局部细节/微小动作] + [光影质感]

## 分镜落差制造情绪

情绪不是演出来的，而是被设计出来的。真正有情绪的畫面一定有变化：
- 前一个镜头很安静 → 几秒后风突然变大 → 落差就是情绪
- 先是混乱的环境 → 突然切到完全静止的人物 → 这种收放反而更压抑

**核心**：情绪最强的时候不是画面本身，而是两个镜头之间的差。你要设计的不是一个画面，而是镜头之间的关系。
