# Tencent Cloud Copilot

Tencent Cloud's LLM API service at `https://copilot.tencent.com/v2`. Serves as the upstream backend for [[Antigravity Tools]], providing access to 15 models across DeepSeek, GLM, Kimi, and MiniMax.
