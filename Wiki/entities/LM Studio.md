# LM Studio

A desktop application for downloading, managing, and running local LLMs. Provides a GUI-based model browser and a local server mode at `http://127.0.0.1:1234/v1`.

Recommended for users who prefer visual interfaces over CLI. Default model: Qwen 3.5 9B (Q4_K_M or Q5_K_M, ~6-8GB).
