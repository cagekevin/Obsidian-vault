# oMLX

A local inference engine built on Apple's MLX framework, optimized for Apple Silicon (M1-M4). Default port: 8000.

Key features: model download via web admin UI, SSD cache for long context, China mirror support. Recommended model: Qwen3.6-27B-8bit.
