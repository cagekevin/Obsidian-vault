## 核心方法论

1. **AI 默认不懂情绪** — 视频生成时必须显式写**发声方式 + 节奏描述 + 音调变化 + 标点符号**
2. **省略号 = 声断气连，意犹未尽** — 视频中演示"你，到底爱不爱我"片段，靠的就是省略号
3. **拆句 + 标点组合** — 通过逗号+换行拆句，分配到不同画面，控制每个阶段情感

## 演示案例

原文（按时间码）：

```
00:00 你，
00:03 到底爱不爱我，
00:05 我知道你一直很爱我，
00:11 可是我已结婚了，
```

**逐句拆解**（基于视频呈现效果）：

- `你，` — 句首 + 逗号 → 试探开场
- `到底爱不爱我，` — 问句 + 逗号 → 质问的克制
- `我知道你一直很爱我，` — 陈述 + 逗号 → 情感升温
- `可是我已结婚了，` — 转折 + 逗号 → 无奈收尾

## 关键洞察

- 标点符号是 AI 视频生成中最被低估的情绪控制工具 — 写了"悲伤地说"AI 不一定懂，但"……"省略号 AI 能映射到"声断气连"
- 拆句+换行 = 视频分镜节奏 — 每行对应一个画面/镜头的情感落点
- Seedance 2.0 的核心进步：把抽象的"语气语调"具象成 AI 可识别的标点指令
