# A5-前后效果对比组件 骨架规范
- **注意：本文件为核心阵型骨架。全局背景色套用 theme-palette，但本组件自带特定的容器与微观质感。**
- **空间与骨架分布 (Skeleton &amp; Content)**：
  - [图层 1：装饰文字层] ➔ 位于画面顶部，小字号英文。若未提供，此层自动隐藏。
  - [图层 2：核心大标题] ➔ 位于画面上方，居中对齐，字号最大。
  - [图层 3：副标题解释] ➔ 位于大标题下方，字号为大标题的 50%。若未提供，此层自动隐藏。
  - [图层 4：左右对比区] ➔ 左侧为"使用前"，右侧为"使用后"，各包含图示和简短说明。
- **微观视觉细节 (Micro-Details &amp; Textures)**：
  - **容器质感**：左右两侧各有圆角边框，中间有垂直分隔线；底部有箭头指向效果区。
  - **图标与字效**："使用前"/"使用后"标签为圆角小色块，文字加粗。
- **负空间门禁**：左右对比区之间保留 10% 间距，整体内容区四周保留 15% 安全边距。
- **弹性加载机制**：若未提供某层文案或图示，该部分自动隐藏，保持原有负空间门禁。
