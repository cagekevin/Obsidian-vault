# D3-天然成分原料展示 骨架规范

## 空间与骨架分布
- [图层 1：装饰文字层] ➔ 位于画面顶部，英文斜体小字，营造自然氛围
- [图层 2：核心大标题] ➔ 位于画面上方，突出天然成分主题
- [图层 3：副标题解释] ➔ 位于大标题下方，说明成分来源
- [图层 4：原料展示区] ➔ 展示天然原料实物图，如植物、矿物等
- [图层 5：成分说明区] ➔ 位于底部，详细说明成分功效与提取方式

## 微观视觉细节
- **容器质感**：自然质朴风格，原料实物展示
- **图标与字效**：天然标识，自然风格字体

## 负空间门禁
- 顶部装饰文字与标题间距：15px
- 标题与副标题间距：10px
- 副标题与展示区间距：20px
- 展示区与说明区间距：15px

## 弹性加载机制
- 若未提供装饰文字层，该层自动隐藏
- 若未提供副标题，该层自动隐藏
- 若未提供成分说明，该层自动隐藏
