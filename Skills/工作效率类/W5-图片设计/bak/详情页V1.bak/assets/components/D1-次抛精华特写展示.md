# D1-次抛精华特写展示 骨架规范

## 空间与骨架分布
- [图层 1：装饰文字层] ➔ 位于画面顶部，英文斜体小字，营造高端氛围
- [图层 2：核心大标题] ➔ 位于画面上方，突出次抛设计特点
- [图层 3：副标题解释] ➔ 位于大标题下方，说明次抛优势
- [图层 4：次抛特写区] ➔ 居中展示次抛精华特写，突出独立包装
- [图层 5：产品说明区] ➔ 位于底部，说明次抛规格与使用场景

## 微观视觉细节
- **容器质感**：高端精致风格，次抛包装特写
- **图标与字效**：次抛编号，精致说明字体

## 负空间门禁
- 顶部装饰文字与标题间距：15px
- 标题与副标题间距：10px
- 副标题与特写区间距：20px
- 特写区与说明区间距：15px

## 弹性加载机制
- 若未提供装饰文字层，该层自动隐藏
- 若未提供副标题，该层自动隐藏
- 若未提供产品说明，该层自动隐藏
