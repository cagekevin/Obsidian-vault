# A2-头屏模特展示组件 骨架规范
- **注意：本文件为核心阵型骨架。全局背景色套用 theme-palette，但本组件自带特定的容器与微观质感。**
- **空间与骨架分布 (Skeleton &amp; Content)**：
  - [图层 1：氛围装饰层] ➔ 位于画面顶部，高透明度大写英文。若未提供，此层自动隐藏。
  - [图层 2：核心大标题] ➔ 位于模特上方或左侧，居中或左对齐，字号最大。
  - [图层 3：副标题解释] ➔ 位于大标题下方，字号为大标题的 50%。若未提供，此层自动隐藏。
  - [图层 4：产品主图] ➔ 位于画面中心或右侧，尺寸占画面宽度 30%-40%。
- **微观视觉细节 (Micro-Details &amp; Textures)**：
  - **容器质感**：文字区域垫有磨砂玻璃态圆角卡片，边缘带有高光白描边，底部附柔和弥散阴影。
  - **图标与字效**：大标题带有细微内阴影效果；装饰文字采用斜体。
- **负空间门禁**：模特与产品图之间保留至少 15% 安全间距，文字禁止侵入产品主图边界。
- **弹性加载机制**：若未提供某层文案，该层自动隐藏，保持原有负空间门禁。
