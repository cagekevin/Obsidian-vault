# C3-使用方法步骤展示 骨架规范

## 空间与骨架分布
- [图层 1：装饰文字层]  位于画面顶部，英文斜体小字
- [图层 2：核心大标题] ➔ 位于画面上方，突出使用方法主题
- [图层 3：副标题解释] ➔ 位于大标题下方，说明使用场景
- [图层 4：步骤展示区] ➔ 分步骤展示使用方法，图文结合
- [图层 5：注意事项区] ➔ 位于底部，展示使用注意事项

## 微观视觉细节
- **容器质感**：清新自然风格，步骤编号图标
- **图标与字效**：步骤序号，清晰指引字体

## 负空间门禁
- 顶部装饰文字与标题间距：15px
- 标题与副标题间距：10px
- 副标题与步骤区间距：20px
- 步骤区与注意事项间距：15px

## 弹性加载机制
- 若未提供装饰文字层，该层自动隐藏
- 若未提供副标题，该层自动隐藏
- 若未提供注意事项，该层自动隐藏
