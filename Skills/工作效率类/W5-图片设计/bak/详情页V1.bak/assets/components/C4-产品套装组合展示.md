# C4-产品套装组合展示 骨架规范

## 空间与骨架分布
- [图层 1：装饰文字层]  位于画面顶部，英文斜体小字
- [图层 2：核心大标题]  位于画面上方，突出套装主题
- [图层 3：副标题解释]  位于大标题下方，说明套装内容
- [图层 4：产品展示区]  居中展示套装产品组合
- [图层 5：产品说明区]  位于底部，列出套装包含产品

## 微观视觉细节
- **容器质感**：清新自然风格，产品阴影效果
- **图标与字效**：产品编号，清晰说明字体

## 负空间门禁
- 顶部装饰文字与标题间距：15px
- 标题与副标题间距：10px
- 副标题与展示区间距：20px
- 展示区与说明区间距：15px

## 弹性加载机制
- 若未提供装饰文字层，该层自动隐藏
- 若未提供副标题，该层自动隐藏
- 若未提供产品说明，该层自动隐藏
