# Lovart 知识库 — root index

## Purpose

Lovart AI 设计代理的逆向工程知识库。涵盖平台功能、提示词工程、品牌系统、Skill 模板、编辑导出、业务落地场景、能力边界及系统参考。

当被问到 Lovart AI 平台的设计能力、使用方式、模型参数、品牌套件、视频生成等话题时，搜索此知识库。

## Files

- `_meta/` — 元数据：碎片跟踪表、套取话术策略、任务队列、集中配置
- `scripts/` — 工具脚本：lovart.py batch（提问）、promote（晋升）、generate-index（索引）、export-for-notebook（导出）
- `00-钓鱼话术/` — 原始渔获暂存（从 Lovart 套出的原始回复，未整理）
- `01-入门/` — 平台概述与新手指南（Brand Kit 前置设置）
- `02-提示与生成/` — 核心生成机制：prompt 构建、参考图、模型选择、视频/图片/音频生成
- `03-品牌系统/` — Brand Kit 品牌视觉资产：创建、数据结构、API、资产检索
- `04-Skill系统/` — Skill 模板生态：创建、管理、市场、嵌套、版本控制
- `05-编辑与导出/` — 图像编辑（edit_media）、HTML/PPT 导出
- `06-业务落地/` — 电商实战：亚马逊 A+、详情页、主图复刻、品牌提案
- `07-能力边界/` — 平台限制：成本、质量、错误处理、LLM 能力
- `08-参考/` — 开发者参考：工具清单、参数默认值、路由决策树、子代理
