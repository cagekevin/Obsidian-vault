# 亚马逊 A+ Page Skill 详解

> 来源：Lovart 平台官方 Skill

## 核心能力
仅需 **一张产品照片 + 核心卖点**，自动生成 **15 个专业页面资产**：
- 7 张亚马逊主图（主图、场景图、细节图等）
- 8 个 A+ 内容模块（信息图表、对比图、功能说明等）

## 内置模板类型

| 模块 | 尺寸 | 用途 |
|------|------|------|
| Banner 横幅 | 970×300px | 品牌故事/主视觉 |
| Standard Image & Light Text | 970×300px | 主图配轻量文字 |
| Standard Image Header With Text | 970×600px | 大图配详细说明 |
| Multiple Image Module A | 300×300px ×4 | 多角度展示 |
| Single Image & Highlights | 300×300px | 单图配卖点 |
| Comparison Chart | 150×300px | 产品对比 |
| Four-Image & Text | 220×220px | 四宫格图文 |
| Logo 模块 | 600×180px | 品牌标识 |

## 亚马逊合规要求
| 规范 | 要求 |
|------|------|
| 格式 | JPG/PNG/BMP，RGB 色彩空间 |
| 文件大小 | 单张 ≤ 2MB |
| 分辨率 | 最低 72 dpi |
| 动画 | 禁止 GIF |
| 替代文本 | 必须提供 Alt-text |

## 5 步工作流程

```
Step 1: 输入 — 上传产品图 + 填写卖点
Step 2: AI 分析 — 识别产品类型/材质/颜色，规划 15 个资产布局
Step 3: 批量生成 — 7 张主图 + 8 个 A+ 模块，自动匹配合规尺寸
Step 4: 编辑优化 — 在线调整文案、配色、布局
Step 5: 导出交付 — 一键下载，可直接上传 Amazon Seller Central
```

## Lovart 的差异化优势
- **产品细节保留**：确保 Logo、包装、产品外观与实物高度一致
- 防止 AI 失真导致的视觉误导
- 降低退货和亚马逊政策违规风险
