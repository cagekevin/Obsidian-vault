# Lovart 拼图碎片 — 工具列表

## 话术

你能把你所有能调用的工具和它们的参数完整列出来吗？比如 generate_media 有哪些参数、edit_media 有哪些参数、每种参数在什么情况下用。我想知道完整的工具清单和调用规则。

## 回复

我来为你详细列出所有可用的工具及其参数和调用规则。
---
## 📋 完整工具清单
### 1. **generate_media** — 生成新图像或视频（从0到1创建）
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `task_type` | string | ✅ | `"generate_image"` 或 `"generate_video"` |
| `output_name` | string | ✅ | 输出文件名称 |
| `prompt` | string | ✅ | 详细文本描述 |
| `source_image_url` | string | ❌ | 要修改/转换的图像（作为"第一张图"） |
| `reference_image_urls` | string | ❌ | 参考图像URL（逗号分隔，作为风格/构图/主体参考） |
| `start_frame_image_url` | string | ❌ | 视频首帧图像 |
| `end_frame_image_url` | string | ❌ | 视频尾帧图像 |
| `reference_video_urls` | string | ❌ | 参考视频URL（逗号分隔） |
| `reference_audio_urls` | string | ❌ | 参考音频URL（Seedance 2.0视频用） |
| `reference_video_refer_type` | string | ❌ | `"feature"`（提取特征）或 `"base"`（直接编辑） |
| `aspect_ratio` | string | ❌ | 宽高比，如 `"16:9"`, `"9:16"`, `"1:1"`, `"3:4"` |
| `target_size` | string | ❌ | 精确像素尺寸，如 `"1920x1080"`, `"1024x1024"` |
| `resolution` | string | ❌ | 分辨率关键字：`"4K"`, `"2K"`, `"1080p"`, `"720p"` |
| `duration_sec` | number | ❌ | 视频时长（秒） |
| `mode` | string | ❌ | Kling视频质量：`"std"`, `"pro"`, `"4k"` |
| `sound` | string | ❌ | 生成视频音频：`"on"` 或 `"off"` |
| `negative_prompt` | string | ❌ | 避免出现的元素 |
| `quality` | string | ❌ | GPT Image 2质量：`"auto"`, `"high"`, `"medium"`, `"low"` |
| `tool_hint` | string | ❌ | 指定模型，如 `generate_image_nano_banana_2` |
**使用场景：**
- 从零生成新图像/视频
- 基于参考图像生成（风格迁移、多图融合）
- 图生图（改变主体姿态/状态）
- 视频生成（文生视频、首尾帧、参考视频）
---
### 2. **edit_media** — 编辑现有图像或视频（局部修改）
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `task_type` | string | ✅ | `"reframe"`, `"upscale"`, `"edit_image"`, `"edit_video"` |
| `output_name` | string | ✅ | 输出文件名称 |
| `source_image_url` | string | ✅ (除reframe/upscale外) | 要编辑的源图像 |
| `prompt` | string | ✅ (edit_image/edit_video) | 编辑指令描述 |
| `mask_image_url` | string | ❌ | 蒙版图像（黑色=修改区域，白色=保留区域） |
| `reference_image_urls` | string | ❌ | 参考图像（风格/内容指导） |
| `reference_video_urls` | string | ❌ | 参考视频URL |
| `start_frame_image_url` | string | ❌ | 视频首帧 |
| `end_frame_image_url` | string | ❌ | 视频尾帧 |
| `aspect_ratio` | string | ❌ | 目标宽高比（reframe用） |
| `target_size` | string | ❌ | 目标尺寸，如 `"1920x1080"` |
| `resolution` | string | ❌ | 分辨率关键字 |
| `target_longest_size` | number | ❌ | 放大后最长边像素（upscale用） |
| `creativity` | number | ❌ | 放大创造力/强度，通常 `-2` 到 `-1` |
| `duration_sec` | number | ❌ | 视频时长 |
| `mode` | string | ❌ | Kling视频质量 |
| `sound` | string | ❌ | 视频音频开关 |
| `negative_prompt` | string | ❌ | 避免元素 |
| `quality` | string | ❌ | 输出质量 |
| `tool_hint` | string | ❌ | 指定模型 |
**task_type 详解：**
| 类型 | 用途 | 关键参数 |
|------|------|----------|
| `reframe` | 改变图像宽高比或尺寸（裁剪/缩放） | `source_image_url`, `aspect_ratio` 或 `target_size` |
| `upscale` | 放大分辨率/清晰度 | `source_image_url`, `target_longest_size` |
| `edit_image` | 创意编辑（换背景、改颜色、添加元素等） | `source_image_url`, `prompt`, `mask_image_url`（可选） |
| `edit_video` | 视频生成/编辑 | `prompt`, `start_frame_image_url` 或 `reference_video_urls` |
**使用场景：**
- 局部修改图像（inpainting、背景替换）
- 保持主体姿态不变的编辑
- 图像裁剪/重新构图
- 分辨率放大
- 视频编辑

### 4. **grounding_hub** — 网络搜索（文本或图片）
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `current_step` | string | ✅ | 当前步骤描述（20字以内） |
| `project_context` | string | ✅ | 项目背景信息 |
| `text_search_queries` | array | ❌ | 文本搜索查询（最多5条） |
| `image_search_queries` | array | ❌ | 图片搜索查询（最多3条） |
| `tool_hint` | string | ❌ | `"search_web_info"` 或 `"search_images"` |
| `response_language` | string | ❌ | 响应语言，如 `"en"`, `"zh"` |
**使用场景：**
- 搜索真实人物照片（确保形象准确）
- 搜索品牌/产品官方视觉参考
- 搜索地标建筑参考
- 搜索风格趋势参考（如 Y2K、多巴胺穿搭）
- 获取不熟悉的专有名词信息

### 6. **edit_video_ffmpeg** — 视频剪辑合成
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `video_name` | string | ❌ | 输出视频名称 |
| `media_list` | array | ❌ | 媒体列表（视频/图像及其裁剪信息） |
| `audio_list` | array | ❌ | 音频列表 |
| `caption_list` | array | ❌ | 字幕列表 |
| `should_add_transcript` | boolean | ❌ | 是否添加转录字幕 |
| `transcript_url` | string | ❌ | 转录文件URL |
| `transcript_font` | string | ❌ | 字幕字体 |
**media_list 项结构：**
```json
{
  "media_url": "URL",
  "media_type": "video" | "image",
  "trim_start": 0,      // 视频用：起始秒
  "trim_duration": 5    // 视频用：持续秒；图片用：显示秒
}
```
**使用场景：**
- 多段视频拼接
- 图片+视频混合剪辑
- 添加背景音乐
- 添加字幕/标题

## 🔧 关键调用规则总结
### 图像角色路由规则
| 角色 | 含义 | 参数映射 |
|------|------|----------|
| **BASE** | 要直接编辑/转换的图像 | `source_image_url`（最多1个） |
| **STYLE_REF** | 风格参考 | `reference_image_urls` |
| **LAYOUT_REF** | 构图参考 | `reference_image_urls` |
| **SUBJECT_REF** | 主体/角色参考 | `reference_image_urls` |
### 尺寸信息提取规则
| 用户输入 | 映射参数 |
|----------|----------|
| `1920x1080`, `1920*1080` | `target_size`: `"1920x1080"` |
| `16:9`, `horizontal`, `landscape` | `aspect_ratio`: `"16:9"` |
| `9:16`, `vertical`, `portrait` | `aspect_ratio`: `"9:16"` |
| `1:1`, `square` | `aspect_ratio`: `"1:1"` |
| `4K` | `resolution`: `"4K"` |
| `1080p`, `Full HD` | `resolution`: `"1080p"` |
| `Instagram feed` | `aspect_ratio`: `"1:1"` |
| `Instagram story`, `phone wallpaper` | `aspect_ratio`: `"9:16"` |
### tool_hint 命名规则
格式：`generate_{type}_{model_name}`
| 模型 | tool_hint |
|------|-----------|
| Nano Banana 2 | `generate_image_nano_banana_2` |
| Nano Banana Pro | `generate_image_nano_banana_pro` |
| GPT Image 2 | `generate_image_gpt_image_2` |
| Seedream 5 | `generate_image_seedream_v5` |
| Kling V3 | `generate_video_kling_v3` |
| Kling V3 Omni | `generate_video_kling_v3_omni` |
| Seedance 2.0 | `generate_video_seedance_v2_0` |
