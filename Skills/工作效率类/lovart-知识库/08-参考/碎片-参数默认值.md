## 关键说明
1. **模型特定默认值**：`aspect_ratio`、`resolution`、`mode` 等参数的实际默认值由底层模型决定，不同模型（Kling、Seedance、GPT Image 等）有自己的默认行为。
2. **互斥参数**：
   - `start_frame_image_url` / `end_frame_image_url` 与 `reference_image_urls` 在视频生成中互斥
   - `aspect_ratio` 与 `target_size` 通常二选一
3. **工具偏好**：你指定的 `generate_image_nano_banana_2` 会通过 `tool_hint` 参数传递，覆盖系统默认路由。
