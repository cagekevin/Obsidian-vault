# 详情页复刻

用户上传/提供参考详情页时，按以下流程执行复刻。

## 阶段1：参考图深度分析（5-8分钟）

让 AI 从 **9 个维度** 分析参考图：

| 维度 | 具体内容 |
|------|---------|
| 构图布局 | 主体位置、留白比例、视觉重心 |
| 光影效果 | 光源方向、软硬光、阴影形态、高光位置 |
| 背景处理 | 纯色/渐变/场景、色值、纹理 |
| 色彩搭配 | 主色、辅助色、点缀色、色温 |
| 产品呈现 | 摆放角度、透视关系、展示面 |
| 质感表现 | 光泽度、反射特性 |
| 氛围元素 | 粒子、光晕、倒影、装饰物件 |
| 文字排版 | 字体风格、字号层级、位置、颜色 |
| 视觉风格 | 高端/亲民/科技感/自然风 |

检查点：识别出所有模块？风格特征明确？文案完整提取？

## 阶段2：复刻策略规划（3-5分钟）

1. 模块优先级排序 — 头图到核心卖点到详细说明到辅助信息
2. 生成方案 — 整体生成 vs 分段生成
3. 技术路线 — 单图生成、参考多图、文字渲染策略
4. 评估资源需求

检查点：顺序合理？有无遗漏？素材充分？

## 阶段3：分段执行生成（15-30分钟）

按规划顺序逐模块生成。典型护肤品详情页分为：

### 3.1 头图区（Hero Banner）
- 工具：generate_image_nano_banana_2 / generate_image_gpt_image_2
- Prompt 要点：产品角度、光影效果、背景氛围、品牌调性

### 3.2 功效说明区
- 3-5个核心卖点，图标+文字组合
- 要点：图标风格统一、信息层级清晰、视觉符号化

### 3.3 成分展示区
- 成分可视化（原料图+说明文字）
- 要点：科学感、高级感、成分真实性

### 3.4 使用方法区
- 步骤分解图（3-4步）
- 要点：清晰易懂、与产品同框

### 3.5 质地展示区
- 产品质地微距特写
- 要点：真实质感、透光度、流动性

### 3.6 品牌背书区
- 实验室/权威认证
- 要点：专业严谨、冷静色调

### 3.7 购买引导区
- CTA按钮+促销信息
- 要点：醒目设计、紧迫感

## 阶段4：质量验收（5分钟）

逐块检查：
- 设计风格与参考图一致
- 颜色准确（无色差）
- 文字清晰可读
- 光影匹配原图
- 产品透视正确

## 光影匹配技术

### 光源方向 Prompt 模板
main light source from 45-degree top-left, secondary fill light from right side (40% intensity), rim light from back-right, studio softbox setup

### 常用光影模板
| 场景 | 模板 |
|------|------|
| 电商通用 | 45度顶侧光+填充光+轮廓光 |
| 高端产品 | 90度戏剧性侧光，高对比 |
| 美妆/食品 | 柔和顶光，均匀漫射 |
| 科技产品 | 逆光轮廓光+冷色调 |

## 容错策略

| 问题 | 处理 |
|------|------|
| 色差 | 指定精确色值 #HEX 重试 |
| 透视变形 | 保持 maintain original perspective |
| 边缘模糊 | 先 upscale_image 再编辑 |
| 整体差 | 回退到最近满意结果，拆解重做 |

