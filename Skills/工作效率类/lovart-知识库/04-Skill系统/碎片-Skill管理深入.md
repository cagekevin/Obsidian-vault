# Lovart 拼图碎片 — Skill 管理深入

## 编辑与版本管理

- **直接编辑**：已发布的 Skill 支持直接修改参数和提示词
- **版本迭代**：推荐用变量扩展而非新建 Skill，一个 Skill 覆盖多场景
- **分支管理**：可复制 Skill 创建变体版本

## 分享与权限

| 级别 | 可见范围 | 适用场景 |
|-----|---------|---------|
| 私有 | 仅创建者 | 个人测试 |
| 团队内共享 | 指定成员 | 部门协作 |
| 只读分享 | 他人可用不可改 | 品牌规范分发 |
| 可编辑分享 | 可修改并保存副本 | 协作开发 |
| 公开发布 | 全平台用户 | 开源社区贡献 |

**权限层级：** Owner → Collaborator → User → Viewer

## 数量限制

| 限制 | 典型配额 |
|-----|---------|
| 个人 Skill 总数 | 20-50 个 |
| 团队 Skill 总数 | 100-500 个 |
| 每 Skill 参考图 | 5-10 张 |
| Skill 调用次数/月 | 100-10000 次 |

## 导出导入

**导出格式：** JSON（配置）/ ZIP（配置+参考图+字体）/ Markdown（文档）

**导出内容结构：**
```
skill_export.zip
├── skill_config.json
├── prompt_template.txt
├── reference_images/
└── brand_kit_link.json
```

**导入方式：** 文件导入 / URL 导入 / 模板市场一键安装
