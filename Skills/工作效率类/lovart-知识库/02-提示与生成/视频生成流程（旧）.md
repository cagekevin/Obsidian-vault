## 视频生成 vs 图片生成：关键差异
| 维度 | 图片生成 | 视频生成 |
|------|---------|---------|
| **输出维度** | 静态单帧 | 时间序列（多帧连续播放） |
| **核心挑战** | 构图、风格、细节质量 | **时间一致性** + 动态合理性 |
| **规划环节** | 单张构思即可 | **必须做分镜/故事板规划** |
| **生成方式** | 一次出图 | 逐镜头生成，后期合成 |
| **音频配合** | 无 | **需同步设计音效/配乐/配音** |
| **后期工作** | 简单调色/裁剪 | **剪辑、转场、字幕、音画同步** |

## 视频生成的完整流程
```
需求分析 → 分镜规划（Storyboard）→ 素材生成 → 后期合成 → 最终输出
```
### 1. 需求分析
- 明确视频时长、风格、用途（品牌片/产品展示/社交媒体）
- 确定是否需要音频（配乐/配音/音效）
### 2. 分镜规划（最关键的差异环节）
分镜（Storyboard）是将视频拆解为多个镜头的视觉脚本：
| 分镜要素 | 说明 |
|---------|------|
| **镜头编号** | Shot 01, Shot 02... |
| **画面描述** | 场景、主体、动作、构图 |
| **运镜方式** | 推/拉/摇/移/固定 |
| **时长** | 每个镜头持续几秒 |
| **参考图** | 每镜头的视觉预览图 |
**为什么必须做分镜？**
- 确保多镜头之间的**视觉连贯性**（色调、人物、场景一致）
- 提前规划**叙事节奏**
- 避免生成后才发现衔接不上
### 3. 素材生成
按分镜逐一生成：
- **图片素材**：场景、背景、静物
- **视频片段**：动态镜头（基于首尾帧或参考图生成）
- **音频素材**：配乐、音效、配音
### 4. 后期合成
- **视频剪辑**：按时间线拼接各镜头
- **转场处理**：镜头间的过渡效果
- **字幕添加**：对话、标题、说明文字
- **音画同步**：确保音频与画面节奏匹配

## 帧序列生成与音频的配合
### 帧序列生成方式
| 方式 | 适用场景 | 说明 |
|------|---------|------|
| **文生视频** | 创意概念片 | 纯文字描述生成动态 |
| **图生视频** | 产品展示、角色动画 | 以单图为首帧，生成后续动态 |
| **首尾帧生成** | 需要精确控制起止状态 | 提供第一帧和最后一帧，AI补间 |
| **参考视频生成** | 风格迁移、动作复刻 | 以参考视频驱动新内容 |
### 音频合成的三种层级
```
┌─────────────────────────────────────┐
│  Layer 3: 配乐（背景音乐）           │  → 营造情绪氛围
│  Layer 2: 音效（环境音/动作音）      │  → 增强真实感
│  Layer 1: 配音/旁白                 │  → 传递信息
└─────────────────────────────────────┘
```
**配合要点：**
- 视频生成时即可开启 `sound=on` 让AI自动生成环境音效
- 后期通过剪辑工具精确对齐配音与口型/画面节奏
- 配乐的情绪曲线要与画面情绪匹配（紧张→舒缓→高潮）

## 实际案例示意
假设你要做一个 15 秒的产品宣传视频：
```
分镜规划：
├─ Shot 01 (0-3s): 产品特写，缓慢旋转展示 → 生成视频
├─ Shot 02 (3-7s): 使用场景，人物手持产品 → 生成视频  
├─ Shot 03 (7-12s): 功能特写，动态演示 → 生成视频
└─ Shot 04 (12-15s): 品牌Logo定格 → 生成图片/视频
后期合成：
├─ 按时间线拼接4个镜头
├─ 添加转场效果
├─ 叠加字幕：产品卖点
├─ 配音旁白 + 背景音乐
└─ 输出最终视频
```
