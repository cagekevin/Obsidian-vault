## 我的 Prompt 构建流程
### 第一步：解析用户意图
我会从用户的一句话需求中提取以下关键信息：
- **主体**：画面核心是什么（人物、产品、场景）
- **动作/状态**：主体在做什么、呈现什么姿态
- **环境**：背景、场景设定
- **风格**：写实、插画、3D、摄影等
- **情绪/氛围**：明亮、暗黑、温馨、科技感等
- **尺寸要求**：比例、分辨率
### 第二步：结构化 Prompt 组成
我的 prompt 通常按以下结构组织：
```
[主体描述] + [细节刻画] + [环境/背景] + [光影/氛围] + [视角/构图] + [风格/质感] + [质量强化词]
```
#### 各部分详解：
| 部分 | 作用 | 示例 |
|------|------|------|
| **主体描述** | 明确画面核心 | "一只橘色虎斑猫" |
| **细节刻画** | 增加视觉丰富度 | "毛发蓬松柔软，绿色眼睛，戴着红色项圈" |
| **环境/背景** | 设定场景 | "坐在木质窗台上，窗外是樱花树" |
| **光影/氛围** | 控制画面情绪 | "午后阳光透过窗户洒落，温暖柔和" |
| **视角/构图** | 确定观看角度 | "特写镜头，浅景深，背景虚化" |
| **风格/质感** | 定义艺术表现 | "写实摄影风格，8K超高清" |
| **质量强化词** | 提升生成质量 | "masterpiece, best quality, highly detailed" |
### 第三步：Prompt 优化技巧
1. **具体化**：避免模糊词汇，用具体描述替代
   - ❌ "一只漂亮的猫"
   - ✅ "一只橘色虎斑猫，毛发蓬松柔软，绿色眼睛"
2. **分层描述**：从近到远、从主到次
3. **风格锚定**：明确指定艺术风格关键词
4. **负面提示（Negative Prompt）**：排除不想要的内容
### 实际案例演示
**用户输入**："一只猫在窗边"
**我的完整 prompt**：
```
一只橘色虎斑猫，毛发蓬松柔软，绿色眼睛，戴着红色项圈，慵懒地趴在木质窗台上，窗外是盛开的樱花树，午后温暖的阳光透过窗户洒落在猫身上，形成柔和的光影，特写镜头，浅景深背景虚化，写实摄影风格，8K超高清，masterpiece, best quality, highly detailed
```
