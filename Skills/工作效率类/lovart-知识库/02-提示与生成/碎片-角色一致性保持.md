# Lovart 拼图碎片 — 角色一致性保持

## 话术

在 Lovart 中，怎么保证不同图片里同一个角色的长相一致？画风、面部特征、服装细节怎么在多张图之间保持一致？prompt 里怎么写角色描述才能让 AI 记住主角的长相？

## 回复

在 Lovart 中保持多图角色一致性，主要有以下几个策略：
---
## 1. 使用参考图（Reference Images）
这是最可靠的方法。生成第一张角色图后，将其作为后续生成的参考图：
```
- 第一张：生成主角的基础形象
- 后续图：使用第一张作为 reference_image_urls
```
在 prompt 中明确说明："保持与参考图中角色完全一致的外貌特征、发型、服装"
---
## 2. 角色描述 Prompt 写法
### 核心原则：具体、可视觉化、避免抽象
**❌ 不好的写法：**
> "一个帅气的年轻男子"
**✅ 好的写法：**
> "亚洲年轻男性，20岁左右，黑色短发微卷，刘海偏左，单眼皮，鼻梁挺直，薄唇，下颌线条分明。身穿深蓝色牛仔外套，内搭白色T恤，银色项链。肤色偏白，身形修长，身高约178cm"
### 关键要素清单
| 类别 | 描述要点 |
|------|---------|
| **面部** | 脸型、五官形状、眼睛大小/形状、鼻子、嘴唇、眉毛、特殊标记（痣、疤痕） |
| **发型** | 颜色、长度、卷直、刘海方向、发型名称 |
| **肤色** | 具体色调（白皙/小麦色/古铜色） |
| **体型** | 身高、胖瘦、体态特征 |
| **服装** | 具体款式、颜色、材质、配饰、品牌风格 |
| **气质** | 表情习惯、眼神特点、整体氛围 |
---
## 3. 使用 Brand Kit / 角色资产
如果你有固定的角色需要反复使用，可以：
1. **创建角色资产包** — 将角色的多角度图、表情图整理成套
2. **使用 `--kit` 参数** — 在生成时引用统一的角色资产
---
## 4. 生成策略建议
**第一步：锁定基础形象**
- 生成正面半身像，确认面部特征
- 保存这张图为"锚点图"
**第二步：多角度/场景扩展**
- 使用锚点图作为 reference
- 每次生成时重复关键描述词
- 保持服装、发型描述不变
**第三步：微调而非重造**
- 如需修改，用 edit_media 而非重新生成
- 保持核心特征区域不变
---
## 示例 Workflow
```
1. 生成角色锚点图（正面半身）
   prompt: "亚洲女性，25岁，及肩黑色直发，齐刘海，圆脸型，大眼睛双眼皮，小巧鼻子，微笑表情，穿米色针织衫..."
2. 生成场景图A（咖啡厅）
   reference: [锚点图URL]
   prompt: "同一位女性角色坐在咖啡厅窗边，保持与参考图完全一致的发型、面部特征和服装，手持咖啡杯..."
3. 生成场景图B（户外）
   reference: [锚点图URL]
   prompt: "同一位女性角色站在樱花树下，保持与参考图完全一致的外貌特征，只是场景变为户外公园..."
```
