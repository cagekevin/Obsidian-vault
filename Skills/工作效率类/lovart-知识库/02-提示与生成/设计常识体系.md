# Lovart 设计常识体系

> 来源：Lovart 设计决策依据

## 1. 色彩理论
- **色彩心理学**：蓝色=信任/专业，红色=紧迫/热情，绿色=自然/健康
- **配色模式**：单色、互补色、类似色、三角色
- **对比度与可读性**：文字与背景有足够对比度
- **品牌色彩一致性**：提取并延续品牌主色调

## 2. 排版与构图
- **视觉层级**：通过大小、粗细、颜色建立信息优先级
- **网格系统**：对齐、留白、节奏感
- **黄金比例/三分法则**：视觉平衡
- **留白原则**：呼吸感与焦点

## 3. 字体搭配
- 衬线体（传统/优雅）vs 无衬线体（现代/简洁）
- 标题与正文字体协调
- 字号、行高、字距优化

## 4. 平台适配
- 各平台尺寸规范
- 安全区域（边缘裁切区）
- 移动端小屏幕可读性

## 5. 风格趋势
- 极简/复古/赛博朋克/新中式等风格特征
- 科技/美妆/食品等行业视觉惯例

## 知识来源
- 训练数据中的设计规律（大量优质案例的共性模式）
- 设计理论的系统学习
- 用户反馈与迭代
