# 电商主图复刻完整指南

> Lovart 复刻参考图的完整技术方案 — 含产品替换、光影匹配、回退容错、品类差异

## 信息提取维度（9 个）

分析参考图时，从以下 9 个维度提取信息：

| 维度 | 具体内容 |
|------|---------|
| 构图布局 | 主体位置、留白比例、视觉重心、三分法/中心对称 |
| 光影效果 | 光源方向、软硬光、阴影形态、高光位置 |
| 背景处理 | 纯色/渐变/场景/虚化、色值、纹理质感 |
| 色彩搭配 | 主色、辅助色、点缀色、色温 |
| 产品呈现 | 摆放角度、透视关系、展示面、悬浮/平放/倾斜 |
| 质感表现 | 材质光泽度（哑光/高光/金属/玻璃）、反射特性 |
| 氛围元素 | 粒子、光晕、倒影、装饰物件 |
| 文字排版 | 字体风格、字号层级、位置、颜色 |
| 视觉风格 | 高端感/亲民感/科技感/自然风 |

## 5 步技术实现（概览）

### 环节1：产品置入构图
- **白底图** → `edit_media`（`source_image_url`=产品图, `reference_image_urls`=参考图）
- **实物照片** → `generate_media`（`reference_image_urls` 同时放参考图+产品图）

### 环节2：光影匹配
- `edit_media`，以参考图为 STYLE_REF，描述光源方向和强度

### 环节3：背景复刻
- A 方式：`edit_media` 直接继承
- B 方式：`generate_media` 重新生成

### 环节4：氛围元素
- `edit_media` 添加光晕/粒子/倒影

### 环节5：文字排版
- 精确控制 → `generate_html`
- 简单添加 → `edit_media`

## SKU 替换（产品替换）

### 核心参数配置

```
task_type: "EDIT_SINGLE_IMAGE"
```

**`image_url_list` 配置逻辑（标准配置）：**

```json
image_url_list: [
  "原电商主图URL",      // 第一位：提供背景、光影、构图参考
  "新产品参考图URL"     // 第二位：提供产品外观参考
]
```

关键原则：
- **第一张图**：原主图，提供场景环境、光影氛围、构图布局
- **第二张图**：新产品的清晰参考图（白底图最佳）
- **顺序很重要，不可颠倒**

### Prompt 编写最佳实践

**基础模板结构：**

```
Replace the [原产品描述] in the first image with the [新产品描述] 
shown in the second image, while preserving all other elements including:
- Background environment and setting
- Lighting direction, intensity, and color temperature
- Shadows and reflections
- Overall atmosphere and mood
- Composition and layout
- Any decorative elements or props
The new [新产品] should:
- Match the perspective and angle of the original product
- Integrate naturally with the existing lighting
- Maintain appropriate scale relative to the scene
- Cast realistic shadows consistent with the light source
```

**案例：护肤品瓶子替换**

```
Replace the skincare bottle in the first image with the serum bottle 
shown in the second image, while preserving:
- The marble surface and water droplets
- Soft diffused lighting from the top-left
- Reflections on the glossy surface
- Green leaf props in the background
- Overall fresh and clean atmosphere
The new serum bottle should:
- Match the vertical orientation and centered position
- Reflect the same lighting highlights on its surface
- Cast a similar soft shadow on the marble
- Maintain proportional size (approximately 15cm height)
```

**案例：电子产品替换**

```
Replace the smartphone in the first image with the tablet device 
shown in the second image, while keeping unchanged:
- The wooden desk background with coffee cup
- Warm ambient lighting from the window (right side)
- Soft shadows cast at 45-degree angle
- Cozy home office atmosphere
- Blurred background bokeh effect
The new tablet should:
- Be positioned at the same angle (slight tilt, screen visible)
- Display similar lighting reflections on the screen
- Match the shadow direction and softness
- Scale appropriately to the desk surface
```

**案例：食品包装替换**

```
Replace the snack package in the first image with the new cookie box 
shown in the second image, preserving:
- Kitchen countertop with scattered ingredients
- Natural daylight from overhead softbox
- Ingredient props (flour, chocolate chips) arrangement
- Warm, appetizing color tone
- Shallow depth of field
The new cookie box should:
- Maintain the 30-degree angled placement
- Show the front label clearly
- Integrate with the existing ingredient scatter
- Reflect the same soft highlights on packaging surface
```

### 多步替换策略

**场景1：简单替换（一步到位）**
- 适用：产品形状相似（如瓶子换瓶子）、背景简洁、光影不复杂
- 操作：直接调用一次 `edit_media`

**场景2：复杂替换（两步法）**
- 适用：产品形状差异大（如手机换平板）、背景元素复杂
- 第一步（粗替换）：`image_url_list`=[原主图, 新产品参考]，focus on 准确外观和大致位置
- 第二步（精修）：`image_url_list`=[第一步生成的图]，refine 光影和边缘融合

**场景3：多产品批量替换**
- 同一背景，替换为多个不同产品
- 推荐并行调用，每个产品独立任务：`[原主图, 产品A]`、`[原主图, 产品B]`、`[原主图, 产品C]`
- 注意：不要串行依赖（如图A生成图B），会导致背景逐渐偏移

### aspect_ratio 设置

必须与原主图保持一致。支持比例：
- 1:1（方图）
- 3:4, 4:5（竖图，电商详情页常用）
- 16:9, 21:9（横图，Banner常用）
- 2:3, 9:16（竖屏视频）

### 新产品参考图的最佳标准
- ✅ 纯白底或浅灰底
- ✅ 光线均匀，无强烈阴影
- ✅ 分辨率 ≥ 1000px
- ✅ 产品居中，占画面 60-80%
- ❌ 避免复杂背景
- ❌ 避免过度修图的产品图

## 光影精确匹配

### Prompt 描述方法

**光源方向描述：**

| 光源类型 | Prompt 描述 |
|---------|------------|
| 顶光 | `lit from above, top-down lighting, overhead light source` |
| 侧光 | `side lighting from [left/right], 45-degree side light` |
| 逆光 | `backlit, rim lighting, light from behind` |
| 正面光 | `front lighting, direct frontal light, flat lighting` |
| 多光源 | `three-point lighting, key light from left + fill light from right + rim light from back` |

精确描述示例：`main light source from 45-degree top-left, secondary fill light from right side, subtle rim light from back-right`

**软硬光描述：**

| 光质 | Prompt 描述 |
|-----|------------|
| 硬光 | `hard lighting, sharp shadows, direct light, high contrast lighting` |
| 软光 | `soft lighting, diffused light, gentle shadows, studio softbox lighting` |
| 中性光 | `moderate lighting, semi-diffused light, balanced shadow softness` |

**色温描述：**

| 色温 | Prompt 描述 |
|------|------------|
| 暖光 (2700-3500K) | `warm lighting, golden hour light, amber glow` |
| 中性光 (4000-5000K) | `neutral white lighting, daylight balanced` |
| 冷光 (5500-7000K) | `cool lighting, blue-tinted light, crisp white light` |

**阴影形态描述：**

| 类型 | Prompt 描述 |
|------|------------|
| 清晰阴影 | `sharp defined shadows, crisp shadow edges, hard cast shadows` |
| 柔和阴影 | `soft gradient shadows, feathered shadow edges` |
| 长阴影 | `long dramatic shadows, extended shadow length` |
| 无阴影 | `shadowless lighting, even illumination, no visible shadows` |

### 复杂光影的分步处理

**主次光源分离法：**
1. 识别主光源 → `primary light source from [direction], creating main shadows`
2. 添加辅助光 → `+ secondary fill light from [opposite direction] at 40% intensity`
3. 描述轮廓光 → `+ rim light from back creating edge highlights`

**光影层次拆解法：**
```
Layer 1 (Base): soft ambient lighting providing overall illumination
Layer 2 (Key): directional hard light from left creating primary shadows
Layer 3 (Accent): focused spotlight on product center for emphasis
Layer 4 (Rim): backlight creating separation from background
```

### 常用光影模板

**模板1：标准三点布光（电商通用）**
```
key light from 45-degree top-left (soft diffused, 5000K), fill light from right side (40% intensity, soft), rim light from back-right (hard, creating edge definition), studio lighting setup, clean shadows with soft edges
```

**模板2：戏剧性侧光（高端产品）**
```
dramatic side lighting from left at 90-degree angle, hard light creating strong contrast, deep shadows on right side, warm 3200K, specular highlights on left edges, moody atmosphere
```

**模板3：柔和顶光（美妆/食品）**
```
soft overhead lighting, diffused top light through large softbox, gentle shadows directly below product, even illumination, neutral 5000K, minimal contrast
```

**模板4：逆光轮廓（科技产品）**
```
strong backlighting creating rim light effect, light source from directly behind product, glowing edges, cool 6500K backlight, subtle front fill light (20% intensity)
```

### 精确匹配的关键技巧

1. **使用数值量化**：`key-to-fill lighting ratio 3:1, shadow density 65%`
2. **参考专业布光术语**：`Rembrandt lighting pattern, butterfly lighting setup`
3. **结合材质描述**：`glossy surface reflecting hard specular highlights, matte texture with diffuse light scattering`
4. **明确光影比例**：`70% of product in light, 30% in shadow`

## 品类差异复刻策略

| 维度 | 美妆 | 3C电子 | 服装 | 食品 |
|------|------|--------|------|------|
| **构图** | 居中对称，45度斜角 | 极简中心，多角度展示 | 模特展示，平铺+局部 | 俯拍/45度，场景化摆盘 |
| **光影** | 柔和漫射光，奶油光感 | 硬光高对比，冷白光 | 自然光模拟，有方向感 | 暖色调自然光，侧逆光 |
| **质感重点** | 玻璃折射，液体流动感 | 金属拉丝，屏幕显示 | 面料纹理，垂坠感 | 新鲜度，质地表现 |
| **文字排版** | 纤细优雅衬线体 | 粗体无衬线，引导线标注 | 杂志风，不规则排列 | 圆润可爱，手写风格 |
| **背景复杂度** | 中等（渐变+装饰） | 极简（纯色） | 复杂（场景化） | 复杂（生活化） |
| **色彩饱和度** | 高 | 中低 | 中高 | 极高 |

复刻实操建议：
1. 先识别品类，再对照上表拆解原图的构图、光影、质感
2. 在 prompt 中分层描述：背景层 → 产品层 → 光影层 → 文字层
3. 关键词优先级：品类特征词 > 材质质感词 > 光影氛围词 > 装饰元素词
4. 首次生成后，针对品类侧重点进行局部微调

## 复刻失败回退路径

### 完整容错决策树

```
edit_media 效果不理想
    │
    ├─ 色差问题
    │   ├─ 轻微偏差 → 叠加修正（再次 edit 带精确色值）
    │   ├─ 严重偏差 → 回退重做（优化 prompt 色彩描述）
    │   └─ 反复失败 → 分离处理（产品+背景分开处理再 fusion）
    │
    ├─ 透视变形
    │   ├─ 可接受范围 → 局部微调
    │   ├─ 明显变形 → 回退上一步 + 锁定透视 prompt
    │   └─ 严重变形 → 改用参考图合成（保持原产品不变）
    │
    ├─ 边缘问题
    │   ├─ 模糊/锯齿 → 提升分辨率后重新 edit
    │   ├─ 融合不自然 → 渐进式精修（先整体后局部）
    │   └─ 持续问题 → 切换合成模式
    │
    └─ 整体不满意
        ├─ 单一问题 → 针对性重做该步骤
        ├─ 多个问题 → 回退到最近的满意状态
        └─ 方向性错误 → 重新规划流程（调整步骤顺序）
```

### 色差修正

**回退路径A — 重新编辑：**
- 在 prompt 中明确指定色值：`keep the exact color #HEX_CODE`
- 添加参考约束：`match the color tone of the reference image exactly`

**回退路径B — 叠加修正：**
- 将色差结果作为 base image，用更精确的 prompt 再次 edit
- 示例：`adjust the product color to warm red (#D32F2F), keep everything else unchanged`

**色彩锚点技术：**
- 描述时关联具体物体：`the product should be cherry red, like fresh cherries`
- 避免抽象描述如"bright red"

### 透视变形纠正

**回退路径A — 重新生成：**
- 强调 `maintain the original product perspective and proportions`
- 指定透视类型：`front view / 45-degree angle / top-down view`

**回退路径B — 分离处理：**
1. 使用原图产品作为 reference image
2. 单独生成新背景
3. 用合成模式合并，prompt 强调：`place the product from the first image into the new background, keep its original size, angle and perspective unchanged`

**透视锁定技术：**
- 在 prompt 中强调：`the product should be placed exactly as shown, without any rotation, scaling or perspective change`

### 边缘毛糙处理

**问题类型A — 边缘模糊/锯齿：**
- 重新 edit 时添加：`sharp and clean product edges, professional product photography quality`
- 如果是分辨率问题，先用 `upscale_image` 提升清晰度再 edit

**问题类型B — 边缘融合不自然：**
- 强调：`natural lighting and shadows at product edges, seamless integration with background`
- 添加：`soft shadow beneath the product, ambient occlusion at contact points`

**问题类型C — 边缘有残留/伪影：**
- 用更精确的区域描述重新编辑：`remove only the background, keep the product completely intact`

### 步骤优化（可互换与可合并）

**可互换顺序的场景：**
- 背景替换 + 产品调色：如果新背景光线差异大 → 先换背景；如果只是色彩校正 → 先调色
- 添加元素 + 调整构图：元素位置固定 → 先调构图；元素需灵活摆放 → 先添加元素

**可合并步骤的场景：**
- 背景替换 + 光影调整：可一次完成，但需要光影调整需求明确
- 去除元素 + 添加新元素：位置相同的元素替换可一次完成
- 多个全局调整：亮度+对比度+色温可一条 prompt 解决

**不建议合并的场景：**
- 大幅背景替换 + 产品细节修改（变化太大，难以控制）
- 透视变换 + 色彩调整（透视变化可能引入新的光影问题）

### 核心容错原则

1. **可回退**：每步保留中间结果，关键步骤后设立检查点
2. **可拆解**：复杂任务分解为简单步骤，从大到小推进
3. **可组合**：简单步骤在明确时可合并，减少质量损耗
4. **可替代**：`edit_media` 失败时切换到合成/参考模式

遇到问题时的优先级：
1. 优化 prompt 重试（成本最低）
2. 调整步骤顺序（改变依赖关系）
3. 切换处理模式
4. 回退到检查点重新规划

## 调用示例

```python
# 第一步：置入构图
edit_media(task_type="edit_image",
  source_image_url="产品图",
  reference_image_urls="参考图",
  prompt="将产品置入参考图的构图框架，保持居中悬浮和透视关系")

# 第二步：精调光影
edit_media(task_type="edit_image",
  source_image_url="上一步输出",
  reference_image_urls="参考图",
  prompt="精确匹配参考图的光影，添加边缘光晕和倒影")

# 第三步：SKU 替换（保持背景不变替换产品）
edit_media(task_type="EDIT_SINGLE_IMAGE",
  image_url_list=["原电商主图URL", "新产品参考图URL"],
  prompt="Replace the skincare bottle in the first image with the serum bottle shown in the second image, while preserving the marble surface, soft diffused lighting, and fresh atmosphere. The new bottle should match the centered position, reflect the same highlights, and cast a similar soft shadow.",
  aspect_ratio="1:1")
```

## 关键原则

- `reference_image_urls` / `image_url_list` = 提取风格参考，不直接修改
- `source_image_url` = 唯一被修改的图，每步更新
- 复杂复刻分 2-3 步完成，避免单步 prompt 过载
- 每个 prompt 都要引用"参考图"明确提取什么
- 新产品参考图使用纯白底，避免背景干扰
- 每步保留中间结果，方便回退
- 编辑顺序：先做大变化（背景、构图）→ 中等调整（光影、色彩）→ 细节优化（边缘、小元素）
