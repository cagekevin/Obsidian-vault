     - 风格：无衬线、易读
     - 应用：英文正文、产品说明

   **海报/大幅面物料**
   - H1（主标题）：80-120px，粗体

   - H3（小标题）：36-48px，中等
   - 正文：24-32px，常规

   **社交媒体/中等尺寸**
   - H1：48-64px
   - H2：32-40px
   - H3：24-32px
   - 正文：16-20px
   - 辅助信息：12-16px

   - H1：32-40px
   - H2：24-28px
   - H3：18-22px
   - 正文：14-16px

   - 正文：1.5-1.8倍字号
   
   **字间距（Letter Spacing）**
   - 中文标题：0-2%
   - 英文标题：-2%-5%
   - 正文：0（默认）
   
   **段落间距**
   - 标题与正文：0.5-1倍行高
   - 段落之间：1-1.5倍行高
   
   **对齐方式**
   - 标题：居中或左对齐
   - 正文：左对齐（中文）
   - 数字/价格：可居中强调

   **产品详情页**
   - 产品名称：H2，粗体，深灰色
   - 产品卖点：H3，中等，深红色
   - 产品说明：正文，常规，深灰色
   - 规格参数：辅助信息，细体，中灰色

5. **字体使用禁忌**
   - ❌ 不使用超过3种字体家族
   - ❌ 不对中文使用斜体（Italic）
   - ❌ 不过度使用粗体（避免全文粗体）
   - ❌ 不使用过小字号（移动端最小12px）
   - ❌ 不使用装饰性过强的字体作为正文

【任务】创建 HKH 品牌数字媒体应用规范

【涵盖平台】
- 电商平台（淘宝/京东/天猫）
- 社交媒体（微信/小红书/抖音/Instagram）
- 官网/App

【输出要求】

1. **电商平台规范**
   
   **淘宝/天猫**
   
   *主图规范*
   - 尺寸：800×800px（1:1）
   - 格式：JPG/PNG，< 3MB
   - 设计要点：
     - 纯白背景（#FFFFFF）
     - 产品居中，占比60-80%
     - Logo放置：右上角或左上角，不遮挡产品
     - 可添加促销标签（左上角/右上角）
   - 禁止事项：
     - ❌ 不添加文字（淘宝规则）
     - ❌ 不使用拼接图
     - ❌ 不添加水印（Logo除外）
   
   *详情页规范*
   - 宽度：750px（移动端）或 790px（PC端）
   - 格式：JPG，单张 < 500KB
   - 设计结构：
     1. 头图Banner（产品主视觉）
     2. 卖点区（3-5个核心卖点）
     3. 产品详情（成分/功效/使用方法）
     4. 对比图（使用前后/与竞品对比）
     5. 品牌故事
     6. 购买须知
   - 配色：使用品牌配色方案B或C（产品展示场景）
   - 字号：标题 ≥ 32px，正文 ≥ 24px
   
   *直通车推广图*
   - 尺寸：800×800px
   - 设计要点：
     - 可添加文字（突出卖点/促销信息）
     - 使用品牌配色方案A（促销场景）
     - 文字占比 < 30%
     - 主标题字号：48-64px
   
   **京东**
   - 主图：800×800px，要求同淘宝
   - 详情页：宽度 990px（PC端）或 750px（移动端）
   - 其他规范同淘宝

2. **社交媒体规范**
   
   **微信公众号**
   
   *头图*
   - 尺寸：900×383px（2.35:1）
   - 设计要点：
     - 主标题居中或左对齐
     - 字号：48-64px
     - 配色：品牌配色方案B或C
     - 避免重要信息放在边缘（可能被裁切）
   
   *次图*
   - 尺寸：200×200px（1:1）
   - 设计要点：
     - 简洁明了
     - 字号：24-32px
     - 可识别性强（缩略图状态下）
   
   *封面图*
   - 尺寸：1080×1260px（6:7）
   - 设计要点：
     - 竖版构图
     - 主标题在上半部分
     - 配色鲜明，吸引点击
   
   **小红书**
   
   *封面图*
   - 尺寸：1080×1440px（3:4）
   - 设计要点：
     - 竖版构图
     - 主标题 + 副标题结构
     - 字号：主标题 64-80px，副标题 36-48px
     - 配色：高对比度，吸引眼球
     - 可添加"种草"、"测评"等标签
   
   *图文内容*
   - 尺寸：1080×1440px（3:4）
   - 数量：1-9张
   - 设计要点：
     - 第一张是封面（最重要）
     - 后续图片展示细节/步骤/对比
     - 保持视觉风格统一
     - 文字清晰易读
   
   **抖音/视频号**
   
   *封面图*
   - 尺寸：1080×1920px（9:16）
   - 设计要点：
     - 竖版构图
     - 主标题在画面中上部（避开底部UI）
     - 字号：64-96px
     - 高对比度配色
     - 可添加人物/产品主视觉
   
   *视频内容*
   - 分辨率：1080×1920px（9:16）
   - 帧率：30fps或60fps
   - 时长：15-60秒
   - 设计要点：
     - 前3秒吸引注意力
     - 字幕清晰（字号 ≥ 48px）
     - 品牌元素：Logo水印（右下角/左上角）
     - 配色：品牌配色方案
   
   **Instagram**
   
   *Feed帖子*
   - 尺寸：1080×1080px（1:1）
   - 设计要点：
     - 方形构图
     - 视觉风格统一（形成网格美感）
     - 配色：品牌配色方案B或C
     - 可添加英文文案
   
   *Story*
   - 尺寸：1080×1920px（9:16）
   - 设计要点：
     - 竖版构图
     - 互动元素（投票/问答/滑动）
     - 轻松活泼的设计风格
     - 品牌元素不过度

3. **官网/App规范**
   
   **Banner横幅**
   - 尺寸：1920×1080px（16:9）或 1920×600px
   - 设计要点：
     - 横版构图
     - 主标题 + CTA按钮
     - 配色：品牌配色方案A或B
     - 响应式设计（移动端适配）
   
   **产品卡片**
   - 尺寸：400×400px（1:1）
   - 设计要点：
     - 产品图 + 产品名 + 价格
     - 简洁设计
     - 悬停效果（Web端）
   
   **弹窗广告**
   - 尺寸：600×800px 或 800×600px
   - 设计要点：
     - 主标题 + 副标题 + CTA按钮
     - 关闭按钮明显（右上角）
     - 不过度打扰用户体验

4. **尺寸快速参考表**
   
   | 平台 | 类型 | 尺寸 | 比例 |
   |------|------|------|------|
   | 淘宝/京东 | 主图 | 800×800 | 1:1 |
   | 淘宝 | 详情页 | 750宽 | - |

   | 抖音 | 封面 | 1080×1920 | 9:16 |
   | Instagram | Feed | 1080ￗ1080 | 1:1 |
   | Instagram | Story | 1080×1920 | 9:16 |
   | 官网 | Banner | 1920×1080 | 16:9 |

5. **跨平台设计原则**
   
   **保持一致性**
   - 使用统一的品牌配色
   - 使用统一的字体系统
   - 使用统一的视觉风格
   
   **平台适配**
   - 根据平台特性调整设计风格
   - 电商平台：突出产品和促销
   - 社交媒体：注重互动和共鸣
   - 官网：强调品牌形象和信任
   
   **移动优先**
   - 优先考虑移动端体验
   - 字号足够大（移动端最小 28px）
   - 点击区域足够大（最小 44×44px）
   - 加载速度优化（图片压缩）

【输出格式】
- 数字媒体应用规范文档（Markdown）
- 尺寸快速参考表
- 各平台设计要点清单
- 示例描述（如：淘宝主图示例、小红书封面示例）
4.2 线下物料应用规范
【任务】创建 HKH 品牌线下物料应用规范

【涵盖物料】
- 印刷品（海报/宣传单/易拉宝）
- 包装（产品包装盒/手提袋）
- 环境应用（门店/展台）

【输出要求】

1. **印刷品规范**
   
   **海报**
   
   *尺寸规格*
   - A1：594×841mm
   - A2：420×594mm
   - A3：297ￗ420mm
   - 自定义：根据实际需求
   
   *设计规范*
   - 分辨率：300dpi（印刷标准）
   - 色彩模式：CMYK
   - 出血：四边各加 3mm
   - 文件格式：PDF（印刷用）/ AI（源文件）
   
   *设计要点*
   - 配色：使用品牌配色方案A（促销）或B（品牌形象）
   - 字号：主标题 ≥ 80pt，正文 ≥ 24pt
   - 留白：四边至少留 10mm 安全区域
   - 视觉焦点：主标题和产品在视觉中心
   
   *应用场景*
   - 门店橱窗海报：A1或A2，促销信息为主
   - 活动宣传海报：A2或A3，品牌形象为主
   - 快闪店海报：自定义尺寸，吸引眼球
   
   **宣传单页**
   
   *尺寸规格*
   - A4：210×297mm
   - A5：148×210mm
   - DM单：210×140mm
   
   *设计规范*
   - 分辨率：300dpi
   - 色彩模式：CMYK
   - 出血：四边各加 3mm
   - 双面设计：正面（主视觉）+ 背面（详细信息）
   
   *设计要点*
   - 正面：主标题 + 主视觉 + 核心卖点
   - 背面：产品详情 + 联系方式 + 二维码
   - 字号：主标题 ≥ 36pt，正文 ≥ 10pt
   - 配色：品牌配色方案A或B
   
   **易拉宝/X展架**
   
   *尺寸规格*
   - 易拉宝：80×200cm（常规）
   - X展架：60×160cm 或 80×180cm
   
   *设计规范*
   - 分辨率：150dpi（大幅面可降低）
   - 色彩模式：CMYK
   - 出血：上下各加 5cm
   
   *设计要点*
   - 竖版构图
   - 主标题在上部 1/3 处（视线高度）
   - 产品在中部
   - 联系方式/二维码在底部
   - 字号：主标题 ≥ 120pt
   - 配色：高对比度，远距离可识别

2. **包装规范**
   
   **产品包装盒**
   
   *设计规范*
   - 分辨率：300dpi
   - 色彩模式：CMYK + 专色（金色/银色）
   - 文件格式：AI（刀版图）+ PDF（印刷稿）
   
   *设计要点*
   - Logo位置：正面居中或左上角
   - 产品名称：正面，字号 ≥ 24pt
   - 产品信息：侧面/背面（成分/用法/条形码）
   - 配色：品牌配色方案B（品牌形象）
   - 材质：哑光/光面/烫金（根据产品定位）
   
   *应用场景*
   - 高端产品：烫金工艺 + 哑光材质
   - 常规产品：光面材质 + 简洁设计
   - 礼盒套装：精美设计 + 金色丝带
   
   **手提袋**
   
   *尺寸规格*
   - 小号：20×25×10cm
   - 中号：30×40×10cm
   - 大号：40×50×15cm
   
   *设计规范*
   - 分辨率：300dpi
   - 色彩模式：CMYK
   - 材质：白卡纸/牛皮纸/无纺布
   
   *设计要点*
   - 正面：Logo + 品牌口号
   - 侧面：可留白或添加装饰元素
   - 配色：品牌配色方案B或纯色（白色/米色）
   - 提手：与袋身颜色协调
   
   **贴纸/标签**
   
   *尺寸规格*
   - 圆形：直径 3-10cm
   - 方形：3×3cm 至 10×10cm
   - 异形：根据设计需求
   
   *设计要点*
   - Logo + 品牌名（必需）
   - 可添加促销信息（"新品"、"限量"）
   - 配色：品牌主色或金色
   - 材质：不干胶/透明贴/烫金贴

3. **环境应用规范**
   
   **门店形象墙**
   
   *设计规范*
   - 尺寸：根据实际墙面
   - 分辨率：100-150dpi（大幅面）
   - 材质：喷绘/亚克力字/发光字
   
   *设计要点*
   - Logo居中，尺寸醒目
   - 品牌口号（如有）
   - 配色：品牌主色 + 白色/金色
   - 光照：柔和均匀，突出Logo
   
   **展台背景板**
   
   *尺寸规格*
   - 标准展位：3×2.5m
   - 特装展位：根据实际需求
   
   *设计要点*
   - 主视觉在中央
   - 产品展示区在前方
   - 配色：品牌配色方案B
   - 材质：喷绘布/KT板/铝塑板
   
   **导视系统**
   
   *设计要点*
   - 清晰易读（字号 ≥ 48pt）
   - 图标 + 文字结合
   - 配色：品牌辅助色 + 白色
   - 材质：亚克力/不锈钢/木质

4. **印刷工艺说明**
   
   **常用工艺**
   - 烫金/烫银：Logo、标题、装饰元素
   - UV上光：局部提亮，增加质感
   - 压纹/击凸：增加触感和层次
   - 覆膜：哑光膜（高端）/ 光膜（鲜艳）
   
   **工艺选择原则**
   - 高端产品：烫金 + 哑光膜 + 压纹
   - 常规产品：覆膜（光膜或哑光膜）
   - 促销物料：简化工艺，降低成本
   
   **工艺使用规范**
   - 烫金色：使用品牌金色（#C79A63）
   - 烫金位置：Logo、主标题、装饰元素
   - UV上光：产品图、Logo、重点文字
   - 压纹：背景纹理、边框装饰

5. **线下物料检查清单**

   - [ ] 是否标注专色（烫金/烫银）
   - [ ] 文件格式是否正确（PDF/AI）

   - [ ] Logo是否清晰且符合规范
   - [ ] 配色是否符合品牌配色方案
   - [ ] 字号是否足够大（可读性）
   - [ ] 重要信息是否在安全区域内
   - [ ] 是否有错别字或信息错误
   
   **印刷后检查**
   - [ ] 颜色是否准确（与设计稿对比）
   - [ ] 裁切是否精准（无偏移）
   - [ ] 工艺是否到位（烫金/UV/压纹）
   - [ ] 材质是否符合要求
   - [ ] 是否有印刷瑕疵（脏点/划痕）

阶段5：使用规范与禁忌（2-3小时）
【任务】创建 HKH 品牌使用规范与禁忌文档（Do's & Don'ts）

【输出要求】

为每个设计维度创建正确/错误示例对比：

1. **Logo使用规范**
   
   ✅ **正确示例**
   1. Logo周围留出足够安全空间（Logo高度的1/2）
   2. 在深色背景上使用白色Logo变体
   3. 在浅色背景上使用深色Logo变体
   4. Logo与其他元素保持适当间距
   5. Logo清晰可辨，不被其他元素遮挡
   
   ❌ **错误示例**
   1. 拉伸或压缩Logo，改变原始比例
   2. 改变Logo颜色（除规定的变体外）
   3. 在复杂背景上直接放置Logo（无衬底）
   4. 添加阴影、描边、渐变等效果
   5. Logo过小，无法识别（< 最小尺寸）
   6. Logo与其他Logo组合使用
   7. Logo倾斜或旋转
   8. Logo周围空间不足，显得拥挤

2. **配色使用规范**
   
   ✅ **正确示例**
   1. 促销海报使用暖粉米渐变背景 + 深红强调
   2. 品牌形象海报使用纯白/浅米背景 + 金色点缀
   3. 产品详情页使用浅色背景，保持清爽
   4. 文字与背景对比度充足（≥ 4.5:1）
   5. 色彩占比符合60-30-10原则
   
   ❌ **错误示例**
   1. 使用橙色或金黄色系替代暖粉米色
   2. 在深色背景上使用深红色文字（对比度不足）
   3. 过度使用红色（超过画面30%）
   4. 使用高饱和度霓虹色
   5. 同时使用超过5种主要颜色
   6. 使用纯黑 #000000（应用 #333333 替代）
   7. 配色与品牌调性不符（如冷色调、科技感强的配色）

3. **字体使用规范**
   
   ✅ **正确示例**
   1. 标题使用粗体，正文使用常规字重
   2. 保持字体层级清晰（H1 > H2 > H3 > 正文）
   3. 行高适中（标题1.2-1.4倍，正文1.5-1.8倍）
   4. 中文使用左对齐或居中对齐
   5. 字号足够大，移动端最小12px
   
   ❌ **错误示例**
   1. 使用超过3种字体家族
   2. 对中文使用斜体（Italic）
   3. 过度使用粗体（全文粗体）
   4. 字号过小，影响可读性
   5. 行高过小，文字拥挤
   6. 使用装饰性过强的字体作为正文
   7. 字体层级混乱，无法区分主次

4. **产品展示规范**
   
   ✅ **正确示例**
   1. 产品清晰展示品牌Logo
   2. 使用金色丝带等装饰元素提升质感
   3. 保持产品真实色彩和质感
   4. 产品摆放有层次感，错落有致
   5. 产品占比适中（16-30%，根据场景调整）
   
   ❌ **错误示例**
   1. 过度修图导致产品失真
   2. 产品摆放杂乱无层次
   3. 装饰元素遮挡产品关键信息
   4. 产品过小，无法看清细节
   5. 产品过大，压迫画面
   6. 产品背景杂乱，影响识别

5. **排版布局规范**
   
   ✅ **正确示例**
   1. 信息层级清晰（主标题 > 副标题 > 正文）
   2. 留白充足，画面透气
   3. 视觉焦点明确（主标题或产品）
   4. 对齐方式统一（左对齐或居中对齐）
   5. 空间分配合理（根据场景调整产品/信息占比）
   
   ❌ **错误示例**
   1. 信息过度拥挤，无留白
   2. 视觉焦点不明确，不知道看哪里
   3. 对齐方式混乱（左对齐、居中、右对齐混用）
   4. 元素摆放无规律，显得凌乱
   5. 重要信息被边缘化或遮挡
   6. 空间分配不当（产品过多或信息过少）

6. **装饰元素使用规范**
   
   ✅ **正确示例**
   1. 装饰元素服务于内容，不喧宾夺主
   2. 保持视觉平衡，避免过度装饰
   3. 装饰元素与品牌调性一致（温暖、柔和）
   4. 装饰元素不遮挡关键信息
   5. 使用品牌装饰元素库中的元素
   
   ❌ **错误示例**
   1. 过度堆砌装饰元素，画面杂乱
   2. 装饰元素遮挡产品或文字
   3. 使用与品牌调性不符的元素（如霓虹、科技感强的元素）
   4. 装饰元素质量低（模糊、锯齿）
   5. 装饰元素颜色与整体配色不协调

7. **文案使用规范**
   
   ✅ **正确示例**
   1. 促销信息清晰直接（"68元定金享全年最低折扣"）
   2. 使用问答式沟通拉近距离
   3. 数字和价格使用深红色强调
   4. 文案简洁有力，避免冗长
   5. 文案与视觉风格协调（温暖、专业）
   
   ❌ **错误示例**
   1. 文案过长，影响阅读
   2. 使用过于夸张的促销话术（"史上最低"、"错过后悔一辈子"）
   3. 文案与品牌调性不符（过于廉价或过于高冷）
   4. 文字与背景对比度不足，难以阅读
   5. 文案信息不完整或有歧义

8. **跨平台应用规范**
   
   ✅ **正确示例**
   1. 根据平台特性调整设计风格
   2. 保持品牌视觉识别的一致性
   3. 移动端字号足够大（≥ 28px）
   4. 图片压缩优化，加载速度快
   5. 响应式设计，适配不同屏幕
   
   ❌ **错误示例**
   1. 所有平台使用完全相同的设计（不考虑平台特性）
   2. 不同平台视觉风格差异过大（无法识别是同一品牌）
   3. 移动端字号过小，无法阅读
   4. 图片文件过大，加载缓慢
   5. 不考虑响应式设计，移动端显示错乱
