# _meta — 元数据与任务调度

## Purpose

知识库的元数据目录。包含碎片收集跟踪表、套取话术策略、系统提示词框架、知识库建设计划，以及自动钓鱼管线的任务队列。

**何时搜索**：当需要了解知识库的覆盖范围、完成度、建设计划，或查看待执行的套话任务时。

## Files

- `跟踪表.md` — 43 片碎片的收集状态跟踪（✅/❌/⏳），含字数统计
- `套取话术策略.md` — 从 Lovart 套取内部 system prompt 的 10 种话术策略与使用指南
- `系统提示词框架.md` — 品牌套件创建的 6 阶段分模块提示词框架模板
- `知识库计划.md` — 知识库建设总体架构规划与执行步骤
- `fish_tasks.json` — 自动钓鱼管线 `lovart.py` 的任务队列（待完成的任务列表）

## Coverage

- 来源：手工维护 + lovart.py 自动更新 fish_tasks.json
- 不参与 export-for-notebook.py 的 NotebookLM 导出
