# Lovart 平台概述

> 来源：Lovart 官方帮助文档

## 核心理念

Lovart 被定义为 **AI 设计代理**，核心工作方式与传统设计工具不同：

- **传统工具**：只提供工具，用户自己操作
- **Lovart**：直接提供**成品**，用户像与设计师合作

### 工作流

1. **描述需求**：输入提示词、参考文件或品牌工具包
2. **自动处理**：Lovart 自动路由到 30+ 个顶级图像/视频模型，选择最佳模型执行
3. **产出资产**：几分钟内获得可编辑的成品资产

## 平台功能

| 功能 | 说明 |
|------|------|
| 统一创意工作区 | 单个无限画布上生成/编辑/组织所有内容，无需切换工具 |
| AI 技能 (Skills) | 针对特定创意任务的一键式工作流，自动串联提示词和模型 |
| 完整编辑控制 | 选中任意元素用自然语言精修，如"让背景更温暖" |
| 自动路由最佳模型 | 内置 30+ 模型，自动选择最合适的 |
| 导出就绪文件 | PNG/SVG/PSD/PPTX/MP4 等格式 |

## 快速上手建议

1. **有问题问 Agent**：直接在输入框询问，它会解释功能、建议用例或直接运行
2. **编辑可逆**：每次编辑生成新版本，原始版本保持不变
3. **从技能开始**：常见任务（Logo、社媒帖子）用技能比自由提示更快
4. **设置品牌套件**：有品牌需求时，先设置 Brand Kit 确保输出一致性

## 支持的操作

- 文本对话咨询
- 图像/视频生成
- 图像编辑（局部修改、风格迁移、尺寸调整）
- PPT 幻灯片生成
- HTML 页面生成
- 音频/配音/BGM 生成
- 品牌资产检索与管理
