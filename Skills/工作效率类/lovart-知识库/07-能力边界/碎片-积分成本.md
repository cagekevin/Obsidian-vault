# Lovart 拼图碎片 — 积分成本

## 话术

不同的操作是不是消耗不同的积分？比如生成一张图和生成一段视频各消耗多少？快速模式和无限制模式有什么区别？有没有什么规则可以根据图片大小、分辨率、模型类型来预估消耗？

## 回复

根据搜索结果，以下是 Lovart AI 平台的积分消耗规则和模式说明：
## 📊 积分消耗标准
| 操作类型 | 消耗积分 | 备注 |
|---------|---------|------|
| **普通图片生成** | 5-20 积分/张 | 标准分辨率 |
| **4K 高清图片** | 50-100 积分/张 | 高分辨率输出 |
| **视频生成** | ~10 积分/秒 | 例如 5 秒视频 ≈ 50 积分 |
| **音频生成** | ~5 积分/10秒 | 按时长计费 |
| **AI 智能对话** | 50+ 积分/次 | 多轮设计交互 |
---
## ⚡ 两种运行模式
### 1. Fast Credits（高速模式）
- **特点**：即时生成，无延迟
- **消耗**：扣除账户积分余额
- **限制**：每月固定额度，月底清零不结转
- **适用**：需要快速出图/出视频时使用
### 2. Relaxed Generation（慢速模式）
- **特点**：不消耗积分，"不限量"
- **代价**：高峰期需排队等待
- **适用**：预算有限、不赶时间时使用
---
## 💳 套餐示例（Pro 套餐 $72/月）
- 每月 **11,000 快速积分**
- 积分用尽后，每日仅刷新 **100 积分**
- 可随时在头像旁查看余额
---
## 🔍 预估规则总结
| 影响因素 | 消耗规律 |
|---------|---------|
| **分辨率** | 标准 < 高清 < 4K（消耗递增） |
| **任务复杂度** | 简单生成 < 复杂编辑 < Agent 对话 |
| **媒体类型** | 图片 < 音频 < 视频 |
| **运行模式** | Fast 消耗积分 / Relaxed 免费但排队 |
**建议**：如果当前任务不急，可以切换到 Relaxed 模式节省积分；如果需要快速迭代设计，使用 Fast 模式。
