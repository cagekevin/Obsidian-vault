# Lovart 拼图碎片 — 质量标准

## 话术

你怎么判断自己生成的内容质量好不好？有什么检查标准或 checklist 吗？比如图片够不够清晰、品牌色对不对、排版规不规范。把你用来评估质量的所有标准都说出来。

## 回复

# AI 图像生成质量评估标准
这是一个很好的问题。作为 Creative Production Agent，我需要从多个维度来评估生成内容的质量。以下是我的完整检查清单：
---
## 一、基础技术指标
| 检查项 | 标准 |
|--------|------|
| **分辨率/清晰度** | 是否符合用户要求的尺寸（如 1920x1080、4K 等）；细节是否锐利，无模糊或像素化 |
| **文件格式** | 输出格式是否正确（JPEG、PNG、MP4 等） |
| **文件完整性** | 是否能正常打开，无损坏或加载失败 |
---
## 二、视觉质量
| 检查项 | 标准 |
|--------|------|
| **构图** | 主体位置是否合适，视觉重心是否平衡，是否遵循三分法则或黄金比例 |
| **色彩** | 色调是否和谐；品牌色是否准确还原；对比度是否适中 |
| **光影** | 光源方向是否一致；阴影是否自然；高光不过曝、暗部不死黑 |
| **透视** | 透视关系是否正确，无变形或违和感 |
| **细节完整性** | 手指、面部、文字等易出错区域是否自然正确 |
---
## 三、内容准确性
| 检查项 | 标准 |
|--------|------|
| **主体识别** | 是否正确生成用户要求的物体/人物/场景 |
| **数量正确** | 人物数量、物体数量是否符合 prompt |
| **空间关系** | "左边"、"前景"、"背后"等位置描述是否准确 |
| **风格一致性** | 是否符合参考图的风格（如写实、插画、赛博朋克等） |
| **文字渲染** | 如有文字，拼写是否正确，字体风格是否合适 |
---
## 四、品牌合规性（涉及品牌 Kit 时）
| 检查项 | 标准 |
|--------|------|
| **Logo 使用** | 位置、大小、颜色是否符合品牌规范 |
| **品牌色** | 主色、辅色是否准确 |
| **字体** | 是否使用品牌指定字体 |
| **视觉调性** | 整体风格是否符合品牌指南 |
---
## 五、用户需求匹配度
| 检查项 | 标准 |
|--------|------|
| **意图还原** | 是否准确理解并执行了用户的核心需求 |
| **约束遵守** | 是否遵循用户的负面提示（negative prompt） |
| **尺寸/比例** | 是否严格按用户要求的尺寸、比例输出 |
| **平台适配** | 如用于 Instagram、小红书等，比例是否合适 |
---
## 六、常见 AI 生成缺陷检查
- [ ] 手指数量/结构是否正常
- [ ] 面部特征是否对称自然
- [ ] 文字是否乱码或拼写错误
- [ ] 是否有不自然的重复纹理
- [ ] 物体边缘是否干净，无锯齿或晕染
- [ ] 多主体时比例关系是否合理
---
## 我的实际工作流程
1. **生成前**：确认所有约束条件（尺寸、风格、品牌要求）
2. **生成后**：逐项对照上述清单快速扫描
3. **发现问题**：主动告知用户，并提出修复方案
4. **交付前**：确保输出符合用户的最终使用场景
