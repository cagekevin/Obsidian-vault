# 详情页复刻

## 适用场景
用户提供参考详情页 + 新产品图，要求复刻相同风格的详情页。

## 前置条件
- 参考详情页图片（清晰度 > 1000px）
- 新产品图片（建议白底 + 多角度，如无则用文字描述生成）
- 文字内容清单（品牌名、卖点、规格参数）
- 品牌色值（如有）

## 阶段0：材料分析与方案确认（Planner执行）
1. 并行分析 — 调用 analyse_image 同时分析参考图和产品图
2. 分析内容：参考图9维拆解（构图/光影/背景/色彩/产品/质感/氛围/文字/风格）+ 模块边界坐标；产品图角度/光影/尺寸
3. 标记可复用元素 vs 需替换元素
4. 输出方案：确认模块数量、生成顺序、预估耗时，问用户是否开始

## 阶段1：产品替换（SKU替换 — 核心）
场景A（产品图+角度匹配）：
generate_image_nano_banana_2(
  task_type: FUSION_MULTI_IMAGES,
  image_url_list: [参考详情页头图, 新产品白底图],
  prompt: 将第二张图的产品无缝替换到第一张图的场景中心位置
)

场景B（产品图+角度不匹配）：
generate_image_nano_banana_2(
  task_type: REFERENCE_TO_IMAGE,
  image_url_list: [参考详情页头图, 新产品图],
  prompt: 参考第一张图的构图/光影/氛围，将第二张图的产品重新渲染至相同视角
)

场景C（仅有文字描述）：先生成产品单品图，再按场景A/B融入

保存头图结果为 hero_image_url（后续所有模块的风格锚点）

## 阶段2-6：其他模块生成
每个模块生成时传递头图作为风格参考：
generate_image_nano_banana_2(
  task_type: REFERENCE_TO_IMAGE,
  image_url_list: [hero_image_url],
  prompt: 延续参考图的色调/光影/质感，生成当前模块内容
)

模块顺序：功效说明区 → 成分展示区 → 使用方法区 → 质地展示区 → 品牌背书区 → 购买引导区

## 文字排版策略
- 英文短标题（<10词）：AI直接生成
- 中文短标题（<15字）：AI生成 + 人工校验
- 中文长段落（>50字）：生成占位区域，交付时注明建议PS添加
- 数字/规格参数：关键数据在交付清单注明

## 质量验收
逐块检查：产品替换自然/光影匹配/色调风格一致/文字无误/比例匹配

## 4级回退
Level 1 微调 → Level 2 重构 → Level 3 降级 → Level 4 AI+PS混合

## 工具对应
- analyse_image：分析图片（阶段0）
- generate_image_nano_banana_2（FUSION_MULTI_IMAGES）：产品替换
- generate_image_nano_banana_2（REFERENCE_TO_IMAGE）：参考风格生成
- generate_image_nano_banana_2（EDIT_SINGLE_IMAGE）：微调修正
- generate_image_gpt_image_2：创意发散时替代

## ⚠️ 新发现的关键遗漏
### 🔴 **严重问题**
#### 1. **阶段0的分析深度不足**
**问题**：仅列出"9维拆解"，但未明确**如何提取可操作参数**。
```diff
- 分析内容：参考图9维拆解（构图/光影/背景/色彩/产品/质感/氛围/文字/风格）
+ 分析内容：
  - 参考图9维拆解 → 输出结构化数据：
    * 光影参数：主光源方向（左上45°）、色温（5500K暖白）、阴影硬度（soft）
    * 色彩方案：主色#HEX + 辅助色 + 背景色
    * 产品位置：中心坐标(x%, y%) + 占画面比例(30%)
    * 文字区域：坐标框 + 字体风格描述
  - 产品图分析 → 输出匹配度报告：
    * 角度差异：参考图为3/4侧视，产品图为正面（需重新渲染）
    * 光影差异：参考图暖光，产品图冷光（需调整色温）
```
**影响**：分析结果无法直接转化为生成prompt，Planner需要"脑补"参数。
---
#### 2. **场景B的prompt过于模糊**
**问题**：角度不匹配时，"重新渲染至相同视角"对AI是黑盒指令。
```diff
场景B（产品图+角度不匹配）：
- prompt: 参考第一张图的构图/光影/氛围，将第二张图的产品重新渲染至相同视角
+ prompt: 
  将第二张图的[产品名称]重新渲染为与第一张图相同的视角：
  - 产品角度：3/4侧视，顶部向下15°
  - 光源设置：左上45°主光 + 右侧补光，色温5500K
  - 产品位置：画面中心偏右，占比30%
  - 背景保持：第一张图的[具体背景描述]
  - 质感匹配：保留第二张图的材质细节（如金属反光、布料纹理）
```
**影响**：当前prompt成功率 < 60%，需多次重试。
---
#### 3. **缺少"模块间过渡"处理**
**问题**：7个模块独立生成，拼接后可能出现**色彩断层**或**风格跳跃**。
```diff
+ ## 阶段1.5：过渡色带生成（可选）
+ 当检测到相邻模块色差 > ΔE 20时，生成100px高的渐变过渡带：
+ generate_image_nano_banana_2(
+   task_type: FUSION_MULTI_IMAGES,
+   image_url_list: [模块A底部, 模块B顶部],
+   prompt: 生成从第一张图底部色调到第二张图顶部色调的自然渐变过渡
+ )
```

### 🟡 **中等问题**
#### 4. **文字排版策略缺少"多语言"处理**
**问题**：仅区分中英文，未考虑日韩/阿拉伯语等复杂场景。
```diff
+ - 日韩文字：竖排文字需标注方向，字体建议使用思源黑体
+ - 阿拉伯语：从右至左排版，需镜像处理文字区域
+ - 特殊符号：™ ® © 等需人工校验位置
```
---
#### 5. **质量验收缺少"量化指标"**
**问题**："光影匹配"、"色调一致"是主观判断，需可执行标准。
```diff
逐块检查：
- 产品替换自然/光影匹配/色调风格一致/文字无误/比例匹配
+ 产品替换自然：边缘无明显抠图痕迹，阴影方向与主光源一致
+ 光影匹配：色温差 < 500K，亮度对比度与参考图误差 < 15%
+ 色调风格一致：主色色差 ΔE < 10，整体饱和度偏差 < 20%
+ 文字无误：关键数字/品牌名100%准确，装饰文字可接受5%误差
+ 比例匹配：产品占比与参考图误差 < 10%，模块高度比例误差 < 15%
```
---
#### 6. **4级回退机制缺少"触发条件"**
**问题**：何时使用Level 1 vs Level 2？需要决策树。
```diff
+ ## 回退决策树
+ - Level 1 微调（generate_image_nano_banana_2 EDIT）：
+   触发条件：单一问题（如产品位置偏移 < 10%、色温偏差 < 1000K）
+   
+ - Level 2 重构（重新生成当前模块）：
+   触发条件：多重问题（如光影+色调同时不匹配）或Level 1失败2次
+   
+ - Level 3 降级（切换到 generate_image_gpt_image_2）：
+   触发条件：Level 2失败2次，或需要更强创意发散
+   
+ - Level 4 AI+PS混合：
+   触发条件：AI连续失败3次，或涉及复杂文字排版/精确数据表格
```
