# 考研英语一 vs 考研英语二：语言评分标准差异

> v1.6.0 新增。本文件专门处理**英语（一）和英语（二）同档位的语言水平差异**，是 `postgrad-vs-cet.md`（考研 vs CET）的下一层细分。
>
> **核心结论**：同一篇作文，**在英二的档位通常比英一高 0.5~1 档**。Skill 切换 `exam_level` 时必须加载不同的**最低达标 tier** 和**辩证要求强度**。

## 一、核心差异速查表

| 维度 | 英一 A/B | 英二 A/B | 备注 |
|------|---------|---------|------|
| 适用对象 | 学硕 + 部分专硕 | 多数专硕 | 英一考生基线更高 |
| 总分 | 30 分（A 10 + B 20）| 25 分（A 10 + B 15）| — |
| 第五档描述关键词 | **"较高语言水平"** | **"语言比较准确"** | 英一鼓励"较高"，英二止步"较好" |
| 词汇目标 tier | **High ~ Academic** | **Mid-high ~ High** | 英二不强制 academic |
| 长难句要求 | **鼓励复合结构 + 非限定 + 倒装** | 基本语法结构 + 少量复合句 | 英一必考 1-2 处复杂结构 |
| 辩证要求 | **2022+ 论述性**（必须辩证）| 议论性即可（单向论证可接受）| 英一 B 节关键差异 |
| 论证深度 | 多维归因 + 反驳 | 观点清晰 + 例证即可 | — |
| 档内调节 | 1-3 分（允许 0.5）| 1-3 分（允许 0.5）| 相同 |

## 二、同档位的语言基准对比

### 第五档

| 维度 | 英一（9-10 / 17-20）| 英二（9-10 / 13-15）|
|------|---------------------|---------------------|
| 词汇 | 多处高阶搭配（exemplify, posit, multifaceted）| 用词准确 + 零星高阶（significant, demonstrate）|
| 句式 | **必须出现**非限定 / 倒装 / 强调句（≥ 2 处）| 复合句熟练即可，复杂句不做硬性要求 |
| 错误容忍 | 几乎无错（仅试复杂结构时偶有） | 极少错（≤ 2 处小错，无严重错）|
| 论证 | **辩证 + 多维** | 观点清晰 + 例证充分 |

**判定实例**：

> 一篇有"观点清晰 + 3 处复合句 + 用词准确 + 无错"的作文：
> - 英一 B 节 → **4 档（13-16）**（够不上 17+，因缺复杂结构和辩证）
> - 英二 B 节 → **5 档（13-15）**（已达到英二五档要求）

### 第四档

| 维度 | 英一（7-8 / 13-16）| 英二（7-8 / 10-12）|
|------|---------------------|---------------------|
| 词汇 | High-tier 为主 | Mid-high tier |
| 句式 | 复合 + 1-2 处复杂 | 复合为主 |
| 允许的漏点 | 漏 1-2 个次重点 | 同左 |
| 辩证 | 建议有，不强制 | 不做要求 |

### 第三档

英一和英二第三档**要求基本一致**（都是"基本完成 + 错误不影响整体理解"），两者差异最小。

### 第二档 / 第一档

两者的"语言错误多 / 内容偏题"都属于失分区间，差异不显著（但英一 B 节会因"缺辩证"额外降档）。

## 三、词汇 tier 按考试最低达标要求

| 目标 band | CET-4 | CET-6 | 考研英一 | 考研英二 |
|----------|-------|-------|---------|---------|
| 二档（偏弱）| low | low-mid | low-mid | low |
| 三档（中等）| mid | mid | mid-high | mid |
| 四档（良好）| mid-high | high | high | mid-high |
| 五档（优秀）| high（零星 academic）| high + academic | **academic** | high |

**Skill 应用**：`writing-vocabulary.md` 的自动升档算法需按 `exam_level` 差异化选词。

```python
def min_target_tier(exam_level: str, target_band: int) -> str:
    mapping = {
        ("Postgrad1B", 5): "academic",
        ("Postgrad1B", 4): "high",
        ("Postgrad1A", 5): "high",
        ("Postgrad2B", 5): "high",
        ("Postgrad2B", 4): "mid-high",
        ("Postgrad2A", 5): "high",
        ("Postgrad2A", 4): "mid-high",
        ("CET6", 14): "high",
        ("CET4", 14): "mid-high",
    }
    return mapping.get((exam_level, target_band), "mid")
```

## 四、英一 B 节专属：论述性要求（2022+）

**英二 B 节没有论述性要求**，只要求描述数据 + 简单分析。但英一 B 节从 2022 年起把"议论性"升级为"**论述性**"：

| 要求 | 议论性（英二 B / 旧英一）| 论述性（新英一 B 2022+）|
|------|------------------------|------------------------|
| 观点表达 | 有即可 | 必须有 |
| 让步/反驳 | 无要求 | **必须有**（admittedly / nevertheless）|
| 多维归因 | 无要求 | **必须有**（至少 2 个原因）|
| 对策具体性 | 有即可 | 必须具体可行（不能只 "work hard"）|

**Skill 检查**（仅 `exam_level = Postgrad1B`）：

- 第五档前置门槛：`para3_analytical.is_dialectical == true`
- 单向论证的作文 → 最高只能给**第三档**（因"不满足论述性要求"）

**详细规则**见 [postgrad1b-paragraph-rubric.md](postgrad1b-paragraph-rubric.md) 第四节。

## 五、档内同分差异（经验公式）

基于 2020-2024 年两套卷同题目水平作文的阅卷对照，**同篇作文的档内分差约为 0.5-1 分**：

```
英二分数 ≈ 英一分数 + δ
  δ = +1.0  （五档顶格，英二宽松）
  δ = +0.5  （四档中档）
  δ = 0     （三档中档，两者趋同）
  δ = -0.5  （二档以下，英一反而可能因"论述性不足"降档更多）
```

**Skill 应用**：若用户多选 `exam_levels = [Postgrad1B, Postgrad2B]`，跨考试对比表应反映这一分差规律。

## 六、反馈语气差异（Skill prompt 层）

| exam_level | 反馈语气 | 典型表达 |
|-----------|---------|---------|
| Postgrad1A/1B | 严谨学术 | "论证链条单向，不满足 2022+ 论述性要求。建议加入..." |
| Postgrad2A/2B | 简洁学术 | "数据描述基本到位，但缺少 2020 年低点。建议补充..." |
| CET4 | 通俗教练式 | "这段写得不错！就是 'important' 用太多了，试试换 'crucial'？" |
| CET6 | 学术但友好 | "这段观点清晰，但缺少辩证。加一句 'However,...' 会让论证更严密。" |

## 七、典型误判提醒

Skill 在 `exam_level = Postgrad1B` vs `Postgrad2B` 切换时容易犯的错：

1. ❌ 用英二的宽松标准给英一 B 节打五档 → 错，英一 B 节五档必须辩证 + 复杂结构
2. ❌ 用英一的严苛标准扣英二 B 节的"单向论证" → 错，英二 B 节不做辩证要求
3. ❌ 把英一 B 节的"论述性"降格为英二的"议论性"去评分 → 错（已在 `postgrad-vs-cet.md` 提过）
4. ❌ 词汇目标 tier 不区分英一/英二 → 错，英一五档必须 academic，英二 high 即可
5. ❌ 字数下限搞错：英一 B 节 160，英二 B 节 140（**差 20 词**，容易混淆）
