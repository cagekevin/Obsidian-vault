# Remotion Video Toolkit

## 来源
`C:\Users\xinye\.codebuddy\skills-marketplace\skills\remotion-video-toolkit\`

## 核心能力
- 用 React 组件写视频，输出真实 MP4
- 个性化视频：JSON 数据驱动，一份模板批量出片
- 自动社交媒体视频：拉取实时数据，每日自动渲染
- 动态广告：替换人名、产品图、价格，无限变体
- 数据可视化动画：把仪表盘/KPI 变成可分享视频
- TikTok/Reels 字幕：逐词高亮字幕

## 适用场景（与皮卡斯项目相关）
- 广告视频自动化生成
- 产品展示动画
- 数据驱动的营销视频
