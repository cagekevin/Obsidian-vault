# 内容提炼规则

## ⚠️ 全局防幻觉约束（最高优先级）

以下约束在所有提炼步骤中**强制执行**，不得违反：

1. **不编造内容**：所有核心观点必须来自原文，不得补充、推断或编造原文未提及的信息
2. **摘录忠实性**：关键摘录必须是原文逐字引用，不得改写、意译或合并，用 `>` 引用块标注
3. **不确定时留空**：若原文信息不足，宁可标注 `⚠️ 信息不足` 留空，不得用猜测填充
4. **数据零容忍**：数字、百分比、日期等数据必须与原文完全一致，不得四舍五入或估算
5. **来源可追溯**：每条观点必须能追溯到具体来源条目编号

## 各素材类型处理方式

### URL 链接

1. 抓取页面正文（忽略导航栏、广告、页脚）
2. 识别内容类型：
   - **文章/博客**：提取标题、摘要、正文核心段落
   - **视频页面**（YouTube/B站）：提取标题、描述、评论区精华（如可访问）
   - **GitHub 仓库**：提取 README 核心内容、Stars/语言/用途
   - **学术论文**：提取摘要、结论、关键贡献（建议改用 paper-quick-reader）
   - **新闻/报道**：提取标题、导语、关键事实
3. 若页面需登录或被反爬拦截，标注 `⚠️ 需登录/访问受限`

### PDF 文件

1. 提取全文文本（保留段落结构）
2. 优先读取：摘要/前言、结论/总结、各章节标题
3. 提取图表标题（图注）作为关键数据点
4. 若 PDF 为扫描件，执行 OCR（标注置信度）

### Word / 文本文件

1. 直接读取全文
2. 识别标题层级（H1/H2/H3）作为结构骨架
3. 提取加粗/高亮内容作为重点

### 图片（截图/照片）

1. OCR 识别文字内容
2. 若为图表/数据图：描述图表类型 + 提取数值/趋势
3. 若为截图（网页/文档）：识别主体文字内容
4. 标注 OCR 置信度（低于 80% 时提示用户确认）

### 纯文本片段

1. 直接处理，无需额外读取步骤
2. 若含 URL，自动识别并尝试抓取

## 核心观点提炼标准

每项素材提炼 **≤5 条核心观点**，遵循以下优先级：

| 优先级 | 内容类型 | 示例 |
|--------|---------|------|
| P0 | 核心结论/主张 | "该方法将准确率提升了 15%" |
| P1 | 关键数据/统计 | "2024年全球AI市场规模达 $1.8T" |
| P2 | 方法论/框架 | "采用三步法：分析→设计→验证" |
| P3 | 反例/局限性 | "该方案在低资源场景下效果下降" |
| P4 | 延伸阅读/参考 | "详见 [论文名]" |

---

## 证据可信度分级

每条核心观点必须标注可信度等级（来自 research-synthesize 最佳实践）：

| 等级 | 标记 | 判断标准 |
|------|------|----------|
| **High** | `🟢 High` | 多个独立来源一致；直接观测/实验数据；可复现 |
| **Medium** | `🟡 Medium` | 单一可信来源；间接支持；合理推断 |
| **Low** | `🔴 Low` | 轶事/单一未验证来源；推测性内容 |

输出示例：
```
- 核心观点：该方法将准确率提升了 15% `🟢 High`
- 核心观点：预计 2026 年市场规模翻倍 `🔴 Low`
```

## 注解分类标记

提炼时识别内容中的可操作信号（来自 reader-recap 最佳实践），标记到对应观点上：

| 标记 | 触发条件 | 示例 |
|------|---------|------|
| `❓Question` | 内容提出问题、存在疑点、需要验证 | "该结论是否适用于中文场景？" |
| `✅ TODO` | 需要跟进、尝试、查阅的行动项 | "建议尝试该工具的免费版" |
| `💡 Idea` | 连接多个概念、提出新思路 | "可与项目X的方案结合" |
| `⚡ Disagreement` | 与其他来源观点相悖（含直接矛盾、数据冲突、结论相反、隐性对立） | "与条目#3的结论矛盾；条目#5数据与此处不一致" |

**隐性矛盾识别指引**（不要求来源直接对立，满足以下任一即触发）：
- 同一指标/数据，不同来源给出差异 > 20% 的数值
- 同一方法/工具，一方评价正面、另一方评价负面
- 同一趋势，一方认为"上升"、另一方认为"下降/持平"
- 同一结论，一方有实验支撑、另一方仅为推测且方向相反

带有上述标记的观点，在最终输出的「✅ 可操作项汇总」节中单独列出。

## 关键摘录规则

- 摘录必须是**原文引用**（不改写），用 `>` 引用块标注
- 每项素材最多 2 条摘录
- 摘录长度：20-150 字
- 若原文为英文，保留英文原文，可附中文简译

---

## 标签生成规则

自动生成 2-4 个标签，来源：
1. **领域标签**：`#AI` `#教育` `#金融` `#医疗` 等
2. **类型标签**：`#研究报告` `#教程` `#工具` `#数据集` `#案例`
3. **用途标签**：`#参考资料` `#方法论` `#数据来源` `#工具推荐`
4. **用户自定义标签**（若用户在输入时指定）

标签格式：小写 + 连字符，如 `#machine-learning`（英文）或 `#机器学习`（中文均可）。
