# emoji 热力图渲染规范

> 让 plan-tracker 输出 GitHub 风格的视觉化热力图

---

## 7 天周热力图

### 视觉规范

| 完成度 | Emoji | 含义 |
|-------|-------|------|
| 未打卡 | ⬜ | 当天无任何打卡 |
| 1-25% | 🟨 | 浅色，做了一点点 |
| 26-50% | 🟧 | 中等浓度，完成一半 |
| 51-99% | 🟩 | 深色，几乎全部完成 |
| 100%+ | 🟢 | 满额，超额完成 |
| 计划休息 | ⚪ | 主动休息日（不计入断签）|

### 输出示例

```
本周热力图（5/04 ~ 5/10）：
Mon Tue Wed Thu Fri Sat Sun
🟩  🟢  🟧  🟨  🟩  ⬜  ⚪
完成度：68% · 打卡 5/7 · streak 12 🔥
```

## 实现要点

1. **不要漏天**：即使没打卡也要画 ⬜，保留视觉连续性
2. **周末加强标记**：周六周日如果是计划休息日，用 ⚪ 而不是 ⬜
3. **里程碑高亮**：streak ≥ 7 时在底部加 🔥，≥ 30 加 🏆，≥ 100 加 ⭐
4. **附加文字洞察**：图下面跟 1-2 句趋势分析
5. **响应式宽度**：终端宽度不够时，用紧凑版（每行 7 天）

---

## 使用方

emoji 色阶映射被以下场景共享：

- `dashboard.py` 渲染 SVG 热力图时（数值档位 → 颜色映射的语义来源）
- `today.py` 输出今日完成度小图标时
- `checkin.py --stats` 输出周报时的密度提示

> 命令行入口已统一收敛到 `dashboard.py`：
> `python3 scripts/dashboard.py --plan <PLAN_ID>`
> 双击生成的 `dashboard.html` 即可看到带色阶的近 30 天热力图。
