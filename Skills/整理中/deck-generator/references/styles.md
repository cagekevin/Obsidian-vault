# Deck Style Presets

Available visual style presets for the deck generator. Pass the style name to `--style` when running `generate-deck.py`.

| Style | Description |
|-------|-------------|
| `whiteboard` | Hand-drawn sketch on white. Black ink, orange accents. **(Default)** |
| `corporate` | Navy/white/gold. Clean sans-serif. Professional. |
| `minimalist` | Pure white, electric blue accent. Maximum negative space. |
| `dark-tech` | Near-black background, neon green. Terminal aesthetic. |
| `playful` | Bright pastels, rounded shapes. Modern startup vibe. |
| `editorial` | Black/white with red spot color. Magazine aesthetic. |

## Custom Styles

To use a custom style, describe it in natural language when prompted:

- Colors (background, text, accent)
- Typography feel (serif, sans-serif, mono, handwritten)
- Overall aesthetic (formal, casual, technical, artistic)
- Any reference images or inspirations

The generator will translate your description into Imagen 4.0 prompts that maintain visual consistency across all slides.
