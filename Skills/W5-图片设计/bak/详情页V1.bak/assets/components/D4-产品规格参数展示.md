# D4-产品规格参数展示 骨架规范

## 空间与骨架分布
- [图层 1：装饰文字层] ➔ 位于画面顶部，英文斜体小字
- [图层 2：核心大标题] ➔ 位于画面上方，突出规格主题
- [图层 3：副标题解释] ➔ 位于大标题下方，说明规格内容
- [图层 4：规格展示区] ➔ 以表格或列表形式展示产品规格参数
- [图层 5：补充说明区] ➔ 位于底部，补充说明特殊规格或注意事项

## 微观视觉细节
- **容器质感**：专业清晰风格，表格边框
- **图标与字效**：规格图标，清晰数据字体

## 负空间门禁
- 顶部装饰文字与标题间距：15px
- 标题与副标题间距：10px
- 副标题与展示区间距：20px
- 展示区与补充区间距：15px

## 弹性加载机制
- 若未提供装饰文字层，该层自动隐藏
- 若未提供副标题，该层自动隐藏
- 若未提供补充说明，该层自动隐藏
