# D5-功效数据可视化展示 骨架规范

## 空间与骨架分布
- [图层 1：装饰文字层] ➔ 位于画面顶部，英文斜体小字
- [图层 2：核心大标题] ➔ 位于画面上方，突出功效数据主题
- [图层 3：副标题解释] ➔ 位于大标题下方，说明数据来源
- [图层 4：数据图表区] ➔ 以柱状图、折线图或饼图形式展示功效数据
- [图层 5：数据说明区] ➔ 位于底部，详细说明数据含义与实验条件

## 微观视觉细节
- **容器质感**：专业数据风格，图表边框
- **图标与字效**：数据图标，清晰图表字体

## 负空间门禁
- 顶部装饰文字与标题间距：15px
- 标题与副标题间距：10px
- 副标题与图表区间距：20px
- 图表区与说明区间距：15px

## 弹性加载机制
- 若未提供装饰文字层，该层自动隐藏
- 若未提供副标题，该层自动隐藏
- 若未提供数据说明，该层自动隐藏
