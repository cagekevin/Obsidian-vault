# D2-前后效果对比展示 骨架规范

## 空间与骨架分布
- [图层 1：装饰文字层] ➔ 位于画面顶部，英文斜体小字
- [图层 2：核心大标题] ➔ 位于画面上方，突出对比主题
- [图层 3：副标题解释] ➔ 位于大标题下方，说明对比内容
- [图层 4：对比展示区] ➔ 左右或上下展示使用前后的效果对比
- [图层 5：数据支撑区] ➔ 位于底部，展示对比数据或实验结果

## 微观视觉细节
- **容器质感**：清晰对比风格，对比线分隔
- **图标与字效**：使用前/后标签，清晰对比字体

## 负空间门禁
- 顶部装饰文字与标题间距：15px
- 标题与副标题间距：10px
- 副标题与对比区间距：20px
- 对比区与数据区间距：15px

## 弹性加载机制
- 若未提供装饰文字层，该层自动隐藏
- 若未提供副标题，该层自动隐藏
- 若未提供数据支撑，该层自动隐藏
