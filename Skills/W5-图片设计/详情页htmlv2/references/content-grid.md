## 3. 间距系统（**v0.5.3 新增 · R9-1 格式塔原则**）

### 3.1 间距变量定义

```css
--s-1: 4px;    /* 极小间距：标签内文字与边框 */
--s-2: 8px;    /* 组件内元素间距：数字与小字 */
--s-3: 16px;   /* 组件内主间距：行内元素 */
--s-4: 24px;   /* 组件间间距：行与行之间 */
--s-5: 32px;   /* 区块间距：标题区与正文区 */
--s-6: 48px;   /* 屏内主间距：mod-body 边距 */
--s-7: 80px;   /* 屏间垂直间距：mod-top-c padding-top */
```

### 3.2 格式塔亲密性原则（**R9-1 自检铁律**）

> **核心铁律**：**组件内间距永远小于组件间间距**。
> 否则用户视觉上无法区分"哪些元素是一组"。

#### 3.2.1 数学约束

```
组件内间距 max ≤ 组件间间距 min
   s-3 (16px)        s-4 (24px)
```

**反例（违规）**：
- 行内元素间距 24px，行间 16px → ❌ 用户看不清"行"的边界
- 标签内文字 8px，标签间 4px → ❌ 标签糊在一起

**正例（合规）**：
- 数字/小字间距 8px，行间距 16px → ✅ 行内紧凑，行间清晰
- 数字 + 标签 + 描述间距 16px，组间 24px → ✅ 组内紧凑，组间清晰

#### 3.2.2 各模板的间距配方（v0.5.3 强制）

| 模板 | 内部 max 间距 | 外部 min 间距 | 来源 |
|---|---|---|---|
| `.dp2-list-row` | s-2 (8px) | s-4 (24px) | 行内紧凑，组间清晰 |
| `.dp2-step-flow` 子节点 | s-3 (16px) | s-5 (32px) | 节点内紧凑，节点间清晰 |
| `.dp2-card-compare` 卡片内 | s-3 (16px) | s-5 (32px) | 卡片内紧凑，卡片间清晰 |
| `.dp2-wrap-block` 行内 | s-2 (8px) | s-4 (24px) | 行内紧凑，行间清晰 |
| `.dp2-feature-orb` 圆 + 文 | s-3 (16px) | s-5 (32px) | 圆+文紧凑，多个 orb 清晰 |
| `.dp2-data-strip` 条内 | s-3 (16px) | s-4 (24px) | 数字+标签紧凑，条间清晰 |

### 3.3 留白节奏（V3 验证过）

- **屏内主间距** = s-6 (48px) = 屏内边界
- **屏顶标题区** = s-7 (80px) padding-top（让标题"呼吸"）
- **屏间描边** = 1px solid var(--line) + 屏内 s-6 留白（v0.5 R8 3 态背景已加描边豁免）

### 3.4 R9-1 自检（人工 + 自动化占位）

**人工 checklist**（每个新屏）：
- [ ] 找 1 个组件（`.dp2-xxx`），**量出**内部间距 px
- [ ] 找下一个组件，**量出**组件间间距 px
- [ ] 前者 < 后者？→ PASS
- [ ] 找屏内所有间距对，至少 1 对满足 → PASS

**自动化占位**（v0.6 写脚本）：
- 解析 CSS 文件，提取每个 `.dp2-xxx` 内的 `gap` / `padding` 值
- 提取组件间 `margin-bottom` 值
- 对比数学约束 `内部 ≤ 外部`
- 0 违规 = PASS

**v0.5.3 暂以人工清单为主**（脚本等 v0.6）。

## 4. 颜色系统（粗略索引，详见各主题 CSS）

### 4.1 三色系 token

| token | 朝圣黎明 | 玫瑰矿泉 | 黄昏胶囊 | HK 黑金 |
|---|---|---|---|---|
| `--bg` | 米色 | 浅粉 | 深紫 | 深黑 |
| `--bg-soft` | 浅米 | 浅粉 | 暗紫 | 深棕 |
| `--card-bg` | 暖白 | 浅粉 | 紫黑 | 黑金纸 |

| `--accent` | 金 | 玫红 | 金 | 亮金 |

### 4.2 用色铁律

- 3 色系（主色 / 浅版 / 重音）**不混用**主题
- 1 屏内最多 1 个重音区（`.dp2-section-dark` 或 `.deco-line` 金色）
- 不允许 inline `style="color: #xxx"`（用 token）

## 5. 字体系统

| 用途 | 字体 | 引用 |
|---|---|---|
| 戏剧性展示 | Fraunces (Serif) | `font-display` |
| 中文展示 | Noto Serif SC | `font-serif` |
| 功能 | Inter Tight (Sans) | `font-sans` |
| 中文功能 | Noto Sans SC | `font-sans-cjk` |
| 数据 | IBM Plex Mono | `font-mono` |

**字体隔离铁律**（R5）：
- 戏剧性（display / h1-sm）**只用** Serif
- 功能（body / lead / meta）**只用** Sans
- 数字 / 装饰英文**只用** Mono
