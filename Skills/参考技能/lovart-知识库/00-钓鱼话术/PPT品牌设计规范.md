## 一、前期准备阶段
### 1.1 资料收集与梳理
**品牌基础资料清单：**
- 品牌故事/使命愿景价值观
- 目标受众画像（年龄、职业、消费习惯）
- 竞品分析报告
- 现有品牌资产（如有旧版VI需评估）
- 行业属性与调性定位
**关键问题确认：**
```
- 品牌核心关键词是什么？（如：专业、温暖、创新、高端）
- 主要应用场景有哪些？（线上/线下/印刷/数字媒体）
- 有无必须规避的元素或色彩？
- 目标市场的文化偏好？
```

## 二、视觉规范系统核心模块
### 2.1 色彩系统（Color System）
**主色板构建逻辑：**
| 层级 | 命名规范 | 用途 | 占比 |
|------|---------|------|------|
| Primary | 品牌主色 | Logo、重点强调 | 60% |
| Secondary | 辅助色 | 分类区分、层次感 | 30% |
| Accent | 点缀色 | CTA按钮、高亮元素 | 10% |
| Neutral | 中性色 | 文字、背景、边框 | 基础层 |
**System Prompt 级别的色彩规范写法：**
```
COLOR PALETTE SPECIFICATION:
Primary Brand Color: #1E3A5F (Deep Navy Blue)
- Usage: Headers, primary buttons, key graphics
- Psychology: Trust, professionalism, stability
- Contrast ratio on white: 7.2:1 (AAA compliant)
Secondary Colors:
- Warm Sand: #D4A574 (human touch, approachability)
- Sage Green: #87A878 (growth, sustainability)
Neutral Scale (Grayscale):
- Text Primary: #1A1A1A
- Text Secondary: #666666
- Border/Divider: #E5E5E5
- Background: #FAFAFA
Functional Colors:
- Success: #2D8659
- Warning: #E8913A
- Error: #C94C4C
```
**PPT呈现方式：**
- 色块展示 + CMYK/RGB/HEX/Pantone四色值
- 色彩搭配示例（主色+辅助色的实际组合）
- 色彩使用比例饼图
- 色彩应用场景对照图
---
### 2.2 字体系统（Typography）
**字体选择原则：**
```
FONT HIERARCHY SYSTEM:
Display/Headline Font: [字体名]
- 风格特征：现代几何/人文衬线/手写体等
- 使用场景：大标题、Hero区域
- 字重范围：Light(300) - Bold(700)
Body Font: [字体名]
- 风格特征：高可读性、中性
- 使用场景：正文、说明文字
- 最小字号：12pt (印刷) / 14px (屏幕)
Accent Font: [字体名] (可选)
- 使用场景：引用、标注、特殊强调
```
**字体层级规范表：**
| 层级 | 字号 | 字重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| H1 | 48pt | Bold | 1.2 | -0.02em | 页面主标题 |
| H2 | 32pt | Semibold | 1.3 | -0.01em | 章节标题 |
| H3 | 24pt | Medium | 1.4 | 0 | 小节标题 |
| Body | 16pt | Regular | 1.6 | 0 | 正文 |
| Caption | 12pt | Regular | 1.5 | 0.02em | 注释说明 |
**PPT呈现要点：**
- 字体家族展示（全字重预览）
- 中英文字体搭配方案
- 字体使用禁忌（如：禁止拉伸、禁止描边）
- 替代字体方案（系统字体备选）
---
### 2.3 布局网格系统（Grid System）
**网格构建逻辑：**
```
GRID SPECIFICATION:
- Base Unit: 8px (所有间距以此倍数计算)
- Column Count: 12列 (灵活适配多种布局)
- Gutter Width: 24px (列间距)
- Margin: 48px (页面边距)
- Max Content Width: 1200px
Spacing Scale:
4px | 8px | 16px | 24px | 32px | 48px | 64px | 96px | 128px
(xs)  (sm)   (md)   (lg)   (xl)   (2xl)  (3xl)  (4xl)  (5xl)
```
**常见布局模板：**
- 全幅Hero布局
- 2栏图文布局（6:6 / 4:8 / 8:4）
- 3栏卡片布局（4:4:4）
- 4栏网格布局（3:3:3:3）
**PPT呈现方式：**
- 网格线框图叠加在真实页面上
- 安全区域标注（Logo最小留白空间）
- 响应式断点说明（Desktop/Tablet/Mobile）
---
### 2.4 Logo使用规范（Logo Guidelines）
**Logo系统架构：**
```
LOGO VARIANTS:
1. Primary Logo (主Logo)
   - 横版/竖版两种形式
   - 全彩版本（标准色）
2. Secondary Logo (辅助Logo)
   - 单色版本（黑/白）
   - 反白版本（深色背景使用）
3. Icon/Mark (图形标志)
   - 独立图形元素
   - 小尺寸场景使用（favicon、头像）
4. Clear Space (安全空间)
   - 以Logo高度为X单位
   - 四周最小留白：0.5X
```
**Logo使用禁忌清单：**
- ❌ 拉伸变形
- ❌ 旋转倾斜
- ❌ 改变颜色比例
- ❌ 添加阴影/描边/渐变（除非特别版本）
- ❌ 置于复杂背景上（降低辨识度）
- ❌ 小于最小尺寸（建议不小于24px高度）
**PPT呈现结构：**
1. Logo家族全览（所有变体）
2. 正确/错误使用对比（Do's & Don'ts）
3. 最小尺寸标注
4. 安全空间示意图
5. 背景适配示例（深浅背景上的应用）

## 三、System Prompt 级别的完整规范写法
当你需要让AI或设计团队严格执行品牌规范时，可以使用以下结构化prompt：
```
BRAND IDENTITY SYSTEM PROMPT - [品牌名]
=== CORE ATTRIBUTES ===
Brand Personality: [专业/亲和/创新/可靠等]
Visual Tone: [现代极简/温暖人文/科技未来等]
Target Audience: [人群描述]
=== COLOR SYSTEM ===
[粘贴2.1的色彩规范]
=== TYPOGRAPHY SYSTEM ===
[粘贴2.2的字体规范]
=== GRID & SPACING ===
[粘贴2.3的网格规范]
=== LOGO USAGE ===
[粘贴2.4的Logo规范]
=== IMAGE STYLE ===
Photography: [商业摄影/生活化/插画风格等]
Illustration: [线性图标/扁平插画/3D等]
Icon Style: [描边/填充/双色等]
=== TONE OF VOICE ===
Language Style: [正式/轻松/专业/活泼]
Key Phrases: [品牌常用语/禁用语]
=== APPLICATION RULES ===
- 所有标题使用Primary Color
- 正文行高不低于1.5
- 图片需统一调色（降低饱和度+暖色调）
- CTA按钮使用Accent Color
- 留白遵循8px网格系统
```

## 四、PPT提案结构建议
### 完整提案框架：
| 章节 | 内容 | 页数建议 |
|------|------|---------|
| 1. 项目背景 | 品牌现状、挑战、目标 | 2-3页 |
| 2. 策略洞察 | 竞品分析、用户洞察、定位策略 | 3-5页 |
| 3. 视觉概念 | 情绪板、设计方向、关键词 | 2-3页 |
| 4. Logo设计 | 概念阐述、方案展示、演变过程 | 3-5页 |
| 5. 色彩系统 | 色板、心理学依据、应用场景 | 2-3页 |
| 6. 字体系统 | 字体家族、层级规范、排版示例 | 2-3页 |
| 7. 图形元素 | 图标、插画、图案、摄影风格 | 2-3页 |
| 8. 应用展示 | 名片/网站/包装/空间等mockup | 4-6页 |
| 9. 使用规范 | 禁忌清单、下载资源、联系方式 | 1-2页 |
