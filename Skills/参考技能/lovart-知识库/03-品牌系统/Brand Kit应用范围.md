# Brand Kit 应用范围与使用方式

## 适用范围
| 场景 | 效果 |
|------|------|
| 图片生成 | ✅ 直接有效，Logo、配色、字体规范都能应用 |
| 视频生成 | ✅ 有效，主要用于保持视觉风格一致性（色调、Logo 样式） |

## 引用方式
对话中直接说：
- "使用我的 Brand Kit `kit_id` 来生成..."
- "引用 Brand Kit `xxx` 的风格规范"
- "--kit=`xxx`"

## Brand Kit vs Prompt 直接描述

| 方式 | 适用场景 | 效果 |
|------|----------|------|
| Brand Kit | 长期项目、多次生成、团队协作 | 一致性最好，无需重复描述 |
| Prompt 直接描述 | 一次性任务、快速测试 | 容易遗漏细节，每次需完整描述 |

## 不用 Kit 时的 Prompt 写法示例
```
品牌：{品牌名}
Logo：{描述}
主色调：{色值}
字体：标题用无衬线粗体，正文用优雅衬线体
风格：温暖自然、现代简约
```
