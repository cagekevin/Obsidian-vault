# Lovart 拼图碎片 — Skill 创建

## Lovart 回复

### Skill 是什么

Skill = 可复用的创作模板，将创意偏好、风格设定和生成规则封装起来。

| 普通对话 | Skill |
|---------|-------|
| 每次重新描述风格 | 一次配置永久复用 |
| 风格容易漂移 | 输出高度一致 |
| 个人使用 | 可分享给团队或社区 |

### 8 步创建流程

**第一步：准备材料**
- 触发词（如 `tech-minimal`）
- 参考图 3-5 张
- 风格描述文字
- 输出规格

**第二步：进入 Skill Studio → 新建**

**第三步：基础信息**
- 名称、触发词、描述、分类

**第四步：核心参数**
- 风格关键词 + 避免元素
- 输出规格（比例、尺寸、推荐模型）
- 上传参考图（系统分析色调、光影、构图）

**第五步：变量定义**
```jinja
{{product}} — 文本，产品名称
{{angle}} — 选项，拍摄角度
{{lighting}} — 选项，光线强度
```

**第六步：提示词模板**
```
{{product}} 的产品摄影，{{angle}}角度拍摄...
```

**第七步：测试调优**

**第八步：发布（私有/社区）**

### Skill 结构
```
Skill = 触发词 + 参数预设 + 风格规则 + 输出模板
```
