### **三、风格统一技巧 🎨**
#### **1. 色彩系统统一**
- **主色：** 红石榴红 (#C73E3A) + 金色点缀 (#D4AF37)
- **辅助色：** 纯白 (#FFFFFF) + 米色 (#F5F5DC)
- **点缀色：** 深绿 (#2C5F2D) 代表天然
**Prompt关键词：**
```
red and gold color scheme, warm tone palette, consistent color harmony
```
#### **2. 光线风格统一**
- 所有图片使用"柔和顶光"或"自然光"
- 避免硬阴影
**Prompt关键词：**
```
soft diffused lighting, natural daylight, gentle top light, no harsh shadows
```
#### **3. 背景风格统一**
- 产品图：纯白背景
- 场景图：米色/大理石纹理
- 信息图：浅色渐变
**Prompt关键词：**
```
pure white background / marble texture / soft gradient background
```
#### **4. 摄影风格统一**
- 统一使用"专业商业摄影"风格
- 景深效果一致（f/2.8-f/4）
**Prompt关键词：**
```
professional commercial photography, shallow depth of field, luxury product style
```
#### **5. 排版风格统一**
- 字体：现代无衬线字体
- 布局：留白充足，呼吸感强
- 图标：统一线性风格
**Prompt关键词：**
```
modern minimalist layout, clean typography, ample white space, linear icon style
```
