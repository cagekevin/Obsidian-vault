# PPT 品牌提案实战指南

> 基于 Lovart 平台的最佳实践

## 需求描述技巧

| 要点 | 说明 |
|------|------|
| 明确主题 | 一句话概括 PPT 核心目的 |
| 结构化内容 | 分点列出每页的关键信息 |
| 指定页数 | 建议 8-15 页，信息密度适中 |
| 风格关键词 | 用 3-5 个形容词描述视觉调性 |

## 内容量控制
- 每页 1 个核心观点 + 2-3 个支撑点
- 标题 ≤ 10 字，正文每行 ≤ 15 字
- 用图/表代替大段文字

## 风格统一方法
- 提供风格参考图（品牌 VI、竞品 PPT）
- 指定配色方案（主色+辅助色）
- 统一字体层级（标题/正文/注释）

## 护肤品品牌提案 PPT 示例（12页）

### 完整 Prompt 模板

```
主题：{品牌名} 品牌提案PPT

品牌背景：
- 品牌名：{名称}
- 定位：{一句话定位}
- 核心理念：{品牌理念}
- 主打产品：{产品线}

PPT结构（共{页数}页）：
1. 封面页
2. 品牌背景（市场痛点+机会点）
3. 目标人群画像
4. 品牌定位
5. 品牌视觉系统（Logo/配色/辅助图形）
6. 产品矩阵
7. 包装设计
8. 渠道策略
9. 营销规划
10. 竞品分析
11. 财务预测
12. 结尾页

风格要求：
- 整体风格：{极简/清新/高端/科技感}
- 配色：{主色}
- 字体：{标题/正文}
- 图片风格：{自然光/产品特写/场景}
```

## 操作流程

```
步骤1：准备素材（Logo、产品图、风格参考图）
步骤2：调用 generate_slides，传入完整 prompt + 页数
步骤3：查看生成结果，检查内容结构和视觉风格
步骤4：针对单页提出修改意见，重新生成
步骤5：导出 PPT 文件
```
