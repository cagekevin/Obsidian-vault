## 一、基本信息（考试范围）

| 项目 | 英语（一） | 英语（二） |
|---|---|---|
| 适用对象 | 学硕 & 部分专硕 | 多数专硕 |
| 写作总分 | **30 分** | **25 分** |
| A 节（应用文） | **10 分**，约 100 词 | **10 分**，约 100 词 |
| B 节（短文） | **20 分**，**160–200 词**，**图画作文**（描述性/叙述性/说明性/**论述性**） | **15 分**，**约 150 词**，**图表作文**（数据解读 + 趋势分析） |
| A 节题型 | 书信 / 通知 / 告示 / 纪要（2022 改版后）| 同左 |

### A / B 节评分侧重

- **A 节（应用文）**：**信息点覆盖 · 内容组织 · 语言准确性 · 格式与语域恰当**
- **B 节（短文）**：**内容完整性 · 组织连贯性 · 语法/词汇多样性 · 语言准确性**

### 重要特殊规则

> **A 节独有**：允许使用提示语中的个别关键词/词组，**但使用提示语中原句将被扣分**。
>
> Skill 必须在 A 节批改时**把 Directions 作为输入**，并做整句照搬检测。

## 二、五档评分标准（A / B 共用描述，仅分数段不同）

### 第五档

| 节 | 分数段 |
|---|--------|
| A 节 | 9–10 分 |
| B 节（英一） | 17–20 分 |
| B 节（英二） | 13–15 分（近似）|

> **很好地完成了试题规定的任务**：
>
> - 包含所有内容要点；
> - 使用丰富的语法结构和词汇；
> - 语言自然流畅，语法错误极少；
> - 有效地采用了多种衔接手法，文字连贯，层次清晰；
> - 格式与语域恰当贴切。
>
> 对目标读者完全产生了预期的效果。

### 第四档

| 节 | 分数段 |
|---|--------|
| A 节 | 7–8 分 |
| B 节（英一） | 13–16 分 |
| B 节（英二） | 10–12 分（近似）|

> **较好地完成了试题规定的任务**：
>
> - 包含所有内容要点，允许漏掉一、两个次重点；
> - 使用较丰富的语法结构和词汇；
> - 语言基本准确，只有在试图使用较复杂结构或较高级词汇时才有个别错误；
> - 采用了适当的衔接手法，层次清晰，组织较严密；
> - 格式与语域较恰当。
>
> 对目标读者产生了预期的效果。

### 第三档

| 节 | 分数段 |
|---|--------|
| A 节 | 5–6 分 |
| B 节（英一） | 9–12 分 |
| B 节（英二） | 7–9 分（近似）|

> **基本完成了试题规定的任务**：
>
> - 虽漏掉一些内容，但包含多数内容要点；
> - 所使用的语法结构和词汇能满足任务的需求；
> - 存在一些语法及词汇错误，但不影响整体理解；
> - 采用了简单的衔接手法，内容基本连贯，层次基本清晰；
> - 格式与语域基本合理。
>
> 对目标读者基本产生了预期的效果。

### 第二档

| 节 | 分数段 |
|---|--------|
| A 节 | 3–4 分 |
| B 节（英一） | 5–8 分 |
| B 节（英二） | 4–6 分（近似）|

> **未能按要求完成试题规定的任务**：
>
> - 漏掉或未能有效阐述一些内容要点，写了一些无关内容；
> - 语法结构单调、词汇项目有限；
> - 有较多语法结构或词汇方面的错误，影响了对写作内容的理解；
> - 未采用恰当的衔接手法，内容缺少连贯性；
> - 格式和语域不恰当。
>
> 未能清楚地传达信息给读者。

### 第一档

| 节 | 分数段 |
|---|--------|
| A 节 | 1–2 分 |
| B 节（英一） | 1–4 分 |
| B 节（英二） | 1–3 分 |

> **未完成试题规定的任务**：
>
> - 明显遗漏主要内容，且有许多不相关的内容；
> - 语法项目和词汇的使用单调、重复；
> - 语言错误多，有碍读者对内容的理解，语言运用能力差；
> - 未使用任何衔接手法，内容不连贯，缺少组织、分段；
> - 无格式与语域概念。
>
> 未能传达信息给读者。

### 零档（0 分）

> 所传达的信息或所用语言太少，无法评价；内容与要求无关或无法辨认。

## 三、评分原则（官方原文）

1. A / B 两节考查要点不同，但**写作能力基本要求相同**，一般评分标准都适用；评分时有不同侧重点。
2. **先根据内容和语言确定档次，再以该档的要求给分**；**档内有 1–3 分的调节分**（注意：**不是 CET 的 ±1，而是 1–3 的调节区间**）。
3. 字数不符（A 节 ~100；英一 B 160–200；英二 B ~150）**酌情扣分**。
4. 拼写/标点按**对交际的影响程度**扣；英美拼写均可接受。
5. **书写较差影响交际**——降一档。

### 2022 年起的重要变化

- A 节应用文题型：私人和公务信函、备忘录、报告 → **书信 / 通知 / 告示 / 纪要**
- A 节评分表述：
  - "内容的组织" → "**文章的组织连贯性**"
  - "格式与语域的恰当" → "**文体格式和语体的恰当性**"
- B 节（英一）考查目标：议论性文章 → **论述性文章**（更强的逻辑论证要求）

## 四、隐含评分观察点（5 维诊断框架，不出独立分）

1. **任务完成度**（内容要点覆盖）
2. **语法结构与词汇**（丰富度 + 准确度）
3. **语言准确性**（错误密度 + 严重程度）
4. **衔接与连贯**（衔接手法 + 层次 + 分段）
5. **格式与语域**（文体 + 正式度 + 读者意识）

> Skill 输出的 `dimension_diagnosis` 字段对这 5 维做**定性描述**。考研情境下比 CET 多出"格式/语域"维度，这一维在 CET 是隐含的、在考研 A 节是**独立的扣分点**。

## 五、A 节 "Directions 原句照搬" 检测规则

**官方扣分依据**：

> 允许使用提示语中出现过的**个别关键词或词组**，但使用提示语中出现过的**句子**将被扣分。

Skill 实现：

1. A 节批改**必须**把 Directions 作为输入（见 `SKILL.md` 输入契约里的 `prompt` 字段）。
2. 用"长 n-gram 重合"检测：
   - 连续 ≥ 8 个单词与 Directions 原句一致 → 判定为**整句照搬**
   - 连续 4–7 个单词一致 → 判定为**词组重复**，不扣分但提醒
3. `issues` 中标记：
   ```json
   {
     "severity": "warning",
     "type": "copy_from_directions",
     "location": "¶1 第 1 句",
     "original": "...",
     "matched_directions": "...",
     "reason": "与 Directions 原句连续 12 词一致，按考研大纲 A 节规则将被扣分"
   }
   ```

## 六、档次边界判定要点（考研情境）

| 边界 | 关键判定 |
|------|---------|
| 五档 vs 四档 | "所有内容要点" vs "允许漏一两个次重点"；"极少错误" vs "复杂结构才有个别错误" |
| 四档 vs 三档 | "**较**丰富的语法词汇" vs "**满足**任务需求"；错误"不影响整体理解"是三档底线 |
| 三档 vs 二档 | 是否"基本完成任务"（三档）vs "未能完成任务"（二档）；衔接手法是否存在 |
| 二档 vs 一档 | "漏掉部分要点" vs "明显遗漏主要内容且有许多不相关内容" |

## 七、分数计算 & final_score 计算

```
考研英语一 A 节：final_score ∈ {0..10}
考研英语一 B 节：final_score ∈ {0..20}
考研英语二 A 节：final_score ∈ {0..10}
考研英语二 B 节：final_score ∈ {0..15}

整数 或 0.5 步进（阅卷允许半分）
```

> **与 CET 的差异**：考研允许 0.5 分调节；CET **严禁半分**。Skill 逻辑里必须按 `exam_level` 区分。

## 八、书写差扣档的 AI 免责

AI 无法判断实际书写质量。Skill 在输出里**必须包含免责声明**：

```
⚠️ 本批改基于输入文本。若为手写稿，实际书写质量（工整度/可识别性）可能触发"降一档"规则，请人工复核。
```
