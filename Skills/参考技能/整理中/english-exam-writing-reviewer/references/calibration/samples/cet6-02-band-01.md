Nowdays, internt is very popular for everyone. Many people use it to read book, paper, news, and so on. But some poeple also like tradional reading way.

Internt have many good and also bad. Good is, we can read fast. Bad is, sometime information not correct, we be confused. And peoples eyes was hurted very serious. I has a friend, who is read phone every day, eye very bad, can not see blackboard. This is very serious problem in our life.

Book also have good, it very kindly to eye. But it is heavy, hard carry. We shoud choice according own stituation. Some one like internt, some one like book, both of them are a kind of way to learn new things. No matter what way, reading is important thing in our life, we should do it always. If we don't read, we become stupid people, not success in future.

# 人工阅卷批注

## 隐含维度诊断（CET-6 2 档基准）

- **条理不清、思路紊乱**：¶2 先说优点一句 + 缺点一句 + 跳到朋友的眼睛 + 跳到"serious problem in our life"，论证线索完全断裂
- **语言支离破碎**：约 14 个句子中，≥ 11 句存在严重结构错误
- **大部分句子都有严重错误**：主谓一致、时态、冠词、用词、拼写全面失控
- **对话意图勉强可辨认**：能看出是在讨论"互联网 / 书本 / 阅读"的主题，但立场、论证、结构全部溃散
- **拼写失控**：`Nowdays` / `internt`（×3） / `poeple` / `peoples` / `tradional` / `shoud` / `choice`（作动词用）/ `stituation`

## 严重错误（critical，占比 ≥ 70%）

1. `Nowdays`（拼写）+ `internt`（拼写）
2. `use it to read book, paper, news`（可数名词缺单复数变化）
3. `poeple` / `tradional reading way`（拼写 + 冠词）
4. `Internt have many good and also bad`（主谓 + 结构错乱）
5. `Good is, we can read fast. Bad is, sometime information not correct, we be confused`（句号逻辑断 + 缺动词 + 缺冠词）
6. `peoples eyes was hurted very serious`（名词所有格 + 被动 + 时态 + 副词）— **四重 critical 叠加**
7. `I has a friend, who is read phone every day`（主谓 + 时态 + 冠词）
8. `eye very bad, can not see blackboard`（缺主谓 + 冠词）
9. `This is very serious problem`（缺冠词）
10. `Book also have good, it very kindly to eye`（主谓 + 缺系动词 + 词性错）
11. `it is heavy, hard carry`（run-on + 缺助动词 + 缺不定式）
12. `We shoud choice according own stituation`（拼写 + 词性 + 介词）
13. `Some one like internt, some one like book`（拼写 + 主谓）
14. `No matter what way, reading is important thing in our life, we should do it always`（冠词 + run-on + 语序）
15. `If we don't read, we become stupid people, not success in future`（时态 + 词性 + 冠词 + 缺主语）

## 拼写错误

- `Nowdays` → `Nowadays`
- `internt` → `Internet`（×3）
- `poeple` → `people`
- `tradional` → `traditional`
- `shoud` → `should`
- `stituation` → `situation`
- `peoples` → `people's`

## 为什么 2 分不是 5 分

| 判定点 | 本文表现 | 5 档描述 |
|-------|---------|---------|
| 基本切题 | ❌ 立场无、建议反题（"If we don't read, we become stupid people, not success in future" 逻辑错位）| 基本切题 |
| 论证结构 | 段落划分勉强存在但段内跳跃严重 | 可辨认结构 |
| 句子完整度 | ≥ 70% 句子严重残缺或缺核心成分 | 多数句子完整 |
| 拼写 | ≥ 7 处基础词汇拼写错 | 5 档不应高频拼写错 |

→ 4 项全部指向 2 档。

## 为什么 2 分不是 0 分

- 虽然条理不清，但**能看出是在讨论"internet / book / reading"的主题**，未完全离题
- 不是白卷、不是仅抄题目、不是仅孤立词堆砌
- 字数达到 160+（CET-6 要求 150 词下限，字数达标）
- 因此判 2 档（1-3 分），而非 0 分

## 升档路径（CET-6 2 → 5）

1. **先保证每个句子完整**：主语 + 谓语 + 宾语齐全，系动词 `is / are` 不能省
2. **修正高频拼写错**：`Nowadays` / `Internet` / `people` / `traditional` / `should` / `situation`（这 6 个词值得背会）
3. **立场一致**：选一边（支持/反对/有条件支持），全文围绕同一立场
4. **至少 3 个可识别段落**：引入（表态）/ 论证（1-2 个理由）/ 结论
5. **消灭最明显的主谓一致错误**（`is/are` / `has/have` / 第三人称单数 `-s`）
6. **避免 run-on sentence**：一句话一个核心意思，不用逗号串联
7. **CET-6 基础预警**：本档提示考生词汇/语法基础严重不足，建议回到 CET-4 基础训练
