# 我的第二大脑

Obsidian vault 管理我的知识体系、技能库、项目文档和个人笔记。

## 结构

- `Context/` — 我是谁（身份、目标、品牌、风格）
- `Skills/` — 我的工作方法论（AI 技能定义）
- `Tools/` — 实用脚本
- `Reading/` — 读书笔记
- `Clippings/` — 网页剪藏
- `Daily Notes/` — 每日工作日志
- `Projects/` — 项目文档

## 使用

首次：`uv pip install -r requirements.txt`  
推送：`bash sync.sh`（自动扫描技能 + commit + push）
