# 02. 公众号头图 (wechat-cover)

> **优先质量维度**: Visual Appeal + Editorial Sense(编辑感 + 高级留白)
> **默认比例**: 2.35:1(900×383)
> **核心转化驱动**: 极端横幅版面(高 ≈ 宽 1/2.35)+ 编辑式留白 + 标题超链接区高识别度

---

## 一、必出元素清单

> 信息递进流:**主视觉建立调性 → 标题超链接区抓核心信息 → 留白透气**
> 核心逻辑:公众号信息流是文字超链接 + 头图缩略图,头图必须在 320×136 缩略尺寸下还能传达调性,标题区不被裁切。

| # | 必出元素 | 视觉权重区 | 占比 | 位置 | 信息职责 |
|---|----------|-----------|------|------|---------|
| 1 | 主视觉 / 编辑插画 | Heavy | 50-60% | 左 1/3 或中央 | 调性奠基 — 杂志式封面感、定位文章主题 |
| 2 | 标题 / 品牌文字(如嵌入) | Medium | 30-40% 宽 | 右 1/3 或上沿 | 关键超链接区 — 缩略图也要识别 |
| 3 | 留白区 | Medium | 20%+ | 周围 / 一侧 | 高级感呼吸 — 编辑式排版核心 |
| 4 | 装饰线 / 品牌色块(可选) | Light | < 5% | 边缘 | 调性辅助 — 杂志条 / 品牌色带 |

---

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 编辑式杂志封面 / 高级感留白 / 严肃议题 / 知识·财经·科技专栏 |
| **色温** | 中性偏冷(4000-5000K)— 体现专业、克制 |
| **背景** | 米白 / 浅灰 / 单色块 + 细节肌理;**不**用满图重背景(缩略图糊成一团) |
| **滤镜** | 高级灰 / 杂志暖白 / 极简单色;不用浓重滤镜 |
| **质感** | 编辑摄影 / 极简插画 / 几何图形;不要过多写实细节(缩略图看不清) |

---

## 三、版式构图要求

| 构图方式 | 适用场景 | 焦点位置 |
|---------|---------|---------|
| 左图右文(黄金分割) | 人物访谈 / 专题报道 | 焦点左 1/3,文字右 2/3 |
| 中央焦点 + 双侧留白 | 单议题 / 大事件 | 焦点中央 ≤ 50%,两侧透气 |
| 满版插画 + 标题块叠加 | 创意议题 / 设计稿 | 插画铺满,标题块右下 / 左上 |
| 文字主导(大字标题) | 财经 / 数字议题 | 大字标题为主,辅以小图标 |

**字体位置规则**:
- 极端横幅(高 < 宽 1/2),嵌入文字字号必须撑住缩略图(320×136 还能看清)
- 标题集中在左 / 右 1/3,避免横铺到边缘被裁
- 上下各保 8% 缓冲带,公众号缩略图轻微裁切

---

## 四、信息层级映射

| 层级 | 视觉权重 | 字号建议(基于 900×383 画布) |
|------|---------|----------------------|
| 主标题 | Heavy | 60-90pt(占宽 35%+) |
| 副标题 / 品牌 | Medium | 28-40pt |
| 装饰元素 / 小标 | Light | 14-22pt |

---

## 五、文字渲染策略(按 Iron Law #3)

| 文字情况 | 推荐策略 + 模型 |
|---|---|
| 中文标题 ≤ 4 字 | A 嵌入:qwen / nano_banana 直接渲染 |
| 中文标题 5-15 字 | **B 纯底图**:kontext / midjourney 出底图 + 后期加文字(默认) |
| 中文标题 > 15 字 | **B 纯底图**:必须后期加文字 |
| 中英混排 / 全英 | **B 纯底图**:openai / kontext 出底图 + 后期加文字 |

**B 策略 prompt 写法**:`editorial magazine cover style, no text, no words, leave clean negative space on right one-third of canvas (about 35% width) for headline overlay, 2.35:1 wide aspect ratio`

---

## 六、必填项(公众号头图 Brief 缺一不可)

不写这些 → sub-agent 凭训练先验补全 → 风格违和或缩略图失败:

- **栏目类型**:知识 / 财经 / 科技 / 文化 / 时尚 / 商业 / 时评 ...(决定调性)
- **主标题完整文字** + 是否嵌入头图(默认 B 策略)
- **品牌色 / 主色调**(公众号品牌资产,决定整篇调性延续)
- **杂志参照**(New Yorker / 三联生活 / 经济学人 / 视觉中国 等)
- **是否有人物**:Y → 人像锁/ N → 静物 / 几何插画

## 七、易错点(Watch For)

- **缩略图糊成一团** — 满版重背景 + 细节过多,320×136 缩略图毫无识别度 → 简化背景,主视觉占比降到 50%
- **横幅比例搞错** — 出 16:9 或 4:3 当公众号头图,导致上下被裁、关键信息丢 → 严格 2.35:1(900×383)
- **嵌入字过小** — 标题 < 50pt(基于 900×383)→ 缩略图根本看不清 → 嵌入字号 ≥ 60pt
- **元素铺到边缘** — 文字 / 焦点贴近 4 边 → 公众号缩略图轻微裁切就丢失 → 4 边各保 5%+ 缓冲
- **滤镜过重** — 浓郁滤镜(油画 / 复古胶片)→ 编辑感丢失 → 改用高级灰 / 杂志暖白
- **背景纯色 + 无装饰** — 单调到像 PPT → 加细节肌理 / 几何线条 / 微噪点
