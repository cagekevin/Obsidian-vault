# 06. 活动海报 (event-poster)

> **优先质量维度**: Information Hierarchy + Print-ready(信息层级清晰 + 可印刷)
> **默认比例**: 3:4(A3 / A4 印刷适配,2480×3508 = A4 @ 300dpi)
> **核心转化驱动**: 时间地点突出 + 嘉宾权威背书 + 报名方式清晰 + 印刷规格达标

---

## 一、必出元素清单

> 信息递进流:**活动主题抓住路过 → 时间地点解决「能不能去」→ 嘉宾解决「值不值」→ 报名解决「怎么去」**
> 核心逻辑:活动海报路人 5 秒决定参不参加,5 秒内必须解答 4 个核心问题:什么 / 谁 / 何时何地 / 怎么报。

| # | 必出元素 | 视觉权重区 | 占比 | 位置 | 信息职责 |
|---|----------|-----------|------|------|---------|
| 1 | 活动主题 / 标题 | Heavy | 标题字宽 60%+ | 上 1/3 | 抓住路过 — 一眼讲清是什么活动 |
| 2 | 时间地点 | Heavy | 占面 15%+ | 上中部独立信息块 | 决策核心 — 缺一不可,必须独立层级 |
| 3 | 嘉宾 / 讲者 | Medium | 占面 30-40% | 中央 | 权威背书 — 头像 + 姓名 + 头衔 |
| 4 | 报名方式 / 二维码 | Medium | 10-15% | 下 1/3 或角落 | 行动驱动 — 二维码 + 报名链接 / 入场方式 |
| 5 | 主办方 / Logo 群 | Light | < 10% | 最底 | 信任背书 — 主办 / 协办 / 赞助 logo |

---

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 商务专业 / 学术严谨 / 文化展览 / 沙龙轻松 — 调性匹配活动性质 |
| **色温** | 主办方品牌色优先 — 学术冷峻 / 沙龙暖调 / 展览艺术 |
| **背景** | 简洁 / 主题相关大图 / 几何图形;**不**用过满素材(信息层级被淹) |
| **滤镜** | 干净专业 / 轻装饰;不要重滤镜(影响印刷颜色) |
| **质感** | 排版优先 > 视觉优先(活动海报核心是信息传达不是视觉炫技) |

---

## 三、版式构图要求

| 构图方式 | 适用场景 | 焦点位置 |
|---------|---------|---------|
| 上图下信息(上 50% 主视 + 下 50% 信息) | 学术讲座 / 沙龙 | 嘉宾照上,信息块下 |
| 嘉宾群像 + 顶部主题 | 多嘉宾论坛 / 峰会 | 嘉宾 2×3 网格中央,主题顶部 |
| 主题字主导 + 信息块 | 纯信息发布 / 招募 | 大字主题占上 1/3,下面是详细信息 |
| 艺术化主视 + 信息侧栏 | 艺术展 / 文化展 | 主视铺满,信息块右侧 / 底部 |

**字体位置规则**:
- 时间地点必须独立信息块(不与其他文字混排,否则读不出来)
- 二维码 ≥ 200×200px(印刷扫描成功率前提)
- 印刷规格:300dpi、CMYK 色域、出血 3mm 上下左右各加(即按 +6mm 设计画布)
- 印刷字号最小 9pt(< 9pt 印刷糊)

---

## 四、信息层级映射

| 层级 | 视觉权重 | 字号建议(基于 A4 / 2480×3508 画布) |
|------|---------|----------------------|
| 活动主题 | Heavy | 180-280pt |
| 时间地点 | Heavy | 60-90pt |
| 嘉宾姓名 | Medium | 50-70pt(头衔 30-40pt) |
| 详细信息 | Light | 24-36pt(印刷最小 9pt 不可低于) |

---

## 五、文字渲染策略(按 Iron Law #3)

| 文字情况 | 推荐策略 + 模型 |
|---|---|
| 中文标题 ≤ 4 字 | A 嵌入:qwen / nano_banana 直接渲染 |
| 中文标题 5+ 字 | **B 纯底图**:nano_banana 出底图 + 后期排版(默认 — 信息密集必须 B) |
| 时间地点 / 嘉宾信息 | **B 纯底图必须**:底图不画文字信息,后期排版加 |
| 中英混排 / 国际活动 | **B 纯底图**:openai 出底图 + 后期排版 |

**B 策略 prompt 写法**:`event poster background design, no text, no words, leave clean negative space at top one-third for title and bottom one-third for event details, professional clean composition, 3:4 aspect ratio, print-ready quality`

---

## 六、必填项(活动海报 Brief 缺一不可)

不写这些 → sub-agent 凭训练先验补全 → 信息缺失或印刷出错:

- **活动类型**:讲座 / 沙龙 / 论坛 / 展览 / 招募 / 发布会(决定调性)
- **完整时间**(年月日 + 时段)+ **完整地点**(缺一不可,决策核心)
- **嘉宾清单**:姓名 + 头衔 + 头像(无图就声明无,否则 sub-agent 编造头像)
- **报名方式**:二维码图片 / 报名链接 / 现场入场方式(缺则转化失败)
- **主办方信息**:Logo + 协办 / 赞助 logo 群
- **是否印刷**:Y → 加 3mm 出血、CMYK 色域、300dpi / N → RGB 即可
- **风格调性**:学术严谨 / 沙龙轻松 / 艺术展览 / 商务峰会

## 七、易错点(Watch For)

- **时间地点不清晰** — 时间地点埋在小字里 → 路人 5 秒看不到 → 强制独立信息块、Heavy 权重
- **二维码失效** — 二维码 < 200×200px → 印刷后扫不出 → 强制 ≥ 200×200px、留白边缘
- **印刷出血缺失** — 设计按 A4 净尺寸 → 印刷裁切露白边 → 加 3mm 出血(按 +6mm 设计)
- **CMYK 色域偏移** — 用 RGB 高饱和色 → CMYK 印刷后变暗 → 切换 CMYK 工作色域
- **嘉宾头衔混乱** — 头衔比姓名还大 / 没头衔 → 权威背书失效 → 姓名 Medium,头衔 Light
- **信息层级混乱** — 时间 / 地点 / 嘉宾用同一字号 → 决策路径乱 → 严格层级,时间地点 Heavy
- **印刷小字读不清** — 详细信息 < 9pt → 印刷糊 → 详细信息最小 9pt(24-36pt 推荐)
