# 05. 电影海报 (movie-poster)

> **优先质量维度**: Visual Appeal + Drama(戏剧性主视觉 + 高级电影感)
> **默认比例**: 2:3(800×1200)
> **核心转化驱动**: 戏剧性主视觉 + 类型化光影氛围 + 完整 Billing Block(演职员表 + 标语 + 上映日期)

---

## 一、必出元素清单

> 信息递进流:**主视觉建立类型与情绪 → 片名锁定身份 → Billing Block 提供完整片场信息**
> 核心逻辑:电影海报承担「售卖一部电影」的功能,缺任何 Billing Block 元素都不像正式片场物料,会被识破成影迷创作。

| # | 必出元素 | 视觉权重区 | 占比 | 位置 | 信息职责 |
|---|----------|-----------|------|------|---------|
| 1 | 主视觉(人物 / 场景) | Heavy | 65%+ | 中央 / 上 2/3 | 类型与情绪 — 一眼看出是什么类型片 |
| 2 | 片名(中英) | Heavy | 标题字宽 60%+ | 中央 / 上 1/3 / 底部上沿 | 身份锁定 — 必须辨识 |
| 3 | 标语 / Tagline | Medium | 标题字宽 40% | 片名上方或下方 | 价值钩子 — 一句话钩起好奇 |
| 4 | Billing Block(演员表) | Medium | 占下 12-15% | 海报底部 | 片场身份 — 主演 / 导演 / 制片 / 出品 |
| 5 | 上映日期 + Logo 群 | Light | 占下 5% | 最底部 | 售票信息 — 上映日期 + 出品方 logo 群 |

---

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 类型决定调性 — 恐怖(暗压)/ 喜剧(鲜亮)/ 文艺(高级灰)/ 动作(动感)/ 科幻(蓝紫冷峻) |
| **色温** | 严格匹配类型 — 恐怖 / 科幻冷,喜剧 / 爱情暖,文艺中性 |
| **背景** | 强氛围背景(光影 / 场景 / 抽象色彩);**不**用纯白 / 纯灰(不像片场物料) |
| **滤镜** | 电影感分级(Teal & Orange / 黑金 / 沙漠橙 / 蓝调) |
| **质感** | 电影摄影感 + 戏剧光影;写实细节高(演员脸要像本人) |

---

## 三、版式构图要求

| 构图方式 | 适用场景 | 焦点位置 |
|---------|---------|---------|
| 中央人物 + 戏剧光 | 人物驱动剧情片 / 传记 | 主角中央,光从一侧打 |
| 群像(2-5 人) | 群像剧 / 喜剧 / 动作团队 | 角色错落分布,主咖最大 |
| 场景概念 + 小人物 | 科幻 / 史诗 / 灾难 | 大场景 80%,人物 < 20% |
| 抽象意象(道具 / 符号) | 文艺 / 悬疑 / 心理 | 单符号居中,氛围光铺满 |

**字体位置规则**:
- 片名集中在中央或上 1/3 / 底部上沿(三选一,不混用)
- Billing Block 严格占下 12-15%(必须紧凑窄字体,演员名 + 职务上下排版)
- 上映日期 + Logo 群在最底 5%
- 标语贴近片名,不与 Billing Block 混在一起

---

## 四、信息层级映射

| 层级 | 视觉权重 | 字号建议(基于 800×1200 画布) |
|------|---------|----------------------|
| 片名 | Heavy | 100-160pt(占宽 60%+) |
| 标语 | Medium | 30-50pt |
| Billing Block | Light | 18-26pt(必须用窄字体如 SteelTongs / 方正兰亭细黑) |
| 上映日期 | Medium | 36-50pt |

---

## 五、文字渲染策略(按 Iron Law #3)

| 文字情况 | 推荐策略 + 模型 |
|---|---|
| 中文片名 ≤ 4 字 | A 嵌入:qwen / nano_banana 直接渲染 |
| 中文片名 5+ 字 | **B 纯底图**:midjourney 出底图 + 后期加文字(默认) |
| Billing Block | **B 纯底图必须**:底图不画演员表,后期排版加(窄字体 + 紧凑层级) |
| 中英混排 | **B 纯底图**:openai / midjourney 出底图 + 后期加文字 |

**B 策略 prompt 写法**:`cinematic movie poster, no text, no words, leave clean negative space at top center for film title (about 30% canvas height) and bottom 20% for billing block, dramatic cinematic lighting, 2:3 aspect ratio`

---

## 六、必填项(电影海报 Brief 缺一不可)

不写这些 → sub-agent 凭训练先验补全 → 不像正式片场物料:

- **电影类型**:恐怖 / 喜剧 / 文艺 / 动作 / 科幻 / 爱情 / 动画 / 悬疑(决定调性 + 滤镜)
- **完整片名**(中文 + 英文)+ 标语
- **Billing Block 内容**:主演(≥ 2 人)/ 导演 / 制片 / 出品方(缺则不像物料)
- **上映日期** + Logo 群清单(出品方 / 发行方)
- **风格参照**:奥斯卡范 / 香港老港片 / A24 文艺 / 漫威大片 / 韩国黑色电影 ...
- **是否有人物**:Y → 演员脸像锁

## 七、易错点(Watch For)

- **缺 Billing Block** — sub-agent 不知道演员表是必出 → 海报像非官方影迷创作 → 强制后期排版补 Billing Block
- **片名字体不电影感** — 用了普通圆体 / 微软雅黑 → 失去戏剧感 → 用类型对应字体(恐怖:尖锐 / 文艺:衬线 / 动作:粗黑)
- **类型调性错配** — 喜剧片用了恐怖暗调 / 科幻用了暖橘 → 类型混乱 → Brief 阶段强制声明类型
- **演员脸不像本人** — 真人电影演员脸偏移 → 片场不会用 → 用 portrait.md 的参考图 Lock
- **底部演员表字太大** — Billing Block 用了 50pt 字 → 占满底部、不专业 → 严格 18-26pt 窄字体
- **背景失去氛围** — 纯白底 / 普通拍摄背景 → 不像片场物料 → 强制氛围光影 + 类型化滤镜
- **上映日期缺失或散乱** — 没日期或日期飘在中间 → 不专业 → 严格在最底 5%
