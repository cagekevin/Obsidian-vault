## 一、组图摄影规划表（建议 9 张）

> 数码消费者是 **"理性研究型"**，组图需 **感性吸引（英雄图 + 场景）+ 理性说服（参数 + 对比 + 细节）双线并行**。

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 科技感 + 精致感：深色背景凸显质感；线条感布光突出工业设计 |
| **色调** | 冷色调为主（5500-6500K）；背景深灰/黑色凸显产品 |
| **光线** | 多灯位精准布光：主光展示造型、侧光展示材质、轮廓光勾勒边缘。反光控制是核心难点（不可过曝也不可无光泽） |
| **背景** | 纯色（黑/深灰/白）或渐变背景；场景图用办公/创意空间 |
| **文字排版** | 参数图需要精心的信息图表设计，**不是简单贴文字** |

## 三、单图构图要求

| 图片角色 | 构图方式 | 焦点位置 | 突出内容 | 禁忌 |
|----------|----------|----------|----------|------|
| 英雄图 | 45° 居中，80-90% | 产品标志性角度 | 设计语言和品牌辨识 | 角度平庸；反光失控 |
| 接口图 | 侧面聚焦 | 接口/按键区域 | 接口类型和数量 | 标注模糊；遗漏接口 |
| 屏幕图 | 正面居中 | 屏幕内容 | 显示效果和色彩 | 屏幕反光；内容不当 |
| 做工特写 | 满幅 | 材质/接缝处 | 做工精度和材质质感 | 对焦不实；灰尘/指纹 |
| 参数信息图 | 信息图表设计 | 核心数字 | 参数的直观可理解性 | 纯文字堆砌、字体太小 |

## 五、组图一致性检查清单

| 检查项 | 标准 | 优先级 |
|--------|------|--------|
| 主体一致性 | 产品外观/颜色完全一致（屏幕内容除外） | P0 |
| 色调一致性 | 色温偏差 ≤ 300K | P0 |
| 背景一致性 | 纯色背景色差 ΔE ≤ 3 | P0 |
| 反光控制 | 金属/玻璃反光不遮挡产品细节 | P0 |
| 指纹/灰尘 | 产品表面无可见指纹和灰尘 | P1 |

## 六、易错点（Watch For）

- **LOGO 与产品上文字** —— 永远不让 AI 重新渲染品牌文字。用 reference 图或后期合成。
- 屏幕显示内容选择 —— 挑讨喜的画面，不要随机
- 金属高光要"抛光感"，不要"过曝白斑"
- **成对/成组件缺件** —— 真无线耳机的两只 + 充电盒、双手柄游戏机、对讲机配对、双风扇等 —— 主图与配件全家福必须把成对/全套展示完整。耳机只画一只 + 盒子读作"丢了一只豆"。详见 `../image-quality.md` Part 2 — C7。
