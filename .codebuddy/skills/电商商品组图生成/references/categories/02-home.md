# 2. 家装家居 (Home & Furniture)

> **优先质量维度**: Information Delivery + Visual Appeal
> **权重覆盖**: Information Delivery 0.25（尺寸至关重要）
> **组图建议张数**: 8–10
> **核心转化驱动**: 空间比例感 + 尺寸准确

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 北欧简约 → 冷淡高级；日式 → 素雅温润；现代轻奢 → 精致质感；中式 → 典雅沉稳。全组统一一种 |
| **色调统一** | 色温偏差 ≤ 200K；白底图 5500K 标准色温；场景图 4500-5500K 但组内一致 |
| **背景选择** | 图 1-3、8-9 纯白/浅灰（RGB 240-255）；图 4-5 真实家居场景；图 6-7 纯色或场景局部 |
| **光线** | 白底：双侧柔光 + 顶部辅助；场景：模拟自然光 + 暖色辅助；特写：侧光突出纹理 |
| **质感传递** | 木质 → 纹理走向和温润感；布艺 → 编织密度和柔软度；金属 → 拉丝/镜面质感；陶瓷 → 釉面光泽 |

| 图片角色 | 构图方式 | 焦点位置 | 突出内容 | 禁忌 |
|----------|----------|----------|----------|------|
| 白底主图 | 居中，80-90% 占比 | 产品正面 | 外观轮廓和设计语言 | 白底不纯净；产品歪斜 |

## 五、组图一致性检查清单

| 检查项 | 标准 | 优先级 |
|--------|------|--------|
| 主体一致性 | 所有图中产品必须为同一件，颜色/外观无差异 | P0 |
| 色调一致性 | 白底图色温偏差 ≤ 200K；场景图组内色温一致 | P0 |
| 背景一致性 | 白底图 RGB 偏差 ≤ 5；场景图风格统一 | P0 |
| 产品颜色一致性 | 同一产品跨图色差 ΔE ≤ 3 | P0 |
| 风格一致性 | 全组同一审美方向（不出现北欧混搭中式） | P1 |
| 光线方向 | 阴影方向全组一致 | P1 |
| 分辨率 | 长边 ≥ 2000px，画幅比例统一 | P1 |

## 六、易错点（Watch For）

- 沙发与房间比例的合理性（最常见 AI 错误）
- 椅子高度 vs 桌子高度
- 投射阴影方向必须与房间窗户/光源一致
- **成对/成套商品缺件** —— 床品 4 件套、餐盘对杯、对枕、窗帘对幅、椅凳对件 —— 主图与套装展示图必须把全套都摆进画面，缺一件就读作"少发了"。详见 `../image-quality.md` Part 2 — C7。
