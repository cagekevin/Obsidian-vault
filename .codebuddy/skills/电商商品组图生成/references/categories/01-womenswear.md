# 1. 女装 (Womenswear)

> **优先质量维度**: Visual Appeal + Trust（颜色/面料/版型还原）
> **权重覆盖**: Trust 0.20, Visual Appeal 0.25
> **组图建议张数**: 7
> **核心转化驱动**: 色彩还原 + 面料质感 + 穿着代入感

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 快时尚 → 明快简洁；轻奢 → 高级简约克制；淑女 → 柔美优雅；运动休闲 → 活力自然。全组统一一种 |
| **色调统一** | 色温控制 ±200K 以内；后期统一 LUT/预设；主色调偏差 ΔE ≤ 3 |
| **背景选择** | 纯色背景（白/浅灰/米色）用于图 1-6；场景背景仅限图 7；纯色背景色差 ΔE ≤ 1 |
| **光线** | 全组统一主光方向（推荐 45° 侧前方）；光比 1:1.5 至 1:2.5 |
| **质感传递** | 棉麻 → 织物纹理和自然褶皱；丝绸 → 光泽流动感；针织 → 编织密度和弹性；皮革 → 光泽度和纹路 |
| **模特要求** | 同组同一模特；妆容发型一致；表情自然 |
| **后期标准** | 不过度磨皮；不改变服装颜色；锐化参数全组统一 |

> **Apparel 默认背景**: 详情/微距/工艺/平铺/挂拍/人台/纯展示类拍摄使用**自然柔白背景**（白色/米色衣物时略微偏离纯白以保留轮廓）。生活化/模特/场景类仍遵循 C1 的同色系一档对比规则。

## 六、必填项（女装 Brief 缺一不可）

否则模型会默默 drift 到男装或 unisex：

- **性别**:女装 (women's) 必须明确,不能含糊。
- **版型**:剪裁 (slim / regular / oversized)、长度 (cropped / regular / longline)、腰线 (cinched waist / 收腰 / 自然腰线)、肩线 (柔肩 / 自然) 都要明确。
- **袖口与领口形状**:在女装常见词汇范围内挑准 (盖肩袖 / 泡泡袖 / 方领 / V 领 / 一字领 等)。
- **必须避免反向性别特征**:broad shoulders / boxy silhouette / masculine drape 等。
- **模特性别**:出现模特时锁定女性 + 年龄段 + 体型类别。
- **整组同一身份**:模特在每张图里必须是同一个人,不漂。

## 七、易错点（Watch For）

- **版型 drift 到男装** —— reference 是 unisex 单品（oversized T、basic crew）时最常见。务必显式声明 cut。
- AI 模特手部 / 手指数 / 关节合理性
- 面料褶皱物理性 —— 垂坠必须服从重力
- 高饱和色（红、玫红、宝蓝）的偏色
- 浅色面料与浅背景融化（白 T 撞白底） —— 见 `../image-quality.md` Part 2 — C1
- 面料/工艺/微距镜头里出现皮肤（杀掉工艺信息） —— 见 `../image-quality.md` Part 2 — C2
- **成对配件缺件** —— 女士手套、长袜对装、对耳饰、对镯等配饰 —— 主图与配件展示图必须摆齐两只/全套，单只读作"丢了一只"。详见 `../image-quality.md` Part 2 — C7。
