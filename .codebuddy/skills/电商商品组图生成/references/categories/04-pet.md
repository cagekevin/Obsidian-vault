# 4. 宠物用品 (Pet)

> **优先质量维度**: Visual Appeal + Information Delivery（情感 + 用途）
> **组图建议张数**: 6–8
> **核心转化驱动**: 萌宠情感 + 安全性

---

> 核心逻辑：宠物用品的情感驱动力极强 ——"我的宠物喜欢"比"产品参数好"更打动消费者。**封面建议有宠物出镜。**

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 温馨活泼，饱和度适中偏暖；宠物场景要自然不刻意 |
| **色调** | 暖色调为主（宠物亲和力）；色温 5000-5500K |
| **背景** | 白底产品图：纯白/浅灰；宠物场景：居家自然环境 |
| **光线** | 柔和自然光为主；宠物场景避免闪光灯惊扰动物；宠物毛发需要柔光拍出蓬松感 |
| **宠物拍摄** | 高速连拍捕捉自然瞬间；避免强制摆拍导致表情僵硬 |

## 三、单图构图要求

| 图片角色 | 构图方式 | 焦点位置 | 突出内容 | 禁忌 |
|----------|----------|----------|----------|------|
| 产品主图 | 居中，80-90% | 产品正面 | 外观和品牌 | 反光；角度不佳 |
| 宠物场景 | 三分法，宠物互动 | 宠物与产品接触点 | 宠物使用的真实状态 | 宠物表情痛苦/不适 |
| 功能演示 | 对比/过程式 | 功能效果区域 | 功能可视化证据 | 效果过度夸大 |
| 材质特写 | 满幅 | 材质表面 | 安全性和质感 | 对焦不实 |

## 五、组图一致性检查清单

| 检查项 | 标准 | 优先级 |
|--------|------|--------|
| 主体一致性 | 产品外观完全一致 | P0 |
| 色调一致性 | 色温偏差 ≤ 300K（含宠物场景） | P0 |
| 背景一致性 | 白底图 RGB 偏差 ≤ 5 | P0 |
| 宠物状态 | 宠物表情自然舒适，无不适表现 | P0 |
| 风格一致性 | 全组温馨活泼风格统一 | P1 |

## 六、易错点（Watch For）

- 宠物-产品互动的自然度 —— 动物 AI 在眼睛/四肢上有典型破绽
- 整组保持暖色温（冷调让画面变得"诊所感"）
- **成套宠物用品缺件** —— 双口粮碗、自动饮水器套件、宠物美容工具套装、爪垫四件、口罩 + 牵引套装等 —— 主图与套装图必须摆齐全套，缺件读作"少发了"。详见 `../image-quality.md` Part 2 — C7。
