> **核心转化驱动**: 尺寸感知 + 收纳能力

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格** | 轻奢 → 高级质感；商务 → 专业稳重；休闲 → 活力自然；旅行 → 实用耐用 |
| **光线** | 皮质需多灯位展示光泽层次；五金件用点光源增加闪耀感 |
| **背景** | 纯色（白/灰/浅米）为产品图；咖啡厅/街头/办公室为场景图 |
| **色调** | 色温 5000-5500K；皮具偏暖展示温润感；尼龙/帆布可偏中性 |
| **道具** | 内部物品用真实物品（MacBook、iPhone、护照、水杯） |

## 三、单图构图要求

| 图片角色 | 构图方式 | 焦点位置 | 突出内容 | 禁忌 |
|----------|----------|----------|----------|------|
| 正面主图 | 居中，80-90% | 包身正面 | 外观和品牌辨识 | 包变形/塌陷 |
| 开合内部 + 容量 | 打开 + 放入物品 | 内部分隔 + 物品排列 | 收纳设计 + 实际容量 | 内部光线不足；物品过多杂乱 |
| 背法展示 | 三分法 | 肩带与身体 | 背法多样性 | 肩带扭曲；背法不自然 |
| 材质特写 | 满幅 | 皮面/帆布/尼龙等表面 | 材质品质 | 对焦不实；灰尘 |

## 四、模特上身镜头框定 —— 焦点是"包"

- **跨身/单肩**：肩到髋，包占视觉重心；下颚以上或整个头都裁掉。
- **手提/托特**：胸到大腿；脸通常在画外。
- 包是焦点主角，**不是模特的脸**。
- **裁脸 ≠ 不算模特** —— 即使只露躯干 + 手臂，**可见的上装/下装/外套/腰带/手部首饰/指甲色依然是 styling 决策**，不能默认。**必须先走 Phase 3.4 Model Profile Gate** 让用户确认本组上身镜头的可见 styling 项；用户确认后纳入整组穿搭锁，所有同组镜头复用同一套搭配。

## 七、易错点（Watch For）

- 内部镜头容易过暗 —— 需要补光
- 五金件必须看起来是金属的，不是涂漆的
- **成套包缺件** —— 子母包、钱包卡包套装、母女背包套、行李箱多件套 —— 主图与套装图必须把全套都摆齐，缺一件读作"少发了"或"配错单"。详见 `../image-quality.md` Part 2 — C7。
- **不出五金件单独 macro、不出多色对比图** —— 五金件(扣件/拉链/锁扣)的精致度由材质特写镜头自然带出,单独切出来无独立信息且容易显得"配件特写堆砌";SKU 多色按独立 listing 拆分,单 listing 不堆多色。仅当用户明确 brief 招牌五金(品牌签名扣)或运营要求多色对比时再加。
- **内部 + 容量拆成两张** —— 开合空内格 + 塞满物品的"两张同 setup"是常见的水图陷阱。**强制合并为一张**(打开 + 物品同框),空内格图既无设计语言又无尺寸感,纯占位。
