# 7. 饰品珠宝 (Jewelry)

> **优先质量维度**: Visual Appeal + Product Clarity
> **权重覆盖**: Clarity 0.30, Visual 0.25, Trust 0.10
> **组图建议张数**: 4–5（含礼盒视用户素材而定）
> **核心转化驱动**: 美感 + 工艺信任

---

## 珠宝铁律 — 成对/成套必须凑齐 + 双件镜像对称

**Pair, not single（成对/成套，非单件）**: 凡是 **耳饰（耳钉/耳环/耳夹）/ 对戒 / 情侣款戒指 / 对镯 / 成套首饰套组** —— **每张棚拍主图、佩戴图、材质图、礼盒图都必须把成对/全套都展示在画面里**。耳饰只画一只读起来像"丢了一只"；对戒只画一枚像"还没拿到货"；首饰套组缺件像"被截胡"。**仅在用户明确 brief 一张专门做工/镶嵌微距、单独突出某一颗宝石或某一面雕刻细节时**，才允许只展示其中一只/一枚。

**Mirror-symmetry guard（镜像对称防线）**: 即使画了两只，AI 也常常把第二只画歪 —— 宝石尺寸不一致、刻面数不同、金属抛光走向不一致、左右朝向打架。**每张成对镜头的 prompt 都要显式写上 `the two pieces are mirror-identical in size, stone count, facet pattern and metal finish — no left/right drift`，并在 Phase 4 把成对镜头放大到 100% 比对左右两件。**

→ 详见 `../image-quality.md` Part 2 — C7（Pair / Set Completeness）。

## 子品类适配规则:Image 2 佩戴效果图

珠宝是大类,内含子品类,佩戴位置完全不同。`Image 2 佩戴效果图` 必须**按子品类映射到对应的实际佩戴位置**,不要硬套"手指/手腕旁"的默认。

| 子品类 | 佩戴位置 | 框架 |
|---|---|---|
| 戒指 / 对戒 | 戴在手指上 | 单手特写,手指自然弯曲 |
| 手镯 / 手链 | 戴在手腕上 | 手腕至手背的自然垂落 |
| 项链 / 锁骨链 / 吊坠 | 戴在颈部 | 锁骨可见,脖子做参照 |
| **耳环 / 耳钉 / 耳骨夹** | **戴在耳朵上** | **ear-to-shoulder 框架,耳朵 + 部分脸侧** |
| 胸针 | 别在衣领或西装翻领上 | 胸前局部特写 |
| 脚链 / 脚饰 | 戴在脚踝上 | 脚踝可见 |

**绝对禁止**:把耳环 / 项链 / 吊坠 / 胸针 / 脚链放在**手上**当佩戴展示 —— 跟实际佩戴位置脱节,买家无法准确感知。

如果用户提供的 ref 中**正好有合适的佩戴位置照片**(如 ref 是耳朵戴耳环的照片),可作为 Image 2 的 informational ref;但 master 写 plan 时仍按上述映射决定**输出图片**的佩戴位置,不要让 ref 牵着走。

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 高级感 + 仪式感：深色系背景（黑/深蓝/深红）衬托珠宝光泽。大牌参考 Tiffany（冰蓝）、Cartier（深红）、Bvlgari（黑） |
| **光线** | 珠宝摄影核心：点光源制造火彩，柔光展示整体造型，反射光增加光泽 |
| **背景** | 深色绒面/丝绸/大理石等质感背景；纯白底用于认证/参数图 |
| **色调** | 根据材质选择：金饰 → 暖色调；银饰 → 冷色调；宝石 → 衬托宝石本色 |
| **后期** | 允许适度增强光泽和火彩，**不可改变产品实际外观** |

## 六、易错点（Watch For）

- 反射准确性 —— 宝石中反射的内容必须合理
- 戒指圆度 —— 不能椭圆或波浪形
- **配对耳饰/对戒/对镯只画了一只** —— 看起来像"丢了一只"。除工艺微距外**只接受成对**，详见本文件顶部「珠宝铁律」与 `../image-quality.md` Part 2 — C7。
- **配对件左右不对称** —— 第二只宝石尺寸/刻面/金属抛光与第一只不一致，必须 100% 比对。
- **图 1 和图 5 视觉重复** —— 两张都是"产品放深色背景+戏剧光"是最常见的套图重复陷阱。图 5（材质图）必须换 sub-set（浅色表面）、换机位（90° 俯拍）、换光源（柔和顶光），否则买家滑过去会觉得"怎么又是一样的图"。
- **图 1/2/3 三张产品角度图机位撞设** —— 同 sub-set + 同光位下,机位/俯角必须显著拉开:图 1 = 45° 微俯;图 2 = 0° 平视;图 3 = 90° 侧/背面。任意两张机位差 < 30° 或俯角差 < 20° 视为重复,需要重生成。同 axis 撞设是 jewelry 多角度组的特许豁免,但不是抄作业的借口。
- **不出工艺微距、尺寸参考、多款多色对比** —— 工艺感由主图戏剧光 + 材质图换面换光呈现；尺寸数字（戒圈/链长/克拉/克重）放进商品标题/详情属性栏，不出图（一旦出图就要凑数字，缺乏真实凭证则触发 Iron Law 1 + Quality 红线 5）；SKU 多色多款按独立 listing 拆分，单 listing 不堆。仅当用户明确 brief 一张专门做工微距（且能提供清晰 ref）、提供尺寸数字凭证、或运营要求多款对比时再加。
- **从零编造礼盒** —— 礼盒图是条件位，**只在用户上传真实礼盒素材时才出**。无素材时直接跳过，不要根据品牌"想象"或"通用礼盒模板"自行生成 —— 礼盒外观与实际不符会触发售后纠纷与 Iron Law 1 真实性问题。
