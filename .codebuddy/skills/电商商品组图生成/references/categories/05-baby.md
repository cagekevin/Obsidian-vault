## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 柔和温馨、安全可信。色调清新不刺激，传递"安全放心"的信任感 |
| **色调** | 暖白色调为主，色温 5000-5500K；柔和粉彩色系（淡粉/浅蓝/鹅黄） |
| **背景** | 白底/浅色系背景；场景图用婴儿房/客厅等柔和家居环境 |
| **光线** | 柔光为主，**绝对避免硬阴影**；光比 ≤ 1:1.5 |
| **安全传达** | 每张图都要传递"安全"信号 —— 圆角、柔软、无锐边 |
| **模特** | 真实宝宝（非过度 PS 的完美婴儿）；亲子互动更有感染力 |

## 六、易错点（Watch For）

- **婴儿脸部自然度** —— 任何 uncanny-valley 脸都会摧毁整个 listing 的信任。**不确定时不要从零生成婴儿脸**，合成真实参考照。
- 婴儿手部 / 手指数 / 关节
- 整组使用柔和暖调，**禁止诊所感冷白**
- **成对/成套缺件** —— 婴儿袜、婴儿手套、口水巾对、奶瓶套装、辅食碗勺套件、婴儿礼盒套装 —— 主图与套装图必须凑齐全套，单只袜子或缺一件读作"少发了"。详见 `../image-quality.md` Part 2 — C7。
- **不出认证/资质卡、配件全家福、年龄/规格图表、多色/多款对比** —— 母婴 listing 的安全感与信任来自宝宝真实使用 + 材质亲肤的视觉证据，**不是来自"信息卡"类静物**。监管要求的认证编号、规格数据、配件清单都该写在商品标题/详情文案/属性栏里，不要把它们塞进图片 —— 一旦塞进去就触发监管红线（任何认证名称/编号/数据缺乏真实凭证就是编造）。仅当用户明确 brief 必须出某张证书图（且提供真实凭证扫描）时才作为可选位加入。
