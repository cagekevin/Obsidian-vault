# 12. 鞋靴 (Footwear)

> **优先质量维度**: Product Clarity + Information Delivery
> **组图建议张数**: 7
> **核心转化驱动**: 颜值 + 舒适暗示

---

## 鞋类两条铁律

**Pair, not single（成对，非单只）**: 鞋是成对售卖与穿着的 —— **每张棚拍/细节镜头都要展示两只鞋**（镜像并排，或一正面 + 一 3/4 角度，或叠放/重叠的干净构图）。单只鞋读起来像"丢了一只"，除非镜头本身就是某个特征的微距。

**Multi-angle over per-feature splits（多角度 > 强行拆细节）**: 鞋本身体积紧凑 —— 没有足够表面积撑起 5+ 张分散的细节图。优先**整双鞋的多角度**（同一产品多视角），而不是强拆 per-feature。仅在有真正独立的特征时才加 macro（鞋底纹路、鞋跟 logo、招牌扣件）。

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格** | 运动鞋 → 动感 + 科技感；皮鞋 → 经典 + 质感；潮鞋 → 街头 + 态度 |
| **光线** | 多灯位：主光展示造型，侧光展示材质纹理，轮廓光勾勒边缘 |
| **背景** | 纯色（白/灰/黑）为产品图；街道/运动场为场景图 |
| **角度** | **3 张产品角度图分工**：图 1 侧面俯拍 (设计语言) / 图 2 正面俯拍 (鞋面图形) / 图 3 纯侧面平视 (鞋身高度 + 鞋底厚度)。三张以"鞋的朝向 + 俯角"双轴区分,不允许任意两张撞 |
| **色调** | 色温 5000-5500K；运动鞋可略偏冷（科技感），皮鞋偏暖（质感） |

## 四、上脚镜头框定规则 —— 焦点是"鞋"，不是"模特的脸"

- **默认框定**：膝盖以下到鞋底以下，**不要半身、不要全身、不要肖像**。鞋占视觉中心；腿/裤是上下文。**不包含模特的脸/头/肩** —— 加上脸会出现一个抢戏的焦点，让镜头变成"穿鞋的模特"而非"被穿的鞋"。
- **裁脸 ≠ 不算模特** —— 即使只露膝盖以下，**可见的下装（裤/袜/裙边/裤脚收口）依然是 styling 决策**，不能默认（"黑色紧身裤""白色短袜""卷边牛仔裤"等都属于默认）。**必须先走 Phase 3.4 Model Profile Gate** 把"下装 + 袜子 + 裤脚处理 + 裸足/穿袜"作为本组上脚镜头的 styling 项让用户确认；用户确认后纳入整组穿搭锁,所有上脚镜头复用同一套搭配。
- **可接受的更宽框定**（需明确 brief）：背后或侧面行走镜头，小腿以下；坐姿系鞋带，脸通常仍在下颚以上裁掉。
- **相机角度**：低角度/腰部高度/地面高度向下看鞋 —— **绝对不要**鞋类镜头里用模特脸的眼平角度。
- **画幅**：3:4 竖版或 1:1，让鞋占足够画面。
- **场景表面**：按 styling 方向（城市街道/影棚地面/赛道/木地板）。
- **永远不要**只因为镜头里有模特就默认全身或半身肖像 —— 模特的工作就是穿这双鞋，仅此而已。

## 七、易错点（Watch For）

- 浅色鞋撞白底 —— 会"消失"。加微弱阴影或渐变。
- 鞋带松紧度与鞋舌位置必须合理。
- **上脚镜头强行露脸** —— 主要框定 bug。脸通常应在画外；如果框定意外切到半张脸，那是 Iron Law 18（framing integrity）失败 —— 重框。
- **英雄/标准镜头里只有单只鞋** —— 看起来未完工。**只接受成对**。
- **强拆 per-feature 细节** —— collar / heel / lace / sole 各拍一张 macro，对紧凑产品来说太勉强。**默认整双多角度（图 1/2/3）+ 一张招牌局部（图 4）**，不要再额外拆 per-feature。
- **不出后跟特写、内部鞋垫/鞋口、鞋面材质 macro、鞋底特写、多色对比图** —— 鞋本身体积紧凑,这些切角属于"已被图 4 招牌局部覆盖的可选项"或"对紧凑产品过分拆解"。**舒适度 by default 由 brief 文案 + 上脚镜头隐性传达**，不为它专门留位置。仅当用户明确 brief 招牌后跟 logo / 签名扣件 / 特殊鞋垫科技 / 鞋底防滑科技 / 必须的多色 SKU 对比时，才考虑作为图 4 的内容选择,或额外加可选 macro 位置。
- **图 4 拍成"通用结构"而非"招牌点"** —— 局部特写的价值在于品牌辨识或工艺独特性。如果产品没有明显 logo / 印记 / 招牌扣件 / 工艺纹路,**图 4 应该省略**(改回 6 张组图),不要为了凑张数拍一个普通鞋头/普通鞋面纹路 macro。
- **图 1 / 图 2 / 图 3 三张产品角度互撞** —— 关键差异是**俯角 + 鞋朝向**两轴:图 1 = 侧俯(鞋 profile 朝镜头, 俯 15-25°);图 2 = 正俯(鞋头朝镜头, 俯 45-60°);图 3 = 侧平(鞋 profile 朝镜头, 俯 0°)。生成后必须三张并排比对,任意两张撞 → 重生成。常见失败:图 3 拍成图 1 (俯角没归零) / 图 2 拍成图 1 (鞋的朝向歪了)。
