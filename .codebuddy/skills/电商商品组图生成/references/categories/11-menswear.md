## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格** | 简洁克制、干净利落。"少即是多"，避免过度装饰 |
| **色调** | 色温 5200-5500K。商务款偏冷灰调，休闲款偏暖中性调。偏差 ≤ ±200K |
| **背景** | 主图/多角度：纯白或浅灰。场景图：低饱和度环境。平铺：纯白/浅木纹 |
| **光线** | 柔光为主，光比 1:1.5-1:2。"干净"光感 |
| **质感** | 棉质 → 柔软蓬松感；毛呢 → 厚实细腻；丝光 → 适度高光反射；牛仔 → 纹理粗犷 |
| **模特** | 身材匀称（非过度健美），气质匹配产品定位 |

> **Apparel 默认背景**: 详情/微距/工艺/平铺/挂拍/人台/纯展示类拍摄使用**自然柔白背景**。生活化/模特/场景类仍遵循 C1 的同色系一档对比规则。

## 六、必填项（男装 Brief 缺一不可）

否则模型会让 unisex 单品 drift 到女性化或中性化:

- **性别**:男装 (men's) 必须明确,不能含糊。
- **版型**:剪裁 (regular / slim / relaxed)、轮廓 (straight body / boxy / fitted)、肩线 (defined shoulder line / 自然肩)、腰线 (straight waist / 直腰)、领型 (crew neck / polo / button-down)、下摆 (shirt-tail hem / straight hem) 都要明确。
- **必须避免反向性别特征**:cinched waist / bust dart / feminine drape / 收腰 等(尤其是修身款)。
- **模特性别**:出现模特时锁定男性 + 年龄段 + 体型类别。
- **整组同一身份**:模特在每张图里必须是同一个人,不漂。

## 七、易错点（Watch For）

- **修身/时尚款 drift 到女性化或中性化** —— 务必显式声明 cut。
- 男性肩宽与手臂肌肉的自然度
- 西装垂坠必须挺括，不能软塌
- Action 姿势克制 —— 太动感会破坏正装感
- 深色面料与深色背景融化（黑 T 撞深木背景） —— 见 `../image-quality.md` Part 2 — C1
- 面料/工艺/微距镜头里出现皮肤 —— 见 `../image-quality.md` Part 2 — C2
- **成对配件缺件** —— 男士手套、袖扣、护腕对、男袜对装等 —— 主图与配件展示图必须摆齐两只/全套，单只读作"丢了一只"。详见 `../image-quality.md` Part 2 — C7。
- **不出多色/多款对比图** —— SKU 多色按独立 listing 拆分,单 listing 不堆多色对比;颜色选择由 listing 标题与属性栏承载,图片专注当前 SKU 的版型与做工。仅当运营明确要求多色对比时再加。
