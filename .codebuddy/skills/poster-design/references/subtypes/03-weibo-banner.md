# 03. 微博 Banner / 配图 (weibo-banner)

> **优先质量维度**: Visual Appeal + Engagement(信息流抓眼球 + 高对比)
> **默认比例**: 16:9(1280×720)或 1:1(1080×1080)
> **核心转化驱动**: 0.3 秒视觉钩子(高对比 + 强焦点)+ 短文案(≤ 8 字主标题)+ 蹭热度色彩

---

## 一、必出元素清单

> 信息递进流:**强对比抓住停留 → 短文案传达观点 → 品牌色辨识**
> 核心逻辑:微博信息流刷屏极快(比小红书更快),0.3 秒抓不住眼球就划走,文案必须短到一眼读完。

| # | 必出元素 | 视觉权重区 | 占比 | 位置 | 信息职责 |
|---|----------|-----------|------|------|---------|
| 1 | 主视觉钩子 | Heavy | 60%+ | 中央 / 黄金点 | 视觉抓眼 — 高对比 / 强情绪 / 强反差 |
| 2 | 短文案(≤ 8 字) | Medium | 标题字宽 50%+ | 中央 / 上 | 价值速达 — 一眼读完,不超过 8 字 |
| 3 | 品牌色块 / Logo | Light | < 10% | 角落 / 边缘 | 辨识度 — 品牌色调强化属性 |
| 4 | 话题标签(可选) | Light | < 5% | 顶部或角落 | 蹭流量 — # 话题词 |

---

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 信息流抓眼球 / 热点话题 / 营销 banner / 时效内容 |
| **色温** | 高对比(不限冷暖)— 越对比越能在信息流中跳脱 |
| **背景** | 高饱和单色 / 渐变冲撞 / 反差色块;**不**用米白柔和(信息流划走) |
| **滤镜** | 高对比硬朗 / 鲜艳饱和;不要高级灰(融进信息流背景) |
| **质感** | 平面 + 强色块 / 商业摄影 + 主体突出;不追求高级感,追求「跳出来」 |

---

## 三、版式构图要求

| 构图方式 | 适用场景 | 焦点位置 |
|---------|---------|---------|
| 中央焦点 + 顶部短文案 | 热点事件 / 营销活动 | 焦点中央占 60%,文案顶部 |
| 左文右图(强反差色) | 议题 / 观点输出 | 文字左半,图右半 |
| 全图 + 角落标签 | 产品发布 / 视觉抢镜 | 主图铺满,话题标签角落 |
| 拼贴对比(before / after) | 测评 / 对比类 | 二分构图 + 中央分隔线 |

**字体位置规则**:
- 主文案居中或顶部 1/3,避免被微博信息流压缩缩略图裁切边缘
- 1:1 方图缩略图边缘各保 5% 缓冲
- 16:9 横图重要信息收敛到中央 80%

---

## 四、信息层级映射

| 层级 | 视觉权重 | 字号建议(基于 1280×720 画布) |
|------|---------|----------------------|
| 主文案 | Heavy | 100-160pt(占宽 50%+) |
| 副文案 | Medium | 40-60pt |
| 话题 / Logo | Light | 18-28pt |

---

## 五、文字渲染策略(按 Iron Law #3)

| 文字情况 | 推荐策略 + 模型 |
|---|---|
| 中文短词 ≤ 4 字 | A 嵌入:nano_banana / qwen 直接渲染 |
| 中文 5-8 字 | **B 纯底图**:nano_banana 出底图 + 后期加文字(默认) |
| 中文 > 8 字 | 减字!微博信息流 > 8 字读不完 → 让用户精简 |
| 英文 / 数字 / # 标签 | **A 嵌入**:openai / nano_banana 直接渲染 |

**B 策略 prompt 写法**:`high contrast advertising banner, no text, no words, leave clean negative space at center for short headline (about 4-8 Chinese characters), bold visual hook, 16:9 aspect ratio`

---

## 六、必填项(微博 Banner Brief 缺一不可)

不写这些 → sub-agent 凭训练先验补全 → 信息流划走:

- **场景类型**:营销 / 热点 / 话题输出 / 产品发布 / 测评 / 抽奖(决定调性)
- **主文案 ≤ 8 字**(超过则要求精简,微博信息流读不完)
- **是否蹭热点**:Y → 提供热点关键词 + 时效要求 / N → 自定调性
- **品牌色 + Logo 位置**(角落 / 不放;无品牌色辨识度归零)
- **比例选择**:16:9(横屏)/ 1:1(移动信息流默认)

## 七、易错点(Watch For)

- **文案太长** — 主文案 > 8 字 → 信息流缩略图读不完 → 强制精简到 4-8 字
- **低对比度** — 莫兰迪 / 高级灰 → 融进微博信息流被划走 → 改高对比 / 高饱和
- **过度精致** — 杂志感 / 编辑式 → 微博信息流违和 → 改 banner / 营销感
- **关键信息边缘** — 文字 / Logo 贴边 → 缩略图裁切丢失 → 边缘 5% 缓冲
- **没用品牌色** — 通用配色 → 信息流刷过没辨识度 → 至少用一处品牌色
- **比例错位** — 用 9:16 当 Banner → 微博自动裁成奇怪缩略图 → 严格 16:9 或 1:1
