# 08. 杂志封面 (magazine-cover)

> **优先质量维度**: Visual Appeal + Creativity（编辑式高级感 + 时尚摄影质感）
> **默认比例**: 3:4（700×933 屏显 / 印刷 A4 2480×3508 @ 300dpi）
> **核心转化驱动**: 刊名顶部巨大权威感 + 主视觉人物大留白 + 编辑式排版高级感

---

## 一、必出元素清单

> 信息递进流：**刊名锚定品牌身份 → 主视觉抓住眼球 → 封面故事吸引翻阅 → 期号/条码标注实物感**
> 核心逻辑：杂志封面是「品牌资产 + 当期内容」双重表达，刊名是品牌锚点（权重最高），主视觉与刊名形成图文呼应。

| # | 必出元素 | 视觉权重区 | 占比 | 位置 | 信息职责 |
|---|----------|-----------|------|------|---------|
| 1 | 刊名（masthead）| Heavy | 占宽 70-90% | 顶部 5-15% | 品牌锚点 — 巨大字号建立刊物权威 |
| 2 | 主视觉（人物/物体）| Heavy | 占面 50-65% | 中下部 | 编辑视觉 — 时尚大片质感 |
| 3 | 封面故事标题 | Medium | 占宽 30-45% | 主视觉两侧或下半部 | 内容钩子 — 当期主打文章引文 |
| 4 | 副刊语 / 引文 | Light | 占宽 20-30% | 散布留白处 | 内容预告 — 2-4 条次级故事 |
| 5 | 期号 / 日期 / 条码 | Light | < 5% | 底部角落 | 实物感 — 刊期标记与零售凭证 |

---

## 二、整体风格与质感

| 维度 | 要求 |
|------|------|
| **风格方向** | 编辑式高级感（editorial）/ 时尚大片 / 极简主义 / 概念艺术 — 反「商业广告感」 |
| **色温** | 中性偏冷或单色调（high-key 高调白 / low-key 低调暗）— 凸显品味 |
| **背景** | 大面积纯色或柔和渐变 / 影棚硬光背景 / 极简场景；**禁止**杂乱花纹 |
| **滤镜** | 胶片质感 / 高级灰 / Hasselblad 中画幅质感 / 杜松子色调 |
| **质感** | 棚拍硬光 + 阴影立体 / 中画幅景深 / 时装大片光线（Vogue / Bazaar / Wallpaper 参照）|

---

## 三、版式构图要求

| 构图方式 | 适用场景 | 焦点位置 |
|---------|---------|---------|
| 正中人物 + 顶部刊名压字 | 时尚 / 名人封面 | 人物居中占 60%，刊名压在头顶有「叠盖」效果 |
| 大留白 + 偏置主体 | 极简编辑 / 设计杂志 | 主体偏左/右 1/3，2/3 留白放刊名+引文 |
| 全身人物贯穿 | 高定 / 时装大刊 | 人物全身居中，刊名顶部，文字描边或镂空避免冲突 |
| 概念静物特写 | 设计 / 美食 / 旅行 | 物体居中铺满，刊名下方或叠加 |

**字体位置规则**：
- 刊名**永远顶部**且**永远最大**（占宽 70%+，不容妥协）
- 封面故事文字与人物**不可重叠面部**（可叠加身体/背景区域）
- 留白区不要塞满，至少保留 25% 纯净呼吸空间

---

## 四、信息层级映射

| 层级 | 视觉权重 | 字号建议（基于 700×933 屏显画布）|
|------|---------|----------------------|
| 刊名 masthead | Heavy（最显眼）| 100-160pt（占宽 70-90%）衬线/无衬线均可，需粗壮 |
| 封面故事标题 | Medium | 28-44pt（占宽 30-45%）|
| 副刊语 / 引文 | Light | 12-18pt |
| 期号 / 条码 | Light | 8-10pt |

---

## 五、文字渲染策略（按 Iron Law #3）

| 文字情况 | 推荐策略 + 模型 |
|---|---|
| 英文刊名（VOGUE/BAZAAR 风）| A 嵌入：openai / midjourney 直接渲染 |
| 中文刊名 ≤ 4 字 | A 嵌入：qwen（明显中式刊名）|
| 封面故事中英混排 | **B 纯底图**：midjourney 出底图 + 后期排版（默认）|
| 多条引文 + 期号条码 | **B 纯底图**：必须后期加文字 |

**B 策略 prompt 写法**：`editorial fashion magazine cover, no text, no logo, no words, leave clean space at top 15% for masthead and side margins for cover lines, high-end studio photography lighting, fashion editorial style`

---

## 六、必填项（杂志封面 Brief 缺一不可）

不写这些 → sub-agent 凭训练先验补全 → 偏离编辑刊调性：

- **刊物定位**：时尚 / 设计 / 旅行 / 商业 / 文艺 / 科技 — 决定整体调性
- **刊名完整文字 + 语言**（中/英/双语）+ 字体倾向（衬线/无衬线/手写）
- **是否含人物**：Y → 模特身份锁+ 表情/姿态 / N → 概念静物
- **当期主题词**（如「春日复苏」「未来之城」）— 决定主视觉方向
- **色调倾向**：高调白 / 低调黑 / 单色调 / 双色对比

---

## 七、易错点（Watch For）

- **刊名太小** — 字号 < 80pt（基于 700 宽）→ 失去权威感像普通海报，强制刊名占宽 70%+
- **商业广告感** — 高饱和 + 多焦点 + 文字铺满 → 像传单不像杂志，必须大留白 + 单一主视觉
- **人物面部被字遮** — 引文叠加面部 → 失去人物魅力，文字只可压在身体/背景
- **滤镜过 AI 完美感** — 皮肤磨皮 / 塑料感 → 失去时装大片真实感，请求"film grain / medium format"
- **背景花纹杂乱** — 复杂底图与刊名抢戏 → 改纯色或柔和渐变背景
- **刊名风格错位** — 设计刊用衬线古典字 / 时尚刊用手写体 → 按刊物定位定字体调性
- **底部信息缺失** — 没有期号/条码 → 失去实物感，加底部细小条码与日期
