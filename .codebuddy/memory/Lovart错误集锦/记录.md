### 案例: 金色胶囊形状失真与尺寸错误 | 错误类型: 形状/尺寸 | 严重度: ⭐⭐⭐⭐⭐

项目背景: 正在制作HKH品牌的"金色胶囊溢出"系列创意图,需要将日常场景(电梯、车后备箱、地铁)中的购物袋替换成品牌的金色胶囊,营造超现实的丰盈感和视觉冲击力。

当前卡点: AI反复生成的胶囊形状与参考图不符(变成了普通药丸形状),且尺寸过小(2-3cm而非预期的30-50cm),即使明确标注尺寸和强调形状,仍然失败。用户多次表达强烈不满("为什么这么小"、"你真是个傻子"、"提示词里面一定有矛盾之处")。

🎯 任务目标

预期效果: 将参考场景图中的购物袋精确替换为HKH金色胶囊,胶囊形状必须与产品图完全一致,尺寸与购物袋相当(视觉占比相同),摆放自然错落。

参考图:

图1 [场景图: 电梯/车后备箱/地铁] - 提供场景、光线、构图
图2 [HKH金色胶囊产品图] - 提供胶囊的精确形状、材质、颜色

❌ 错误现场与诊断

第一轮失败

原提示词:

```
Commercial photography of golden capsules overflowing in [场景]. 
The capsules are HKH brand golden soft capsules, each approximately 
50cm in length, with a distinctive elongated oval shape, metallic 
golden surface with soft sheen. Capsules densely packed and naturally 
scattered, filling the entire [场景空间]. Soft diffused lighting, 
warm color palette, hyper-realistic DSLR quality.
```

实际生成: 胶囊变成了2-3cm的小药丸,形状是标准的椭圆形药丸,而非参考图的独特形状

致命偏差:

- 尺寸完全错误(2-3cm vs 50cm)
- 形状失真(普通药丸 vs 独特胶囊形状)
- 数量过多导致密集恐惧症

根本原因 (Root Cause):

- 词语冲突
- 权重失衡
- 抽象词过多
- 缺少关键约束

深度剖析: AI把 capsule 错误理解成了 小药丸(2-3cm的标准胶囊),这是AI训练数据中"capsule"的主流概念。无论我如何用文字描述"50cm"、"elongated oval shape"、"distinctive shape",AI都会:

1. 先调用"capsule = 小药丸"的默认认知
2. 然后尝试在这个基础上"放大"或"修改形状"
3. 但本质认知锚点还是"药丸",导致形状始终不对

我的盲区: 我当时的思路是"用更精确的文字描述尺寸和形状" → 为什么这是错的: 这是在对抗AI的固有认知,试图用文字"说服"AI改变它对"capsule"的理解。但AI的词汇联想是训练时固化的,文字描述无法覆盖这种深层偏见。数字"50cm"对AI来说只是一个参数,它会理解成"生成一个放大版的药丸",而不是"生成一个与购物袋同等视觉占比的物体"。

第二轮失败(尝试替换逻辑)

原提示词:

```
Replace the shopping bags in the scene with golden capsules. 
The capsules should be large, approximately 50cm, with the same 
visual presence as the shopping bags. Golden metallic texture, 
soft sheen, naturally arranged.
```

实际生成: 胶囊仍然很小,形状仍然不对

致命偏差: 同上

我的盲区: 我以为"replace"+"same visual presence"能让AI理解"尺寸要一样大" → 为什么这是错的: "visual presence"是抽象词,AI无法精确量化。而且"replace...with capsules"这个句式中,"capsules"这个词仍然触发了AI的"小药丸"认知,后续的所有修饰词都在这个错误的基础上叠加。

✅ 解决方案与沉淀

修正提示词:

```
Replace the shopping bags in the first image with the golden capsule 
shown in the second image. The capsule shape must exactly match the 
second image.
```

关键改动 (Before vs After):

| 改动 | 原词 → 新词 | 原因 |
|------|-------------|------|
| 1 | golden capsules, 50cm, elongated oval → the golden capsule shown in the second image | 不描述,直接指向参考图,让AI"看图"而不是"理解词" |
| 2 | with the same visual presence as shopping bags → Replace the shopping bags...with... | 用"替换"逻辑而非"描述+生成"逻辑,让AI做物体交换而非重新创造 |
| 3 | 删除所有尺寸/形状/材质描述 | 这些描述都是噪音,会与"capsule"的固有认知冲突 |
| 4 | 增加 must exactly match | 强调一致性约束,防止AI"创意发挥" |

成功结果:

- 电梯场景: https://a.lovart.ai/artifacts/agent/jwJMMC2qX1FVCIt5.png
- 车后备箱: https://a.lovart.ai/artifacts/agent/FcJDEJ0jCD1eHsFr.png

用户确认: "这次就对了"

📚 核心教训 (Do & Don't)

❌ 不要:

**不要试图用文字描述对抗AI的固有认知**

- 错误做法: 写"50cm capsule"、"large capsule"、"elongated capsule"
- 会导致: AI仍然生成小药丸,只是稍微放大或拉长,本质形状不变
- 原因: "capsule"这个词触发的默认认知太强,后续修饰词无法覆盖

**不要堆砌同义词试图"加强"指令**

- 错误做法: "densely packed, naturally scattered, filling the space"
- 会导致: 权重失衡,AI把注意力分散到"密集"上,反而忽略形状
- 原因: 描述字数 = 视觉权重,次要元素描述过多会抢主体的戏

**不要用抽象词描述具体要求**

- 错误做法: "same visual presence"、"similar size proportion"
- 会导致: AI无法精确理解,按自己的方式解读
- 原因: 抽象词没有明确的视觉锚点

✅ 应该:

**用"替换"逻辑绕过AI的词汇偏见**

- 正确做法: "Replace A in image1 with B shown in image2"
- 适用于: 当目标物体有AI固有认知偏见时(如capsule、pill、button等)
- 原理: 让AI做"物体交换"而非"理解词汇+重新生成",直接从参考图提取视觉特征

**用"must exactly match"强化一致性约束**

- 正确做法: 在替换指令后加"The [元素] must exactly match the reference image"
- 适用于: 需要精确复制参考图中某个元素的形状/颜色/质感时
- 原理: 明确告诉AI"不要创意发挥,要1:1复制"

**提示词极简化,只保留关键指令**

- 正确做法: 2句话 = 替换指令 + 一致性约束
- 适用于: 有明确参考图的替换/编辑任务
- 原理: 参考图已经包含了所有视觉信息(光线、材质、颜色),文字不要重复描述,否则会产生冲突

🎓 通用规律提取

这个案例揭示的底层规律:

当AI对某个词有强烈的固有认知时,不要试图用更多的文字描述去"纠正"它,而应该用参考图+替换逻辑绕过这个词汇陷阱。

AI的理解机制:

```
文字提示词 → 触发训练数据中的概念联想 → 在这个概念基础上叠加修饰
                ↑
            如果这一步就错了,后续所有修饰都是在错误基础上累积
```

正确的策略:

```
参考图 → AI直接提取视觉特征 → 应用到新场景
    ↑
不经过"词汇理解"这一步,避免了固有认知偏见
```

🔖 可复用的提示词模板

场景: 需要将场景A中的物体X替换为参考图B中的物体Y,且Y的形状/颜色/质感必须精确一致

模板:

```
Replace the [物体X] in the first image with the [物体Y] shown in 
the second image. The [物体Y] [关键特征] must exactly match the 
second image.
```

注意事项:

- 不要描述物体Y的任何视觉特征(尺寸/颜色/材质),让参考图说话
- 如果需要调整摆放方式,可以加一句"Arrange the [物体Y] like the [物体X] in the original image"
- 保持提示词在3句话以内
- 使用quality="low"快速迭代,确认方向对了再提升质量

### 透明瓶装胶囊产品场景图替换 | 错误类型: 排版布局/产品尺寸/元素保留 | 严重度: ⭐⭐
项目背景
正在为透明瓶装珠光白胶囊产品制作电商场景图，需要将原场景图中的产品替换成新产品，保持原有的温暖高级感氛围和文字排版区域，主打"纯净、高端"的视觉概念。

当前卡点
AI 在产品替换时容易"过度创作"，要么改变太多原有元素（背景、构图、装饰物），要么产品尺寸不合适，或者无法精准保留/去除特定元素（如镊子）。

🎯 任务目标
预期效果: 在保持原场景氛围和文字区域的前提下，将产品替换为新的透明瓶装胶囊，产品放大突出，去除镊子，保留盖子和散落胶囊。

参考图:

图1 [原场景图] - 参考其温暖米金色背景、光线氛围、文字布局区域
图2 [新产品图] - 15°俯视角度、开盖状态的透明瓶装胶囊
❌ 错误现场与诊断
第一次尝试（过度保守）
原提示词:

Replace the product in the first image with the transparent cylindrical jar from the second image. Keep the exact same scene composition...
实际生成: [ZhBFD9YxhVDwaNQK.png]

致命偏差: 产品太小，不够突出；镊子保留了但用户不需要

根本原因:

 缺少关键约束
 权重失衡
深度剖析: AI 把"Keep the exact same scene"理解成了"所有元素都不能动"，导致产品尺寸和原图一样小。

我的盲区: 我当时想的是"保持场景一致" → 为什么这是错的：应该明确"哪些要保持，哪些可以调整"

第二次尝试（过度创作）
原提示词:

Create a product scene inspired by the first image's layout and atmosphere, but replace with the transparent cylindrical jar from the second image. 

Key changes:
- Enlarge the jar significantly...
- Tweezers holding one pearl white water-drop shaped capsule...
实际生成: [B90Xp3XZYmHkDcMW.png]

致命偏差: 改变太多！背景材质、胶囊数量、整体构图都变了，完全不像原场景

根本原因:

 词语冲突
 抽象词过多
深度剖析: AI 把"inspired by"理解成了"可以自由发挥"，而不是"严格基于"。

我的盲区: 我当时想的是"给AI更多创作空间" → 为什么这是错的：产品替换任务需要精准控制，不是创意发散

✅ 解决方案与沉淀
修正提示词（第三次成功）
Based on this image, make these specific changes:
1. Enlarge the transparent jar significantly (make it 1.5-2x larger)
2. Remove the tweezers completely
3. Keep everything else exactly the same: the warm beige-gold background, the text layout areas, the single capsule on the right, the lid placement, the overall composition and lighting

The jar should remain in the lower right area with the same angle and lighting, just larger in size.
关键改动 (Before vs After)
改动1: "inspired by" → "Based on this image"
原因: "Based on"建立了明确的基准约束，防止AI过度创作

改动2: "Keep the exact same scene" → "make these specific changes" + 列举保留项
原因: 用"specific changes"明确哪些要改，用列举法明确哪些要保留，打破AI的"全盘保留"或"全盘重做"二元思维

改动3: "Enlarge the jar significantly" → "make it 1.5-2x larger"
原因: 给出具体倍数，避免AI对"significantly"的理解偏差

改动4: 增加 "Remove the tweezers completely"
原因: 明确去除不需要的元素，而不是在保留列表中"不提它"

核心教训 (Do & Don't)
❌ 不要: 用"inspired by"做产品替换任务
会导致: AI 过度发挥，改变太多原有元素

✅ 应该: 用"Based on this image, make these specific changes"
适用于: 需要精准控制改动范围的编辑任务

❌ 不要: 用"Keep the exact same scene"这种笼统描述
会导致: AI 理解成"所有元素都不能动"，包括产品尺寸

✅ 应该: 明确列举"保留项"和"改动项"，用数字列表分离
适用于: 需要部分保留、部分修改的复杂编辑任务

❌ 不要: 用"significantly"、"much larger"等模糊词描述尺寸
会导致: AI 对"大"的理解可能是 1.2 倍，也可能是 3 倍

✅ 应该: 给出具体倍数或百分比（如"1.5-2x larger"、"increase by 50%"）
适用于: 所有涉及尺寸、比例调整的任务

❌ 不要: 通过"不提某元素"来暗示去除它
会导致: AI 可能保留也可能去除，不可控

✅ 应该: 明确写"Remove [元素] completely"
适用于: 需要去除特定元素的编辑任务

🎓 通用原则沉淀
1. 产品替换任务的提示词结构
Based on [base image], make these specific changes:
1. [改动1 - 用具体数值]
2. [改动2 - 明确去除/添加]
3. Keep everything else exactly the same: [列举保留项]

[补充约束：位置、角度、光线等]
2. 词汇选择优先级
场景	❌ 避免	✅ 使用
基于现有图编辑	inspired by, reference	Based on, using this image
描述改动范围	keep the same scene	make these specific changes + 列举
描述尺寸	significantly, much	1.5x, 50% larger, double the size
去除元素	不提它	Remove [元素] completely
保留元素	keep everything	Keep: [列举具体项]
3. 多角度产品图的比例选择
产品形态	❌ 错误比例	✅ 正确比例
圆柱形瓶装	1:1 方形	3:4 或 9:16 竖版
扁平包装	9:16 竖版	16:9 或 4:3 横版
9宫格展示	1:1 (会压扁产品)	根据产品形态选择整体画布比例
原因: 产品的真实比例应该决定画布比例，而不是为了"好看"强行用方形
