# 新建品牌技能

快速搭建一个新品牌的完整项目结构。**一次沟通，自动生成，直接能用。**

---

## 什么时候用

Kevin 说"新建品牌"、"建个新品牌项目"、"XX品牌的项目搭一下"时使用。

---

## 输入（一次性问清楚）

新建品牌时，一次性问清楚以下信息，后面不用再问：

| 类别 | 必填 | 说明 |
|------|------|------|
| **品牌名** | ✅ | 中文+英文（如"婷美 TIMIER"） |
| **品牌定位/口号** | ✅ | 一句话说清楚品牌是做什么的 |
| **品牌官网** | ❌ | 有的话可以搜索更多信息 |
| **核心色/视觉调性** | ✅ | 比如"红金配色、轻奢科技感"、"琥珀色调、高级暖调" |
| **第一个产品名** | ✅ | 比如"虾青素胶囊精华" |
| **第一个产品图** | ✅ | 至少1张正面图，最好有多角度 |
| **第一个项目名** | ✅ | 比如"唯品会-虾青素主图" |
| **第一个项目需求文件** | ✅ | Excel/brief/需求文档 |
| **出图风格** | ✅ | 用什么风格？琥珀光影？还是其他？有没有现成的提示词矩阵？ |

---

## 执行流程

### 1. 复制模板

把 `Projects/_模板/品牌模板/` 整个复制到 `Projects/{{品牌名}}/`。

### 2. 重命名和替换变量

把所有模板里的占位符替换成实际内容：

| 占位符 | 替换为 | 说明 |
|--------|--------|------|
| `{{品牌名}}` | 实际品牌名 | 如"婷美" |
| `{{品牌英文名}}` | 品牌英文名 | 如"TIMIER" |
| `{{产品名}}` | 第一个产品名 | 如"虾青素胶囊精华" |
| `{{项目名}}` | 第一个项目名 | 如"唯品会-虾青素主图" |
| `YYYY-MM-DD` | 当天日期 | 项目文件夹名的日期前缀 |

需要重命名的文件/文件夹：
- `资产/产品名/` → `资产/{{产品名}}/`
- `资产/产品名.md.template` → `资产/{{产品名}}.md`
- `YYYY-MM-DD-项目名/` → `YYYY-MM-DD-{{项目名}}/`
- `brand.md.template` → `brand.md`
- `作图经验/00-出图工作流.md.template` → `作图经验/00-{{品牌名}}出图工作流.md`

### 3. 生成 brand.md 内容

根据用户提供的品牌信息，填写 brand.md：
- 品牌基本信息（品牌名、官网、定位、核心主张、口号）
- 视觉风格（主色调、设计风格、视觉元素）
- 电商主图规范（首图、辅图逻辑）
- 详情页叙事逻辑
- 设计原则

**注意：** 这是初稿，需要用户确认。不要自己瞎编，基于用户提供的信息+搜索到的公开信息整理。

### 4. 建第一个产品档案

- 把用户提供的产品图放到 `资产/{{产品名}}/` 下
- 写产品外观描述（根据图片）
- 参考图链接放进去
- 提示词、避坑经验、成功案例先空着，出图后补

### 5. 建第一个项目

- 把用户提供的需求文件放到 `YYYY-MM-DD-{{项目名}}/01-输入/` 下
- 写需求整理.md（提取卖点、调性、设计要求）

### 6. 整理作图经验

- 出图工作流文件替换变量后直接用
- 如果用户指定了出图风格（如琥珀光影），且有现成的提示词矩阵，把矩阵文件复制到 `作图经验/风格/` 下

### 7. 加反向链接

给相关的 Wiki 概念加上指向新品牌工作流的反向链接。

### 8. 一次性确认

搭完后，一次性给用户看：
1. 目录结构
2. brand.md 内容
3. 产品档案内容
4. 需求整理内容

用户说哪里不对，就改哪里，改完直接开始干活。

---

## 输出

一个完整的、可以直接开始出图的品牌项目目录。

**验收标准：**
- ✅ 目录结构完整
- ✅ 两个 instructions.md 有内容
- ✅ brand.md 有初稿
- ✅ 第一个产品档案建好（有图有描述）
- ✅ 第一个项目目录建好，需求整理完
- ✅ 作图经验入口文件（00-出图工作流）有内容
- ✅ 风格定义文件有内容（如果用现成风格的话）

## 模板文件说明

| 文件 | 类型 | 说明 |
|------|------|------|
| `SKILL.md` | 技能说明 | 本文件，怎么用这个模板 |
| `instructions.md` | 通用 | 品牌根目录说明，直接复制 |
| `brand.md.template` | 模板 | 品牌档案模板，需要填内容 |
| `资产/产品名.md.template` | 模板 | 产品档案模板，需要填内容 |
| `作图经验/instructions.md` | 通用 | 作图经验说明，直接复制 |
| `作图经验/00-出图工作流.md.template` | 模板 | 出图工作流，替换品牌名即可 |
| 各种空目录 | 通用 | 直接复制，保持结构一致 |

---

## 注意事项

1. **不要自己瞎编品牌信息**：基于用户提供的+公开搜索到的信息整理，不确定的标出来让用户确认
2. **不要跳过确认步骤**：搭完必须给用户确认一次，不能直接就开始出图
3. **模板本身也要迭代**：发现新的通用文件，就加到模板里，下次所有新品牌都有
4. **源文件规则**：`资产/产品名/源文件/` 里只放真正的源文件（.tif、.psd等），常用导出图直接放产品文件夹下
