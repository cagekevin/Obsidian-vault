## 候选 1：提示词分层结构

### 什么时候用
- 写复杂场景的提示词时
- 需要控制画面多个元素时
- 提示词写得很乱、不知道怎么组织时

### 核心内容
提示词按重要性分三层，从前往后权重递减：

**第一层：核心指令（前 10-15 词，权重最高）**
- 核心主体 + 关键动作 + 决定性风格
- 例：`Cistus ladanifer flower growing on coastal cliffs`

**第二层：细节增强（中间 20-30 词）**
- 材质细节、颜色特征、环境元素、空间关系
- 例：`white papery petals with deep crimson basal spots, golden stamens, blue-green ocean waves, dark rocks`

**第三层：微调优化（最后部分）**
- 画质、光线、镜头、色彩倾向等修饰性指令
- 例：`soft diffused light, overcast sky, documentary photography style, cool-toned`

### 注意事项
- 重要的东西一定放前面
- 不要把风格/技术参数放最前面，会抢主体的权重
- 和「描述比重原则」配合使用：主体描述占比最大，放最前面

## 候选 2：顺序权重原则

### 什么时候用
- 需要强调某个元素时
- 发现 AI 总是忽略你说的某个东西时
- 调整画面元素的重要性时

### 核心内容
AI 模型对提示词的理解有「位置偏见」：
- **越靠前的词，权重越高**
- **越靠后的词，权重越低**
- 开头的几个词决定了画面的整体基调

### 实战用法
- 最重要的主体放最前面
- 次要元素往后放
- 技术参数、风格修饰放最后

**❌ 错误（风格放最前面，抢了主体权重）：**
```
poetic cinematic style, a bottle of serum on white background
```

**✅ 正确（主体放最前面）：**
```
A bottle of capsule serum on pure white background, professional product photography, clean studio lighting
```

### 注意事项
- 这是通用原则，不同模型程度不同，但都适用
- 和「描述比重原则」是两件事：比重是"说多少"，顺序是"放在哪"
- 两个配合用效果最好：重要的东西既多说，又放前面

## 候选 3：负面提示词技巧

### 什么时候用
- AI 总是生成你不想要的东西时
- 需要排除特定元素时
- 画面有瑕疵需要规避时

### 核心内容
告诉 AI **不要什么**，有时候比告诉它要什么更有效。

### 常见的负面提示词分类
**1. 质量问题**
```
blurry, low quality, pixelated, out of focus, oversharpened
```

**2. 人体/手部问题（如果有人物）**
```
extra fingers, mutated hands, poorly drawn hands, extra limbs
```

**3. 不想要的元素**
```
text, watermark, logo, signature, frame, border
```

**4. 风格冲突**
```
3D render, cartoon, anime, illustration, painting, digital art
```

### 注意事项
- 负面提示词不是越多越好，太多会让画面变僵
- 只加你确实不想要的，不要乱加一堆
- 诗意写实风格有自己的负面词模板，不要混用

## 候选 4：材质质感描述词汇

### 什么时候用
- 描述产品材质时
- 需要精确控制质感时
- 觉得画面"塑料感"、"不真实"时

### 核心内容
不同材质有对应的精确描述词汇，用对了质感会真实很多。

### 常用材质词汇
**玻璃/透明材质：**
```
transparent, translucent, frosted glass, clear glass, glossy, refractive, light passing through
```

**金属：**
```
matte metal, brushed metal, polished chrome, gold foil, metallic sheen, reflective
```

**塑料/胶囊：**
```
pearlescent, iridescent, matte finish, smooth plastic, soft silicone, gel-like
```

**植物/自然：**
```
papery texture, velvety, waxy, leathery, fibrous, delicate, brittle
```

**皮肤/人体：**
```
skin texture, pores, fine lines, natural skin, dewy, matte skin
```

### 注意事项
- 不要堆砌太多材质词，选 1-2 个最准确的
- 「视觉描述优先原则」：用具体的材质词，不用"高级感"这种抽象词
- 产品图里材质描述很重要，直接影响画面真实感

## 候选 5：光线控制词汇

### 什么时候用
- 需要控制画面氛围时
- 觉得光线不对、太硬或太软时
- 统一同系列图片的光线时

### 核心内容
光线是影响画面氛围最大的因素之一，精确描述光线比说"氛围感"有用得多。

### 常用光线词汇
**柔和/漫射光：**
```
soft diffused light, overcast lighting, natural window light, soft shadows
```

**硬光/定向光：**
```
hard lighting, directional light, dramatic shadows, rim light, backlit
```

**时间段：**
```
golden hour, blue hour, midday sun, morning light, dusk, dawn
```

**光线质量：**
```
even lighting, flat lighting, high contrast, low key, high key
```

**工作室灯光：**
```
studio lighting, three-point lighting, softbox lighting, ring light
```

### 注意事项
- 光线描述要具体，不要说"氛围感好"，要说"soft diffused light, overcast"
- 「一致性锚点原则」：同系列图片光线描述要一致
- 诗意写实风格偏好柔和漫射光，避免硬光和戏剧化光线

## 候选 6：构图与视角词汇

### 什么时候用
- 需要控制画面构图时
- 需要特定视角时
- 同系列图片需要统一构图时

### 核心内容
明确描述视角和构图，AI 就不会乱摆。

### 常用视角词汇
**景别：**
```
close-up, macro shot, medium shot, wide shot, full shot, extreme close-up
```

**角度：**
```
front view, eye level, top view, bird's eye view, low angle, 45 degree angle
```

**构图：**
```
centered composition, rule of thirds, negative space, symmetrical, leading lines
```

**景深：**
```
shallow depth of field, bokeh, deep focus, sharp focus, soft background
```

### 注意事项
- 产品图常用：front view, centered, clean white background
- 场景图常用：wide shot, rule of thirds, natural depth of field
- 「参考图优先原则」：有对应角度的参考图比纯文字描述准确得多

## 候选 7：具体性原则

### 什么时候用
- AI 生成的东西总是很"通用"时
- 觉得画面没有特点、记不住时
- 需要独特性和辨识度时

### 核心内容
**越具体，越真实；越模糊，越 generic。**

AI 不怕你说得多，就怕你说得模糊。

**❌ 模糊描述：**
```
a beautiful flower, elegant, high quality
```

**✅ 具体描述：**
```
Cistus ladanifer flower, five white papery petals with deep crimson basal spots, dense golden stamens
```

### 具体到什么程度
- 有学名的用学名（植物、动物）
- 有具体颜色的用具体颜色（不是"红色"，是"deep crimson, maroon"）
- 有具体材质的用具体材质（不是"光滑"，是"matte finish, pearlescent"）
- 有具体动作的用具体动作（不是"放着"，是"resting naturally on"）

### 注意事项
- 和「视觉描述优先原则」是一个意思的不同角度
- 具体 = 视觉化 = 可被画出来
- 但也要注意「克制原则」：不要为了具体而堆砌，重点突出就好
