## 风格一：诗意写实（Lyrical Realism）

> 完整方法论 + 工作流详见 `作图规范.md`

### 核心方法论

#### 1. 反极端化原则

AI 生成系统存在固有的"极端化倾向"——过度强调某个元素会导致整体失衡。

**主体权重法则：**
- 主体描述：占提示词前 50-60%，使用精确的形态/材质/色彩词汇
- 环境描述：占 20-30%，使用抽象的氛围词汇
- 技术参数：占 10-20%，仅含必要的摄影术语

**反面案例：**
- ❌ `一只手优雅地托着花朵，手指修长，指甲光洁...` → AI 将焦点转移到手部
- ✅ `Cistus ladanifer 花朵特写，一只手自然托举于画面底部边缘` → 手部是构图元素而非主体

#### 2. 三层解构框架

**Layer 1: 主体结构** — 植物学级别的精确描述（材质、纹理、色彩分布、形态特征）
**Layer 2: 空间关系** — 主体与环境的位置关系，使用摄影构图术语
**Layer 3: 情绪质感** — 克制的氛围词，不超过 3-5 个

#### 3. 偶发美学构图

拒绝摆拍痕迹，利用环境自带元素形成"刚好被捕捉"的天然构图：
- 光线切割（百叶窗条纹光、树影斑驳）
- 框架构图（门框、窗框、镜子反射）
- 前景虚化（雨水车窗、玻璃杯、半透明织物）
- 自然遮挡（手部、植物枝叶、桌面边缘）

#### 4. 抒情自然光

**时间-光线对应表：**

| 时间段 | 光线特征 | 情绪 | 禁用 |
|--------|---------|------|------|
| 清晨 6-8 点 | 冷调柔光，低角度侧光 | 宁静、希望 | 顶光、环形光 |
| 正午 11-13 点 | 高对比度硬光（需柔化） | 日常、真实 | 直射硬光 |
| 黄昏 17-19 点 | 暖调漫反射，长投影 | 忧郁、诗意 | 霓虹灯、彩色光 |
| 夜晚 20-22 点 | 微弱台灯/烛光，局部照明 | 私密、内省 | 影棚灯、闪光灯 |

### 色调 / 光线 / 情绪 / 构图 / 质感

| 维度 | 要求 | 拒绝 |
|------|------|------|
| **色调** | 低饱和为主，冷调（灰蓝银绿）或低饱和暖调（淡黄、米白、浅金） | 高饱和、复古暖色 |
| **光线** | 柔和漫射光、阴天光 | 斑驳光影、戏剧性光线 |
| **情绪** | 克制、宁静、忧郁但治愈 | 过度渲染、用力过猛 |
| **构图** | 主体清晰占主要空间，次要元素退到边缘或虚化 | 喧宾夺主 |
| **姿态** | 自然、放松、精确描述每个部位的状态 | 僵硬、刻意、功能性描述 |
| **质感** | 真实、有细节、有瑕疵 | 过于完美、塑料感 |

### 适用场景

- 品牌情绪片（生活方式、使用场景）
- 社媒内容（小红书、Instagram 氛围感素材）
- 品牌故事视觉化（朝圣之路、岩玫瑰意境）

### 不适用场景

- 电商详情页主图（需要清晰展示产品）
- 产品功能对比图
- 促销活动海报

### AI 提示词模板

> 作图协作流程 → 详见 `作图规范.md`
> 实战踩坑记录 → 详见 `资产/提示词经验/2026-06-22-岩玫瑰海岸场景.md`

**标准结构：**
```
[主体精确描述 50-60%] + [空间/构图关系 20-30%] + [光线环境 10-15%] + [情绪氛围 5-10%]
```

**实战案例对比：**

❌ 失败（主次失衡）：
```
A woman's elegant hands gently holding a Cistus ladanifer flower, fingers delicate, nails polished, skin smooth, soft lighting, poetic atmosphere
```
→ 手部描述占 40%，AI 将手识别为主体

✅ 成功（权重正确）：
```
Cistus ladanifer in close-up, five papery white petals with deep crimson basal spots, dense golden stamens, naturally held at bottom edge of frame by a hand, soft morning light from left, gentle coastal breeze, calm and poetic
```
→ 花朵 55%，手部仅 10%，光线情绪 35%

**负向提示词：**
```
studio lighting, looking at viewer, posed, oversharpened, too clean, commercial, 3D render, epic, dramatic
```

### 写实度校准标准

**材质真实性检查：**
- [ ] 花瓣是否呈现"纸质褶皱感"而非塑料光泽？
- [ ] 皮肤是否有微妙的纹理而非过度磨皮？
- [ ] 织物是否有自然的垂坠感而非僵硬？

**光影逻辑检查：**
- [ ] 阴影方向是否统一？
- [ ] 高光位置是否符合光源角度？
- [ ] 是否存在"悬浮光"（无光源的亮斑）？

### 诗意质感评估四准则

1. **日常的诗意** — 画面是否传达"平凡瞬间的美"而非宏大叙事？
2. **偶发美学** — 构图是否看起来"刚好被捕捉"而非刻意摆拍？
3. **抒情自然光** — 光线是否柔和且符合真实场景逻辑？
4. **隐形的凝视** — 人物（如有）是否沉浸在自己的世界而非表演给镜头？

### 参考大师风格

- 维姆·文德斯（Wim Wenders）— 公路诗意的凝视
- 是枝裕和 — 日常生活中的细腻温情
- 上田义彦 — 日系静谧光影

### 诗意纪实美学字典

> 生图时从此字典中挑选关键词组合使用。

**📸 构图与质感**

| 关键词 | 说明 |
|--------|------|
| `Candid photography / Unposed` | 极度自然、毫无防备的抓拍感 |
| `Frame within a frame` | 画中画构图（门框、窗户、后视镜） |
| `Poetic mundane / Slice of life` | 诗意的平凡 / 生活切片 |
| `Kodak Portra 400` | 经典人文胶片，色调温润 |
| `Cinestill 800T` | 适合夜间微光与霓虹光晕的胶片 |
| `Halation / Soft bloom` | 胶片特有的高光红晕与柔和泛光 |
| `Soft film grain` | 柔和胶片颗粒感 |
| `35mm photography` | 35mm 胶片摄影质感 |

**💡 诗意光源**

| 关键词 | 说明 |
|--------|------|
| `Soft window light` | 透过窗户的柔和漫射光（高级生活感必备） |
| `Dappled sunlight` | 斑驳树影光（午后慵懒诗意） |
| `Blue hour` | 蓝调时刻 |
| `Melancholic overcast` | 忧郁质感的阴天柔光 |
| `Cinematic practical lighting` | 画面内真实光源打光（台灯、电脑屏幕、路灯） |
| `Golden hour window light` | 傍晚金色窗光 |
| `Warm ambient light` | 温暖的氛围光 |
