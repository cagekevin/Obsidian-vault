# 设计风格指南

> HKH 品牌设计视觉风格可选方案。每次出图前先确认使用哪种风格。

---

## 风格一：诗意写实（Lyrical Realism）

### 风格定位

纪实摄影感，不是摆拍。日常中的诗意，不刻意。看起来像"偶然捕捉"。

### 色调体系

冷调、低饱和、灰蓝银绿。
❌ 拒绝：暖调、高饱和、复古暖色

### 光线

柔和漫射光、阴天光。
❌ 拒绝：斑驳光影、戏剧性光线

### 情绪

克制、宁静、忧郁但治愈。
❌ 拒绝：过度渲染、用力过猛

### 构图

- 主体清晰占据主要空间
- 次要元素退到边缘或虚化，不喧宾夺主

### 姿态（涉及手/人时）

- 自然、放松、毫不费力
- 精确描述每个部位的状态
- ❌ 拒绝：僵硬、刻意、功能性描述

### 质感

真实、有细节、有瑕疵。
❌ 拒绝：过于完美、塑料感

### 适用场景

- 品牌情绪片（生活方式、使用场景）
- 社媒内容（小红书、Instagram 氛围感素材）
- 品牌故事视觉化（朝圣之路、岩玫瑰意境）

### 不适用场景

- 电商详情页主图（需要清晰展示产品）
- 产品功能对比图
- 促销活动海报

### 参考大师风格

- 维姆·文德斯（Wim Wenders）— 公路诗意的凝视
- 是枝裕和 — 日常生活中的细腻温情
- 上田义彦 — 日系静谧光影

### 诗意纪实美学字典

> 生图时从此字典中挑选关键词组合使用。

**📸 构图与质感**

| 关键词 | 说明 |
|--------|------|
| `Candid photography / Unposed` | 极度自然、毫无防备的抓拍感 |
| `Frame within a frame` | 画中画构图（门框、窗户、后视镜） |
| `Poetic mundane / Slice of life` | 诗意的平凡 / 生活切片 |
| `Kodak Portra 400` | 经典人文胶片，色调温润 |
| `Cinestill 800T` | 适合夜间微光与霓虹光晕的胶片 |
| `Halation / Soft bloom` | 胶片特有的高光红晕与柔和泛光 |
| `Soft film grain` | 柔和胶片颗粒感 |
| `35mm photography` | 35mm 胶片摄影质感 |

**💡 诗意光源**

| 关键词 | 说明 |
|--------|------|
| `Soft window light` | 透过窗户的柔和漫射光（高级生活感必备） |
| `Dappled sunlight` | 斑驳树影光（午后慵懒诗意） |
| `Blue hour` | 蓝调时刻 |
| `Melancholic overcast` | 忧郁质感的阴天柔光 |
| `Cinematic practical lighting` | 画面内真实光源打光（台灯、电脑屏幕、路灯） |
| `Golden hour window light` | 傍晚金色窗光 |
| `Warm ambient light` | 温暖的氛围光 |

## 风格二：功效美学（Efficacy Aesthetics）

> HKH 品牌定位"功效美学抗老品牌"，此风格为产品展示主风格。

### 核心准则

- 产品主体清晰、质感突出
- 背景干净、极简、留白
- 光线柔和但有层次，突出材质（胶囊质感、精华流动性）
- 色彩控制在白 + 日落金主色调内

### 适用场景

- 电商详情页、主图
- 产品介绍图
- 官网产品页

---

## 风格三：杂志诗意（Poetic Editorial）

### 核心准则

- 极简排版、大面积留白
- 高级低饱和色彩搭配
- 纸质感、光影氤氲
- 适合前卫时尚调性

### 适用场景

- 品牌海报
- 官网 banner
- 品牌宣传册
