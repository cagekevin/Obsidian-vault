# 输入文件夹

甲方/运营给的原始需求放这里。

## 命名规范

按需求批次建文件夹：`YYYY-MM-DD-需求描述/`

```
✅ 2026-06-22-小红书岩玫瑰溯源/
✅ 2026-06-23-社媒素材/
❌ 详情页需求/
```

## 批次的完整内容

每个批次文件夹应包含：

| 文件 | 来源 | 说明 |
|------|------|------|
| 原始文件 | 甲方/运营直接给的 | 保持原文件名，不改 |
| Markdown 文字稿 | AI 用 markitdown 转 | 方便 AI 直接读文字内容 |
| 配图/ | 如有配图压缩包，解压到此 | 图片保持原名或按内容重命名 |

## 流程

1. Kevin 收到需求文件 → 扔下载目录
2. AI 检测到新文件 → 建批次文件夹，复制原始文件
3. AI 用 markitdown 转文字稿（如 xlsx/docx/pdf 等）
4. 如有配图压缩包 → 解压到 `配图/`
5. 需要时 AI 可写需求摘要
