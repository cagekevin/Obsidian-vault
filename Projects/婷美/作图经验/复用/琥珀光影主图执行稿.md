# 琥珀光影主图执行稿

> 可复用的电商主图提示词模板。替换产品变量和垫图即可用于其他品牌/产品。
> 完整的col分组选项 → 读 `../风格/琥珀光影展台提示词矩阵.md`

---

## 适用场景

- 电商主图（白底/场景图都适用）
- 护肤品、化妆品、保健品等高端产品
- 需要暖色调、高级感、光影质感的产品图

---

## 快速上手

### 1. 准备垫图

- **必须垫图**：产品形态复杂的必须用真实产品图垫图，文字描述不准
- 多角度：如果有多个角度的产品图，可以每组用不同角度垫图
- 组合图：需要产品组合图时，用组合图垫图

### 2. 替换产品变量

把提示词里的 `[产品变量]` 替换成具体产品的英文描述。

**常见产品变量：**
| 产品 | 变量值 |
|------|--------|
| 通用 | `the premium main product` |
| 婷美虾青素胶囊 | `the premium astaxanthin capsule serum jar` |
| [其他产品] | [替换成对应产品描述] |

### 3. 组合提示词

从 `../风格/琥珀光影展台提示词矩阵.md` 中随机组合：

```
全局引擎 + 产品变量 + col-1随机1个 + col-2随机1个 + col-3随机1个 + col-4随机1个 + col-5随机1个 + col-6
```

**建议：** 每组10张，用不同的col组合，增加多样性。

### 4. 图生图生成

- 用垫图 + 完整提示词生成
- 每组生成后保存到项目的 `02-中间过程/` 目录
- 最终筛选好用的放到 `03-输出/`

---

## 注意事项

1. **必须垫图**：不要用纯文字生图，产品形态会跑偏
2. **主次比例**：主体描述占主导，次要元素不要抢戏
3. **克制原则**：少用程度词（非常、极其），AI容易走极端
4. **留白设计**：用自然元素形成文字区域，不要写"留白"、"空区域"
5. **一致性**：同一系列图保持统一的色调、光线、质感
