# 琥珀光影展台提示词矩阵

来自 Prompt_Matrix_V19，用于电商主图生成。**婷美主图默认使用此风格，原版色调，不改成红金。**

---

## 使用方法

从6个分组里各随机选1个，加上全局引擎，组合成完整提示词。

```
全局引擎 + 插入变量 + col-1随机1个 + col-2随机1个 + col-3随机1个 + col-4随机1个 + col-5随机1个 + col-6
```

**原理：** 人类就是这样随机组合的，每次组合都不一样，能出很多不同感觉的图。

---

## 全局引擎（global-constant）

```
High-end commercial product photography of [变量], shot on medium format 85mm lens with extreme micro-detail resolution, luxury aesthetic with a seamless color transition from creamy white to soft amber and champagne gold, elegant and breathable visual flow, immaculate studio curation with zero visual clutter, stunning natural optical depth and flawless material truth,
```

---

## 插入变量（insert-variable）

把全局引擎里的 `[变量]` 替换成具体产品描述。

**婷美虾青素胶囊：** `the premium astaxanthin capsule serum jar`

---

## col-1 - 物理陈列清单（底座/陈列台/材质/装饰元素）

控制产品放在什么上面、周围有什么装饰。

**常用选项：**
- 金色圆柱展台 + 丝带
- 橡木截面 + 棕榈叶
- 石灰华石块 + 水晶
- 石灰岩 + 浅水
- 金色托盘 + 丝绸
- 石灰华 + 水 + 马蹄莲
- 琥珀玻璃砖 + 兰花
- 胡桃木 + 水 + 山茶花
- 金色底座 + 丝绸
- 大理石块 + 金箔

**完整列表：** 参考外部文件 `Prompt_Matrix_V19.json`

---

## col-2 - 镜头排版（视角/留白空间）

控制构图视角和留白位置。

| 视角 | 留白特点 |
|------|----------|
| 平视正面 | 上方留白为主 |
| 高角度俯视 | 侧边留白多 |
| 绝对对称正面 | 四周均匀留白 |
| 稍高角度 | 下方留白多 |
| 低角度仰视 | 上方大量留白 |
| 荷兰角（倾斜） | 留白不规则 |
| 等距视角 | 特殊视角 |

**完整列表：** 参考外部文件 `Prompt_Matrix_V19.json`

---

## col-3 - 核心光影（明暗/雕塑感）

控制打光方式和3D体积感。

**常用选项：**
- 柔光箱 + 硬光阴影
- 连续光 + 聚光焦点
- 伦勃朗光（明暗对比）
- 底光 + 轮廓光
- 蝴蝶光 + 侧光
- 聚光灯 + 阴影线
- 背光 + 前补光
- 连续光 + 棱角阴影

**完整列表：** 参考外部文件 `Prompt_Matrix_V19.json`

---

## col-4 - 空气微观介质（氛围/物理特效）

控制整体氛围和特效。

**常用选项：**
- 光学纯度（干净通透）
- 光晕
- 散景
- 光泄漏
- 微粒子
- 色散
- 焦散

**完整列表：** 参考外部文件 `Prompt_Matrix_V19.json`

---

## col-5 - 背景（深邃空间与光影幕墙）

控制背景风格。

**常用选项：**
- 渐变背景
- 无限曲线（infinity curve）
- 磨砂玻璃纵深
- 径向光渐变
- 工作室扫光
- 画布背景 + 线性gobo
- 抽象阴影投影

**完整列表：** 参考外部文件 `Prompt_Matrix_V19.json`

---

## col-6 - 产品紧凑 + 留白

固定加在最后，确保产品不要太大，预留文字位置。

```
product positioned compactly, leave blank space for text placement.
```

**注意：** 留白位置不固定，取决于col-2的视角选择，可能右上角、左上角、上方、侧边等。
