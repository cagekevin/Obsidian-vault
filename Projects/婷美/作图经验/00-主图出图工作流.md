# 婷美主图出图工作流

婷美电商主图的标准出图流程。**核心原则：用组合图垫图 + 提示词矩阵随机组合，批量生成后人工筛选。**

---

## 总览

```
Kevin 说"出婷美主图"
    ↓
先读本文件，按流程走
    ↓
┌─ 必选（每次都要走）─────────────────────────────┐
│  阅读理解 → 资产调用 → 拼提示词 → 批量出图      │
└─────────────────────────────────────────────────┘
    ↓
┌─ 可选（按需插入）───────────────────────────────┐
│  ├ 风格调整       → 需要改色调/光影时            │
│  ├ 返工           → Kevin 说"不行"时             │
│  └ 沉淀           → 踩了新坑或有好用组合时       │
└─────────────────────────────────────────────────┘
```

---

## 必选步骤（每次都要走）

### 1. 阅读理解

**做什么：** 先想清楚需求，再动手。

先确认：
- **产品**：是虾青素胶囊精华吗？还是其他产品？
- **用途**：唯品会主图？还是其他渠道？
- **尺寸**：先做 1200×1200，确定后改 950×1200（默认）
- **数量**：要几组？每组几张？（默认8组×10张=80张）
- **风格**：琥珀光影展台原版色调？还是要改色？（默认原版琥珀色）

**关键提醒：**
- 主图只有背景+产品，没有文字，Kevin自己在PS里排版
- 必须用**组合图**垫图，不能用正面单瓶图
- 产品是**金鱼形状**的红色胶囊，不是水滴形

**输出：** 一段需求理解，给 Kevin 确认

---

### 2. 资产调用

**做什么：** 读这次出图需要的固有资产。

| 资产 | 位置 | 用途 |
|------|------|------|
| 产品档案 | `资产/虾青素胶囊精华.md` | 产品的形状、颜色、材质等固有特征 |
| 产品垫图 | `资产/虾青素胶囊精华/组合1-8.jpg` | 图生图的参考底图，必须用组合图 |
| 品牌规范 | `brand.md` | 婷美品牌色彩、调性等视觉规范 |
| 提示词矩阵 | 外部文件 `Prompt_Matrix_V19.json` | 琥珀光影展台的6组提示词模块 |
| 作图经验 | `作图经验/` | 之前踩过的坑和好用的组合 |

**注意：**
- 垫图必须用**组合图**（两瓶组合，一立一倒），不能用单瓶正面图
- 提示词矩阵里的"琥珀光影展台"是核心风格引擎

**输出：** 从各资产提取的关键信息摘要

---

### 3. 拼提示词

**做什么：** 用提示词矩阵的方式拼装，6个分组各选1个。

```
全局引擎（固定） + 插入变量（替换产品名）
    +
col-1 物理陈列（随机选1个）
    +
col-2 镜头排版（随机选1个）
    +
col-3 核心光影（随机选1个）
    +
col-4 空气微观介质（随机选1个）
    +
col-5 背景（随机选1个）
    +
col-6 留白要求（固定）
```

**具体操作：**
1. 全局引擎里的 `[变量]` 替换成具体产品描述（如 "the premium astaxanthin capsule serum jar"）
2. col-1 到 col-5 每个分组随机选1个选项
3. 最后加上 col-6 的留白要求："product positioned compactly, leave blank space for text placement"
4. 每组10张，用不同的随机组合

**自审与修改（必须做）：**
写完提示词后，必须做一次自审：

| 检查项 | 说明 |
|--------|------|
| **逻辑矛盾** | 前后不打架，不混用冲突词 |
| **极端化** | 有没有词可能让模型走极端（如"极致""完美"等） |
| **主次比例** | 产品描述占主导，背景和装饰不喧宾夺主 |
| **参考图** | 是否引用了正确的组合图作为垫图 |
| **产品形态** | 有没有明确是金鱼形状胶囊？（AI容易画成水滴形） |

自审发现问题 → 修改提示词 → 再自审一遍，直到无问题。

**输出：** 10组不同的提示词（对应10张图）

**参考：** `作图经验/琥珀光影展台使用技巧.md`

---

### 4. 批量出图

**做什么：** 按提示词执行，批量生成。

**执行规范：**
- 必须用**图生图**模式，引用对应组合图作为垫图
- 每组用不同的组合图垫图（第1组用组合1.jpg，第2组用组合2.jpg...）
- 每组生成10张，用不同的提示词组合
- 尺寸：1200×1200（默认）
- 输出到项目 `02-中间过程/第N组-组合N垫图/`

**文件命名：**
- 每组一个文件夹：`第N组-组合N垫图/`
- 图片按顺序命名：`01.jpg`、`02.jpg`...`10.jpg`

**输出：** 生成好的图片文件，保存到对应目录

---

## 可选步骤（按需插入）

### 风格调整

**什么时候插：** 需要改色调、光影或整体风格时。
**做什么：**
- 调整全局引擎的色彩描述（如把琥珀色改成红金色）
- 调整 col-3 光影类型
- 调整 col-5 背景风格
- 先改1-2张测试，确认后再批量改

---

### 返工

**什么时候插：** Kevin 说"XX 不行"时。
**铁律：**
- 只重跑指定编号的图
- 严格遵循原提示词，只改 Kevin 指出的问题
- 不重新描述整个画面
- 不改其他没问题的图

**常见返工原因：**
- 产品颜色不对（偏金/偏橙）→ 加强红色描述，或换垫图
- 胶囊形状不对（不是金鱼形）→ 换更清晰的组合图垫图
- 留白不够 → 调整 col-2 构图，或加强留白描述
- 质感不够 → 调整 col-3 光影，加强材质描述

---

### 沉淀

**什么时候插：** 踩了新坑、发现好用的组合、或有可复用的执行稿时。
**做什么：**
- 踩坑 → 记到当天校准日志
- 同一场景踩 2-3 次 → 沉淀到 `作图经验/案例/`
- 好用的提示词组合 → 归档到 `作图经验/琥珀光影展台使用技巧.md`
- 可复用执行稿 → 归档到 `作图经验/复用/`

**沉淀内容要求：**
- 问题是什么
- 原因是什么
- 怎么解决的
- 下次怎么避免

---

## 重要提醒

### 关于AI筛选
- **AI判断不准产品形态和颜色**，筛选由 Kevin 自己做
- AI只负责批量生成和整理，不做质量判断
- Kevin 一眼就能找到好的，不用AI瞎掺和

### 关于垫图
- 必须用**组合图**垫，不能用单瓶正面图
- 组合图有8张（组合1-8.jpg），对应8组
- 垫图越清晰，产品形态保持得越好

### 关于色调
- 琥珀光影展台**默认用原版色调**（奶油白→琥珀→香槟金）
- 不用特意改成婷美的红金色，产品本身的红色会自然透出来
- 如果 Kevin 说要改色，再调整

### 关于留白
- 留白位置不固定，取决于 col-2 的构图选择
- 可能是上方留白，也可能是左上角/右上角
- 统一加 "product positioned compactly, leave blank space for text placement" 即可
