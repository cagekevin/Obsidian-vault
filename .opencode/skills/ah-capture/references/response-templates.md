# ah-capture 交互文案与输出模板

## 1) 单条写入确认

```text
已记录到今日日记。
- 动作：{{记录/新增任务/标记完成}}
- 摘要：{{条目摘要}}
- 位置：{{notePaths.dailyNotes}}/{{YYYY-MM-DD}}.md#{{区块名}}
```

## 2) 批量写入确认

```text
本轮捕获已完成。
- 记录：{{n}} 条
- 新任务：{{n}} 条
- 标记完成：{{n}} 条
- 文件：{{notePaths.dailyNotes}}/{{YYYY-MM-DD}}.md
```

## 3) 歧义追问（只问一句）

```text
这是要标记任务完成，还是新增一个任务？
```

## 4) 任务未命中追问

```text
没找到对应任务，要把这条作为已完成记录写到“记录区”吗？
```

## 5) 结束追问

```text
还有吗？
```

## 6) 交互模式结束汇总

```text
本轮捕获结束：记录 {{n}} 条，新增任务 {{n}} 条，完成 {{n}} 条。
如需晚间统一收口，可使用 /ah-review。
```
