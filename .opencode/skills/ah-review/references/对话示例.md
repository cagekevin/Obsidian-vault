# 对话示例

## 标准模式（有时间）

```
用户：/ah-review

AI：我已读取今日日记。先快速回顾完成度，然后处理记录与未完成任务。

[回顾完成度后]

AI：现在处理第 1 条记录：...
1. 转永久笔记
2. 转任务
3. 已处理
4. 延期到 /ah-inbox

用户：1
AI：调用 ah-card 完成。继续下一条。

用户：3
AI：已标记 ✅ 已处理。

[记录处理结束]

AI：接下来安排未完成任务去向：明天做 / 本周做 / 以后再说 / 已完成 / 不做了。

AI：收尾前确认一下明日计划。
目前明天有以下任务：...
请回复“否”，或直接告诉我明天有哪些任务。

用户：明天加两项：10点前完成方案初稿；晚上复盘会议纪要。
AI：已写入今日日记 `## 明日计划`，本次回顾完成。
```

## 快速移交模式（时间不足）

```
用户：今晚没时间，快速过一下。

AI：收到。我会保留 1-2 条高价值记录优先处理，其余记录标记为 ⏭️ 待 /ah-inbox，方便你后续批量清理。

[处理后]

AI：已完成快速移交：
- 高价值已处理：2 条
- 延期到 /ah-inbox：5 条

你之后可直接执行 /ah-inbox 批量处理这些残留记录。

AI：另外，明日计划这一步请你二选一：
请回复“否”，或直接告诉我明天有哪些任务。

用户：否
AI：好的，本次回顾到这里结束。
```
