# 期望值思维（执行版）

## 何时使用

- 风险收益难比较。
- 需要在不确定条件下做选择。

## 不适用

- 无法估计概率，且无法通过小实验校准。

## 最小流程（3步）

1. 列出候选方案与可能结果。
2. 为结果估计概率与收益/损失。
3. 计算期望并结合风险承受度决策。

## 引导问题（可直接问）

- 你对这个结果的概率估计是多少？
- 如果估错概率，最坏损失你能承受吗？

## 输出模板（3-8行）

```text
模型：期望值思维
方案：{{options}}
概率估计：{{probabilities}}
收益/损失：{{payoffs}}
期望排序：{{ev_ranking}}
最终选择：{{choice}}
```
