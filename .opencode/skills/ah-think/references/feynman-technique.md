# 费曼技巧（执行版）

## 何时使用

- 用户说“好像懂了但说不清”。
- 需要把概念写成可复用卡片。
- 需要检验自己是否真理解，而不是只会复述术语。

## 不适用

- 只需快速摘录，不做深挖。
- 用户明确要求“先记录，不展开”。

## 最小流程（4步）

1. 选定一个概念（一次只选一个）。
2. 用非术语语言解释给“外行”。
3. 找到卡壳点（含糊、跳步、无法举例）。
4. 回到资料补洞，再次简化重述。

## 引导问题（可直接问）

- 你能不用专业词，解释 `X` 是什么吗？
- `X` 为什么重要？给一个生活中的例子。
- 哪一段你最难讲清楚？那里通常就是理解盲区。

## 与笔记工作流结合

- `ah-read`：阅读后先做一次“外行解释”，再决定是否入卡。
- `ah-card`：把“简化解释 + 例子 + 边界条件”写入永久笔记。

## 输出模板（3-8行）

```text
模型：费曼技巧
概念：{{concept}}
一句话解释：{{plain_explanation}}
关键例子：{{example}}
仍不清楚点：{{gap_or_none}}
下一步：{{re-read_or_record}}
```

## 退出条件

- 用户已能流畅解释并给出例子。
- 用户说“先记下来”或“先到这里”。
